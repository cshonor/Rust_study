# Item 1 · 02 标量类型

← [Item 1 目录](./README.md)

## 标量 vs「基础类型」

| 概念 | 范围 |
|------|------|
| **标量（scalar）** | 不能再拆的**原子**类型：`i32`/`u64`、浮点、`bool`、`char`、以及单元 `()` |
| **基础 / 内建类型**（广义） | 标量 + 简单组合：`[T; N]` 数组、元组 `(T1, T2)` 等（见 [03 聚合](./03-aggregate-types.md)） |

笔记里「基础类型严谨性」[01](./01-fundamental-types.md) 侧重**整数定长、无隐式转换**；**标量**专指**单个值、不可再分**的那几类。HFT 热路径优先标量（如 `i64` 时间戳、`f64` 价格）：栈上、布局固定、无动态类型开销。

标量 = **单个值**的类型。

| 类型 | 要点 |
|------|------|
| `bool` | `true` / `false`，占 **1 字节**（见下） |
| `f32` / `f64` | IEEE 754 浮点，32 / 64 位 |
| `()` | **单元类型**，唯一值 `()`；表示「无可用返回值」（见下） |
| `char` | **Unicode 标量**，固定 **4 字节**；不能与整数静默互转 |

---

## 1 字节 = 8 个二进制位

1 字节（byte）= **8 bit**，每位是 `0` 或 `1`，共 2⁸ = **256** 种组合。

- `u8`：无符号，可表示 **0～255**
- 例：ASCII `'a'` 在编码里是十进制 **97**，二进制 **`01100001`**（占 1 字节）

这和「bool 是不是全 0 / 全 1」是不同层次：下面单独说 `bool` 在内存里怎么放。

---

## `bool`：`true` 不是「8 个 1」

`bool` **占 1 字节**，但合法值只有两个，常见内存表示：

| 值 | 典型字节（十六进制） | 说明 |
|----|----------------------|------|
| `false` | `0x00` | 全 0 |
| `true` | `0x01` | **最低位为 1**，其余位为 0（不是 `0xFF` 八个 1） |

直觉：只用 **1 bit** 的语义（真/假），类型仍占 1 字节对齐；**不是** `true = 11111111`、`false = 00000000`。

### 和整数严格分开

不能把 `1` 当 `true`、`0` 当 `false` 混用：

```rust
let flag: bool = true;
// let x: i32 = flag;  // ❌ 不能当整数用
// let y: bool = 1;    // ❌ 不能拿非 bool 整数当 bool

if flag { /* ... */ }  // 必须本来就是 bool
```

HFT 里条件分支（是否触发下单、是否熔断）用 `bool`，避免 C 风格「非 0 即真」误触发。

---

## `char`：4 字节，不是 1 字节 ASCII

```rust
let c: char = 'a';  // Unicode 标量，内存里 4 字节存码点 U+0061
// let x: i32 = c;  // ❌ 不能与整数静默互转
```

- ASCII `'a'` 作为 **u8** 是 1 字节 `0x61`；
- Rust **`char`** 固定 **4 字节** 存 Unicode 码点（`'中'` 也是 `char`，不是多字节 `str` 切片）。

字符串文本用 `&str` / `String`，见 Book 字符串章节；这里只记：**标量 `char` ≠ 1 字节 char**。

---

## 单元类型 `()`

**定义**：类型名是 `()`，唯一值也是 `()`，表示「执行完了，但没有可继续用的返回值」。语义上接近 C 的 `void`，但 `()` 是**真正的类型和值**，可以出现在泛型、枚举里。

### 函数什么时候返回 `()`？

| 情况 | 结果 |
|------|------|
| 没写返回类型 | 默认 `-> ()` |
| 最后一行是**语句**（表达式后加了 `;`） | 该块返回 `()` |
| 显式 `-> ()` 或只写副作用 | 返回 `()` |

```rust
// 默认返回 ()
fn log_heartbeat() {
    println!("ping");
} // 隐式 -> ()

fn log_heartbeat2() -> () {
    println!("ping");
}

fn broken() -> i32 {
    println!("side effect");
    // 若这里分号结束、没有表达式 → 返回 ()，与 -> i32 冲突，编译错误
}
```

```rust
fn add(a: i32, b: i32) -> i32 {
    a + b   // 无分号：表达式作为返回值
}

fn add_void(a: i32, b: i32) {
    let _ = a + b;  // 分号：语句，函数体返回 ()
}
```

**区分两类逻辑**（你理解得对）：

- **有产出**：计算价格、数量 → 返回 `i32`、`f64`、`Result` 等；
- **只执行**：打日志、发心跳、写盘 → 返回 `()`，调用方不应把结果当数据继续算。
