# Item 28 · 重点结论

← [Item 28 目录](../README.md)

### 只在无可替代时用宏

- 先：**函数** → **泛型 + trait** → 最后 **宏**。

### 优先派生宏

- 为 struct/enum **每个字段/变体**生成代码 → **`#[derive(...)]`**，而非整类型 `foo!()` 生成。

### 对齐普通 Rust 直觉

| 避免 | 推荐 |
|------|------|
| 宏内隐式 `return` / `?` | 控制流像普通代码一样可见 |
| `my_macro!(list)` 内部偷偷加 `&` | `my_macro!(&list)` 调用方显式引用 |
| 自拼格式化 | 用 **`format_args!`**（如 `println!` 同款） |
