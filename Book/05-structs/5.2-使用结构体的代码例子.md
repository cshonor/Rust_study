# 5.2 使用结构体的代码例子：计算长方形面积

目标：计算长方形面积，并逐步重构代码，让数据更有“意义”。

对应 demo 项目：`Book/05-structs/5.2-rectangles-demo/`

---

## 版本 1：分别用变量表示宽高

```rust
fn area(width: u32, height: u32) -> u32 {
    width * height
}
```

- 问题：`width` 与 `height` 是“关联数据”，但函数签名看不出来它们必须成对出现

---

## 版本 2：用元组组合宽高

```rust
fn area(dimensions: (u32, u32)) -> u32 {
    dimensions.0 * dimensions.1
}
```

- 优点：只传 1 个参数，宽高被组合起来
- 缺点：`0/1` 索引没语义，容易弄混（特别是后续做绘制、打印等）

---

## 版本 3：用结构体命名字段

```rust
struct Rectangle {
    width: u32,
    height: u32,
}

fn area(rectangle: &Rectangle) -> u32 {
    rectangle.width * rectangle.height
}
```

- `width/height` 字段名让代码更清晰
- 传 `&Rectangle`（借用）避免移动所有权：`main` 仍可继续使用 `rect1`  
  这与第 4 章「引用与借用」一致

---

## 调试打印：Debug trait

结构体默认没有 `Display`（`{}`）也没有 `Debug`（`{:?}`）。详见 **[5.1.4 Debug / Display 与 Trait 约束](./5.1.4-Debug-Display与Trait约束.md)**（含 vs Python 鸭子类型）。

### 使用 `{:?}` / `{:#?}`

1. 先派生 `Debug`：

```rust
#[derive(Debug)]
struct Rectangle { ... }
```

2. 再用调试格式打印：

```rust
println!("rect1 is {:?}", rect1);
println!("rect1 is {:#?}", rect1); // 更漂亮的格式
```

---

## dbg! 宏

`dbg!` 会打印：
- 文件名与行号
- 表达式的值（Debug 格式）
- 并**返回该表达式的值**

```rust
let scale = 2;
let rect1 = Rectangle {
    width: dbg!(30 * scale),
    height: 50,
};
dbg!(&rect1);
```

注意：`dbg!` 输出到 **stderr**，`println!` 输出到 **stdout**。
