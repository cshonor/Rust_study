# 5.3 方法语法

**方法（method）** 与函数类似，但它们在某个类型（结构体/枚举/trait 对象）的上下文中定义，并且**第一个参数总是 `self`**（代表调用该方法的实例）。

对应 demo 项目：`Book/05-structs/5.3-method-syntax-demo/`

---

## 在 `impl` 块中定义方法

把针对 `Rectangle` 的 `area` 函数改为方法：

```rust
#[derive(Debug)]
struct Rectangle {
    width: u32,
    height: u32,
}

impl Rectangle {
    fn area(&self) -> u32 {
        self.width * self.height
    }
}
```

- `impl Rectangle { ... }`：与 `Rectangle` 类型相关联的实现块
- `&self` 是 `self: &Self` 的简写（在 `impl Rectangle` 中，`Self` = `Rectangle`）
- 方法可以选择：
  - `&self`：不可变借用（只读）
  - `&mut self`：可变借用（可修改实例）
  - `self`：获取所有权（较少见，常用于“消费自身并转换”）

---

## 方法名与字段同名：getter

```rust
impl Rectangle {
    fn width(&self) -> bool {
        self.width > 0
    }
}

// rect.width() 调方法
// rect.width  访问字段
```

Rust 不会自动为字段生成 getter。手写 getter 常用于：字段私有、方法公开，作为公共 API 的只读入口。

---

## 自动引用与解引用（为什么没有 `->`）

C：`ptr->func()` = `(*ptr).func()`。Rust **统一用 `.`**，编译器按方法签名自动补 `&` / `&mut` / `*`。

```rust
p1.can_hold(&p2);
(&p1).can_hold(&p2); // 等价：接收者 self 自动加 &
```

### 三种 `self` 的自动补全

| 方法签名 | 你写 | 编译器等价 |
|----------|------|------------|
| `fn area(&self)` | `r.area()` | `(&r).area()` |
| `fn set(&mut self, ...)` | `r.set(30, 40)` | `(&mut r).set(...)`（`r` 须 `mut`） |
| `fn into_data(self)` | `r.into_data()` | 直接 move `r`，不加引用 |

**口诀**：点号左边（**self 接收者**）自动加引用；**括号里其它参数不自动加 `&`**。

---

## 带参数的方法：`can_hold`

```rust
impl Rectangle {
    fn can_hold(&self, other: &Rectangle) -> bool {
        self.width > other.width && self.height > other.height
    }
}
```

```rust
rect1.can_hold(&rect2); // other 须手动 &；self 由 r1. 自动处理
```

> **只有 `.` 左边的实例（self）自动加引用；其它形参按普通函数规则，该 `&` 就手写 `&`。**

## 关联函数（associated function）

在 `impl` 中定义但**不以 `self` 为第一参数**的函数，称为关联函数（不是方法）。

典型用途：构造函数风格的创建器，比如正方形：

```rust
impl Rectangle {
    fn square(size: u32) -> Rectangle {
        Rectangle { width: size, height: size }
    }
}

let sq = Rectangle::square(3);
```

通过 `类型名::函数名` 调用（例如 `String::from`）。

### 三种 `self` 完整示例（可运行）

见 demo [5.3-method-syntax-demo](./5.3-method-syntax-demo/)：

```rust
impl Rectangle {
    fn area(&self) -> u32 { /* 只读 */ }
    fn set_size(&mut self, w: u32, h: u32) { /* 改字段 */ }
    fn destroy_to_tuple(self) -> (u32, u32) { /* 消费 self */ }
    fn new(w: u32, h: u32) -> Self { Self { width: w, height: h } }
}

let mut rect = Rectangle::new(10, 20);
let _ = rect.area();              // rect 仍可用
rect.set_size(50, 60);            // 须 mut
let (w, h) = rect.destroy_to_tuple(); // rect 此后不可用
```

| `self` 形式 | 含义 | 调用方 |
|-------------|------|--------|
| `&self` | 不可变借用 | 实例不必 `mut` |
| `&mut self` | 可变借用 | 实例须 **`mut`** |
| `self` | 拿走所有权 | 调用后实例 **move 失效** |

## 多个 `impl` 块

同一类型可拆成多个 `impl`（编译等价于一个大块），便于按功能分组：

```rust
impl Rectangle {
    fn area(&self) -> u32 { /* 只读 */ }
    fn can_hold(&self, other: &Rectangle) -> bool { /* ... */ }
}

impl Rectangle {
    fn set_size(&mut self, w: u32, h: u32) { /* 修改 */ }
}

impl Rectangle {
    fn new(w: u32, h: u32) -> Self { /* 关联函数 */ }
    fn square(size: u32) -> Self { /* ... */ }
}
```

## 方法 vs 关联函数（对照 Python）

| Rust | 调用 | 近似 Python |
|------|------|-------------|
| **`fn foo(&self)`** 方法 | `user.greet()` | 实例方法（`def foo(self)`） |
| **`fn new(...) -> Self`** 关联函数 | `User::new(...)` | 像 `@classmethod` / 工厂，但**无隐式 `cls`** |
| 无 `self` 的关联函数 | `Type::func()` | 像 `@staticmethod`（纯函数挂在类型上） |

```rust
struct User { name: String }

impl User {
    fn greet(&self) {
        println!("Hi, {}", self.name);
    }
    fn new(name: String) -> Self {
        User { name }
    }
}

let u = User::new("aa".into());
u.greet();
```

### 表象像，机制不同

1. **`self` 是显式第一个参数**（`&self` = `self: &Self`），不是 Python 那种编译器/解释器隐式塞进来的 `self`。
2. **三种接收者 = 三种所有权/借用**（Python 没有这套）：

| 签名 | 含义 |
|------|------|
| `&self` | 不可变借用，只读 |
| `&mut self` | 可变借用，可改字段 |
| `self` | 拿走实例所有权（消费 `self`） |

3. **区分靠签名，不靠装饰器**：有 `self` → 方法；`impl` 里无 `self` → 关联函数。Python 用 `@classmethod` / `@staticmethod` 标记。

> **用法相似**：带 `self` 要实例，不带用 `Type::` 调。  
> **本质不同**：Rust `self` 显式 + 绑定借用/所有权规则；Python `self`/`cls` 隐式 + 运行时动态。

## 记忆卡片（压缩版）

| 形式 | 首参 | 调用 | Python 近似 |
|------|------|------|-------------|
| 实例方法 | `&self` | `r.area()` | `def f(self)` |
| 实例方法 | `&mut self` | `mut r.set(...)` | 改属性的方法 |
| 实例方法 | `self` | `r.into_data()` | 无（消费原对象） |
| 关联函数 | 无 self | `Rect::new()` | `@staticmethod` |

1. **无 `->`**：`.` 调用；self 自动 `&` / `&mut`；其它参数手动 `&`。
2. **关联函数**：`impl` 里无 self → `Type::func()`。
3. **多 `impl`**：同类型可拆分，仅组织代码。
