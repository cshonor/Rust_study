# 4.3.4 DST 与 Sized（动态大小类型）

← 主笔记：[4.3 切片](./4.3-切片slice.md) · struct 栈/堆：[5.1.5](../05-structs/5.1.5-结构体变量实体与引用.md) · [4.3.3 str/&str](./4.3.3-str与str类型.md)

**一句话**：**DST 本体不能直接当栈变量**；栈上只能放 **指针 / 胖指针 / `Box`**，本体在堆、静态区或别处。

---

## 1. Sized vs DST

| | **Sized** | **DST** |
|--|-----------|---------|
| 编译期大小 | 固定 | **未知**（运行时才定） |
| 例子 | `u32`、`Rect`、`String` | `str`、`[T]`、`dyn Trait` |
| 裸变量 | `let r = Rect { ... };` ✅ | `let s: str;` ❌ |

```rust
struct Rect { w: u32, h: u32 } // Sized，8 字节
let r = Rect { w: 1, h: 2 };   // 栈上整块实体

// let x: str = ...;            // ❌ 不能单独绑定 DST
let s: &str = "abc";            // ✅ 栈上胖指针，指向 .rodata
```

---

## 2. DST 的三种包装（栈里是指针）

### 胖指针 `&str` / `&[T]`（ptr + len，常见 16 字节）

```rust
let s: &str = "abc123";           // 本体：.rodata；栈：胖指针
let st = String::from("abc");
let s2: &str = &st[..];           // 本体：堆；栈：胖指针

let buf = vec![1u8, 2, 3];
let sl: &[u8] = &buf;              // 本体：堆 Vec；栈：胖指针
```

### `Box<str>` / `Box<[u8]>`（栈上窄指针，本体在堆）

```rust
let b: Box<str> = Box::from("hello"); // 堆上 str 字节，栈上 Box 指针
```

### `Box<dyn Trait>`（trait 对象，同样 DST + 堆）

数据本体在堆（或堆上 vtable 布局），栈上指针。

---

## 3. 与结构体字段对上

**`User { name: String }`** — `String` 是 **Sized**（栈上 ptr+len+cap），故 **`User` 整体 Sized**，变量 `u` 在栈；**字符在堆**。

**`User { name: &str }`** — `&str` 是 **Sized 的胖指针**，`User` 仍固定大小；**`name` 只是栈上 16 字节指针**，指向静态区或堆。

→ [5.1.5 三种布局](../05-structs/5.1.5-结构体变量实体与引用.md)

## 4. 对比小代码

```rust
// Sized：栈实体
let r = Rect { w: 1, h: 2 };

// DST：只能包一层
let lit: &str = "hi";                    // 胖指针 → .rodata
let owned = String::from("hi");
let borrow: &str = &owned;               // 胖指针 → 堆
let on_heap: Box<str> = Box::from("hi"); // 窄指针 → 堆上 str 本体

// 函数返回
fn f() -> &str { "ok" }       // ✅ 借 static
fn g() -> String { owned }    // ✅ Sized，拥堆
// fn bad() -> str { ... }     // ❌ DST 不能作返回值类型
```

---

## 5. 记忆卡片

| 要点 | 一句 |
|------|------|
| Sized | `let x = T` 栈上可放**实体**（含 `String` 壳） |
| DST | 不能 `let x: str`；要 `&str` / `Box<str>` 等 |
| `&str` / `&[T]` | 栈**胖指针**；数据在堆或静态区 |
| `Box<str>` | 栈**窄指针**；str 本体在堆 |

---

## 小结

1. **Sized**：变量可以是整块实体（全栈字段时全在栈；含 `String` 时壳栈、字堆）。  
2. **DST**：栈上**只有指针**；`str`/`[T]` 本体在堆 / `.rodata` / 借用处。  
3. 前后说法一致：纯 `Rect` 全栈；`User{String}` 壳栈字堆；`&str` 字段 = 栈上胖指针。
