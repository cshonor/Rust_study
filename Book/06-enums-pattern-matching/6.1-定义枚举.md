## 一、枚举核心概念

**`enum`：一个变量同一时刻只能是枚举中某一个成员（变体 variant）** — 用来归类有限的可选种类（IP 只有 V4/V6、消息只有退出/移动/写入/改色）。

### 1. 最简无数据枚举（单元变体）

```rust
enum IpAddrKind {
    V4,
    V6,
}

let four = IpAddrKind::V4;
let six = IpAddrKind::V6;
```

成员在**枚举命名空间**内，用 **`::`** 构造/访问。

### 2. 老式写法：枚举 + 结构体拆分（不推荐）

```rust
enum IpAddrKind { V4, V6 }

struct IpAddr {
    kind: IpAddrKind,
    address: String,
}

let ip = IpAddr {
    kind: IpAddrKind::V4,
    address: String::from("127.0.0.1"),
};
```

> 缺点：类型与数据拆开，啰嗦 → **Rust 推荐把数据直接绑在变体上**。

## 二、四种变体格式（Rust 枚举灵魂）

**单个 enum 内，每个变体可以是不同形态** — 与 C/C++ 枚举最大区别：

```rust
enum Message {
    Quit,                        // ① 单元变体：无数据
    Move { x: i32, y: i32 },     // ② 结构体变体：命名字段
    Write(String),               // ③ 单元素元组变体
    ChangeColor(u8, u8, u8),     // ④ 多元素元组变体
}

let m1 = Message::Quit;
let m2 = Message::Move { x: 10, y: 20 };
let m3 = Message::Write(String::from("hello"));
let m4 = Message::ChangeColor(255, 0, 0);
```

### Message 四种变体完整详解（4 种形态全覆盖）

> **Quit / Move / Write / ChangeColor = 变体标签（区分种类）；括号/大括号里 = 该分支实际携带的值。**

| # | 变体 | 形态 | 构造示例 |
|---|------|------|----------|
| ① | **`Quit`** | 单元变体：无括号、无数据 | `Message::Quit` |
| ② | **`Move { x, y }`** | 结构体变体：大括号、命名字段 | `Message::Move { x: 10, y: 20 }` |
| ③ | **`Write(String)`** | 单元素元组变体：小括号、1 个类型 | `Message::Write(String::from("你好"))` |
| ④ | **`ChangeColor(u8,u8,u8)`** | 多元素元组变体：小括号、按顺序 | `Message::ChangeColor(255, 0, 0)` |

**① Quit** — 只是分类标记，不捆绑数据；适用：退出指令等无需附带信息的场景。

**② Move { x, y }** — 大括号 = 命名字段，和 struct 一样，靠字段名不靠顺序；可读性强，适合多参数。

**③ Write(String)** — `Write` 是标签，括号里字符串是正文数据。

**④ ChangeColor(r,g,b)** — 3 个 `u8` 按顺序对应 RGB，无字段名。

**IpAddr 同理**：`V4` / `V6` 是标签；`(127,0,0,1)` / `String` 是具体值。

配套 **`match` 取标签内的值** → [6.2 match](./6.2-match.md)：

```rust
match msg {
    Message::Quit => println!("退出"),
    Message::Move { x, y } => println!("移动坐标：{} {}", x, y),
    Message::Write(s) => println!("消息内容：{}", s),
    Message::ChangeColor(r, g, b) => println!("颜色：{} {} {}", r, g, b),
}
```

### IP 进阶：变体绑定不同类型

```rust
enum IpAddr {
    V4(u8, u8, u8, u8),
    V6(String),
}

let localhost = IpAddr::V4(127, 0, 0, 1);
```

### 元组变体 `()` 详解：括号里每一项都是独立类型

> **变体后面小括号 = 元组结构；括号里 N 个类型 = 这个变体必须携带 N 个对应类型的数据。**

#### `V4(u8, u8, u8, u8)` — 四个独立 `u8`

1. 四个 `u8` 对应 IPv4 四段：`192.168.1.1`
2. 实例化：`IpAddr::V4(192, 168, 1, 1)` — 四个数**按顺序**填入四个位置
3. 本质：**元组式绑定**，一个 V4 变体身上捆 4 个值

#### `V6(String)` — 单个 `String`

只存一串文本 IP：`IpAddr::V6(String::from("::1"))`

#### 括号里多种、多个不同类型

```rust
struct User { name: String }
struct Point { x: i32, y: i32 }

enum Test {
    Item1(u8, i32, String),  // 3 个不同类型，按顺序传
    Item2(User, Point),      // 2 个完整结构体
}

Test::Item1(1, 42, String::from("hi"));
Test::Item2(
    User { name: String::from("张三") },
    Point { x: 10, y: 20 },
);
```

| 变体 | 含义 |
|------|------|
| `Item1(a, b, c)` | 固定绑定 **3 个数据**：`u8` + `i32` + `String` |
| `Item2(user, point)` | 捆绑 **两个完整 struct** |

括号里可以是 **`u8`、`String`、自定义 struct** 等任意类型。

#### `()` 元组 vs `{}` 结构体变体

| | 小括号 `()` | 大括号 `{}` |
|---|-------------|-------------|
| 字段 | **无名**，靠**顺序**取值 | **有名**，靠**字段名**取值 |
| 例子 | `V4(u8, u8, u8, u8)` | `Move { x: i32, y: i32 }` |
| 构造 | `V4(192, 168, 1, 1)` | `Move { x: 10, y: 20 }` |

### 标准库 `std::net::IpAddr`（工业级写法）

变体直接嵌套现成结构体：

```rust
enum IpAddr {
    V4(Ipv4Addr),
    V6(Ipv6Addr),
}

use std::net::{IpAddr, Ipv4Addr};

let ip: IpAddr = IpAddr::V4(Ipv4Addr::new(127, 0, 0, 1));
// Ipv4Addr 自带 is_loopback()、parse() 等
```

## 三、枚举可以 `impl` 写方法（和 struct 一样）

```rust
impl Message {
    fn call(&self) {
        println!("执行消息逻辑");
    }
}

let msg = Message::Quit;
msg.call();
```

## 四、标准库 `Option<T>`（Rust 无 null）

### 源码（内置泛型枚举）

```rust
enum Option<T> {
    Some(T),
    None,
}
```

### 使用规则

1. **`Option<T>` 与 `T` 是两种完全不同类型**，不能混用；
2. Rust 强制处理 `None`，杜绝空指针式崩溃。

```rust
let num: Option<i32> = Some(5);
let empty: Option<i32> = None; // None 须能推断 T
```

### 常用方法（提前记）

| 方法 | 作用 |
|------|------|
| `is_some()` / `is_none()` | 判断 Some / None |
| `unwrap()` | 取内部值；**None 则 panic**（生产慎用） |
| `unwrap_or(default)` | None 返回默认值，Some 返回原值 |

## 五、下一节：`match` 匹配

枚举取值**常用 `match`**，按变体分支并解构内部数据：

```rust
match msg {
    Message::Quit => println!("退出"),
    Message::Move { x, y } => println!("坐标 {} {}", x, y),
    Message::Write(s) => println!("{}", s),
    Message::ChangeColor(r, g, b) => println!("rgb {} {} {}", r, g, b),
}
```

→ [6.2 match](./6.2-match.md)

## 精简口诀

1. 裸枚举：`enum 名 { A, B }`，`::` 取成员  
2. 带数据：无数据 / 结构体 `{}` / 单值 `()` / 多值 `()` 四类变体  
3. 枚举能 **`impl` 方法**  
4. **`Option`** = `Some(T)` / `None`，替代 null
