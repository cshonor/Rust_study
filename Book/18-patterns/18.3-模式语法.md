## 核心结论（先背）

模式可用于 `match`、`if let`、`while let`、`for`、`let`、函数参数等（→ [18.1](./18.1-所有可能会用到模式的位置.md)）。

| 语法 | 作用 |
|------|------|
| 字面量 | 匹配固定常量 |
| `A \| B` | 多模式，满足其一 |
| `a..=b` | 闭区间（**仅整数/字符**） |
| 解构 | 拆分结构体/枚举/元组/嵌套 |
| `_` | 忽略、**不绑定**、**不转移所有权** |
| `_var` | 绑定、**转移所有权**、消警告 |
| `..` | 忽略剩余（元组**只能一个** `..`） |
| `if 条件` | 匹配守卫 |
| `var @ 模式` | 校验 + 绑定二合一 |

---

## 一、字面量模式

直接匹配**固定常量**：整数、字符、字符串、布尔值等。

```rust
match num {
    1 => println!("1"),
    2 => println!("2"),
    _ => println!("其他"),
}

match c {
    'a' => println!("a"),
    'b' => println!("b"),
    _ => (),
}
```

---

## 二、`match` 中的变量与遮蔽

`match` 每个分支是**独立作用域**；模式中绑定的变量会**遮蔽**外层同名变量。

```rust
let y = 10;
let opt = Some(10);

match opt {
    Some(y) => println!("内部 y: {y}"),  // 新绑定，遮蔽外层 y
    None => (),
}
println!("外层 y: {y}");  // 仍是 10
```

**想引用外层变量** → 不能写同名绑定，须用**匹配守卫**（§七）或换名（如 `Some(n)`）。

---

## 三、多模式 `|`

`模式1 | 模式2 | 模式3` — 满足**任意一个**即可匹配。

```rust
match n {
    1 | 2 | 3 => println!("1/2/3"),
    4 | 5 => println!("4/5"),
    _ => println!("其他"),
}
```

---

## 四、范围模式 `..=`（闭区间）

`起始..=结束` — **闭区间**，包含两端。

| 支持 | 不支持 |
|------|--------|
| 整数、`char` | 浮点数、字符串 |

```rust
match age {
    0..=18 => println!("未成年"),
    19..=60 => println!("成年"),
    _ => println!("长者"),
}

match ch {
    'a'..='z' => println!("小写"),
    'A'..='Z' => println!("大写"),
    _ => (),
}
```

## 五、解构模式

### 5.1 元组

```rust
let (a, b) = (10, 20);

match t {
    (0, y) => println!("第一为 0，第二 {y}"),
    (x, 20) => println!("第二为 20，第一 {x}"),
    _ => (),
}
```

### 5.2 结构体

- 简写：`let Point { x, y } = p;`
- 重命名：`let Point { x: a, y: b } = p;`
- 字面量约束字段：`Point { x: 0, y }`

```rust
match p {
    Point { x: 0, y } => println!("x=0, y={y}"),
    Point { x, y: 0 } => println!("y=0, x={x}"),
    _ => (),
}
```

### 5.3 枚举

变体形状须与定义**完全一致**（单元 / 元组 / 结构体变体）。

```rust
match c {
    Color::Rgb(r, g, b) => println!("{r} {g} {b}"),
    Color::Hsv(h, s, v) => println!("{h} {s} {v}"),
    Color::Black => println!("黑"),
}
```

### 5.4 嵌套解构

```rust
match msg {
    Msg::ChangeColor(Color::Hsv(h, s, v)) => println!("{h} {s} {v}"),
    _ => (),
}
```

### 5.5 混合解构

```rust
let ((feet, inches), Point { x, y }) = ((3, 10), Point { x: 3, y: -10 });
```

## 六、忽略值：`_` / `_var` / `..`

### 6.1 `_` — 不绑定、不转移所有权

```rust
let s = Some(String::from("hello"));
if let Some(_) = s {
    println!("匹配");
}
println!("{s:?}");  // s 仍可用

fn calc(_: i32, b: i32) -> i32 { b * 2 }
```

### 6.2 `_x` — 绑定、转移所有权

```rust
let s = Some(String::from("hello"));
if let Some(_s) = s {  // String 被移入 _s
    println!("匹配");
}
// println!("{s:?}");  // E0382: borrow of moved value
```

| | `_` | `_var` |
|--|-----|--------|
| 绑定 | 否 | 是 |
| 所有权 | 不转移 | **转移** |
| 用途 | 通配/忽略 | 消未使用警告 |

### 6.3 `..` — 忽略剩余

```rust
let Person { name, .. } = p;       // 结构体：忽略其余字段
let (first, .., last) = t;         // 元组：忽略中间
// let (.., mid, ..) = t;           // E0425: 只能一个 ..
```

## 七、匹配守卫 `if`

写在模式后：`模式 if 条件 =>`

用途：

1. 同一种模式下追加布尔判断
2. **引用外层变量**（避免遮蔽）
3. 与 `\|` 组合 — 守卫对**整个多模式组**生效

```rust
let y = 5;
match opt {
    Some(n) if n == y => println!("等于外层 y"),  // 用守卫，不用 Some(y)
    Some(n) => println!("{n}"),
    None => (),
}

match num {
    1 | 2 | 3 if flag => println!("1/2/3 且 flag"),  // 守卫作用于整组
    _ => (),
}
```

---

## 八、`@` 绑定

`变量 @ 模式` — **先校验模式**，**同时绑定值**。

```rust
match n {
    id @ 3..=7 => println!("{id} 在 3~7"),
    _ => (),
}

match s {
    Status::Code(val @ 200..=299) => println!("正常: {val}"),
    _ => (),
}
```

## 九、高频易错点 & 编译报错

| # | 易错 | 说明 / 报错 |
|---|------|-------------|
| 1 | **变量遮蔽** | `Some(y)` 内 `y` 是新变量；引用外层用守卫 `if n == y` |
| 2 | **`_` vs `_x`** | `_` 不转移所有权；`_x` 会 move → **E0382** |
| 3 | **两个 `..`** | 元组模式只能一个 `..` → **E0425** |
| 4 | **浮点范围** | `1.0..=5.0` 非法；仅整数/`char` |
| 5 | **守卫 + `\|`** | `1 \| 2 \| 3 if flag` 守卫作用于**整组** |
| 6 | **解构形状** | 枚举变体须与定义一致，否则 **E0023** |
| 7 | **`match` 穷尽** | 未覆盖所有情况 → **E0004**（用 `_` 兜底） |
