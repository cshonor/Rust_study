## 定义与实现

```rust
pub trait Summary {
    fn summarize(&self) -> String;
}
```

孤儿规则：只能为**本 crate 的 trait** 或**本 crate 的类型**实现 trait。

默认实现见 demo `Tweet` 使用 `summarize` 默认版。

---

## 一、函数参数：两种传 Trait 的写法

### 写法 1：`impl Trait`（语法糖）

→ 全解：[10.2.2 impl Trait](./10.2.2-impl-Trait全解.md)

```rust
pub fn notify(item: &impl Summary) {
    println!("Breaking news! {}", item.summarize());
}
```

编译器自动推导隐藏泛型，**每个 `impl Trait` 参数可独立是不同类型**。

### 写法 2：泛型 `T` + trait bound

```rust
pub fn notify<T: Summary>(item: &T) { /* ... */ }
```

**可复用同一个 `T`** — 多参数强制同类型：

```rust
fn notify2<T: Summary>(a: &T, b: &T) {} // a、b 必须同类型 T

// impl 版：a、b 可以是不同具体类型，各自实现 Summary
fn notify2_impl(a: &impl Summary, b: &impl Summary) {}
```

### 多约束：`TraitA + TraitB`

```rust
pub fn notify(item: &impl Summary + Display) {}
pub fn notify<T: Summary + Display>(item: &T) {}
```

`+` = **并且**，同时实现多个 trait。

### 约束太多 → `where` 分句

```rust
fn f<T, U>(t: T, u: U)
where
    T: Display + Clone,
    U: Clone + Debug,
{ }
```

## 二、返回值：`-> impl Trait`

```rust
fn returns_summarizable() -> impl Summary {
    Post {}
}
```

### 硬性限制（易错）

**不能分支返回两种不同具体类型**：

```rust
// ❌ if 返回 Post，else 返回 News — 编译失败
fn bad(flag: bool) -> impl Summary {
    if flag { Post {} } else { News {} }
}
```

→ 完整对比 · 异构 Vec · 口诀：[10.2.4 impl vs dyn](./10.2.4-impl与dyn-Trait对比.md) · [17.2](../17-oop/17.2-为使用不同类型的值而设计的trait对象.md)

## 三、`largest`：三种 trait bound 方案

| 方案 | 签名 | 特点 |
|------|------|------|
| **Copy** | `T: PartialOrd + Copy` → `T` | 简单，大类型拷贝贵 |
| **Clone** | `T: PartialOrd + Clone` → `T` | 堆数据可，有 clone 开销 |
| **引用（推荐）** | `T: PartialOrd` → `&T` | 只借用，工业常用 |

```rust
fn largest_copy<T: PartialOrd + Copy>(list: &[T]) -> T { /* ... */ }
fn largest_clone<T: PartialOrd + Clone>(list: &[T]) -> T { /* list[0].clone() ... */ }
fn largest_ref<T: PartialOrd>(list: &[T]) -> &T { /* ... */ }
```

- **`PartialOrd`**：支持 `>` 比较

## 四、条件 impl + Blanket impl

### 条件 impl：满足 bound 才有方法

```rust
impl<T: Display + PartialOrd> Pair<T> {
    fn cmp_display(&self) { /* ... */ }
}
```

`T` 不满足 → 调用 `.cmp_display()` **编译报错**。

### Blanket impl：符合条件**全部类型**自动获得 trait

标准库经典：

```rust
impl<T: Display> ToString for T { /* ... */ }
```

```rust
let s: i32 = 10;
s.to_string(); // i32 有 Display → 自动有 ToString
```
