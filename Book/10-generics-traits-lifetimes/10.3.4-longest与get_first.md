## 一、`get_first`

```rust
fn get_first<'a>(s: &'a str) -> &'a str {
    &s[0..1]
}
```

单入参 + 返回引用 → 可省略 `'a`（见 [10.3.5 省略规则](./10.3.5-显式与隐式生命周期.md)）。

---

## 二、`longest`

```rust
fn longest<'a>(x: &'a str, y: &'a str) -> &'a str {
    if x.len() > y.len() { x } else { y }
}
```

- `x`、`y`、返回值**共用 `'a`**；
- 每次调用：`'a` = 两实参中**较短**的作用域；
- **不是**「一长一短传参就报错」→ 看 **`res` 是否跨红线**。

| 场景 | 写法要点 | 结果 |
|------|----------|------|
| A | `res` 在内层定义并使用 | ✅ |
| B | `res` 在外层或出块后用 | ❌ |

```rust
// ✅ A
// fn main() {
//     let outer = String::from("aaa");
//     {
//         let inner = String::from("bb");
//         let res = longest(&outer, &inner);
//         println!("{}", res);
//     }
// }

// ❌ B
// fn main() {
//     let outer = String::from("aaa");
//     let res: &str;
//     {
//         let inner = String::from("bb");
//         res = longest(&outer, &inner);
//     }
//     println!("{}", res);
// }
```

## 三、同作用域两实参

```rust
fn main() {
    let s1 = String::from("abc");
    let s2 = String::from("abcd");
    let res = longest(&s1, &s2);
    println!("{}", res);
}
```

---

## 四、`'a` / `'b` 分组

```rust
fn two_str<'a, 'b>(x: &'a str, y: &'b str) {
    println!("{} {}", x, y);
}
```

一长一短、不同标签 → 内层调用 ✅（见 [10.3.2 §三](./10.3.2-同a约束与红线.md#三三种场景)）。
