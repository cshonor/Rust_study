# 10.2.1 Trait 语法速查

← [10.2 trait](./10.2-trait.md) · demo：`10.2-traits-demo/`

---

## 极简总结表

| 写法 | 作用 | 特点 |
|------|------|------|
| `fn(x: impl Trait)` | 参数语法糖 | 每参数可不同类型；**不能**强制多参数同类型 |
| `fn<T: Trait>(x: T)` | 泛型约束 | 多参数可**共用同一个 T** |
| `fn(x: impl A + B)` / `T: A + B` | 多约束 | `+` = 同时实现 |
| `where T: A + B` | 约束拆分 | 多泛型、长约束时整洁 |
| `-> impl Trait` | 返回 impl | **只能单一固定具体类型** |
| `impl<T: Bound> Struct<T> { … }` | 条件方法 | 满足 bound 才有方法 |
| `impl<T: Bound> Trait for T` | **Blanket impl** | 符合条件**全类型**自动拥有 trait |

---

## `impl Trait` vs `<T: Trait>` 选型

| 需求 | 选 |
|------|-----|
| 单参数、简单约束 | `impl Trait` |
| 两参数必须**同类型** | `<T: Trait>(a: T, b: T)` |
| 两参数**可以不同类型** | `(a: impl Trait, b: impl Trait)` |
| 约束很长 / 多泛型 | `where` |

---

## 易错点

1. **`-> impl Trait` 分支返回不同类型** → 用 `Box<dyn Trait>`
2. **`|x| x + 1` 未调用**无法推断 — 闭包见 [13.1.1](../13-iterators-closures/13.1.1-闭包四段式语法.md)
3. **孤儿规则** — 不能给外部类型 impl 外部 trait
4. **`largest` 大类型** — 优先 `&T`，别强 Copy

## 自测小题（口答）

1. `fn f(a: impl Summary, b: impl Summary)` — a 和 b 能否是 `Tweet` 和 `NewsArticle`？**能**（不同类型均可）。
2. `fn f<T: Summary>(a: T, b: T)` — 同上？**不能**（必须同类型 T）。
3. `fn g() -> impl Summary { if true { Post{} } else { News{} } }` — 编译？**不过**（两种具体类型）。
4. `10i32.to_string()` 为何合法？**Blanket impl**：`Display` → `ToString`。
