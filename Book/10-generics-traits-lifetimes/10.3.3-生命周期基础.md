## 一、变量寿命 vs `'a` 标注

| | 变量寿命 | `'a` 标注 |
|---|----------|-----------|
| 是什么 | 从声明到 `{}` 结束 | 编译期给**引用**分的组 |
| 管什么 | 变量何时 drop | **引用**能安全用到何时 |

```rust
fn main() {
    let num = 10;
    let r = &num;
} // num、r 一起结束
```

> `'a`：**同组引用的使用边界**在推断区间内须与数据存活一致；**不改变**变量 drop 时机。

---

## 二、标注 vs 作用域

| 规则 | 管什么 |
|------|--------|
| `'a` / `'static` | **被指向的数据**能活多久 |
| `{}` | **引用变量**何时销毁 |

```rust
fn main() {
    let s: &'static str = "test";
} // 变量 s 消失；"test" 仍在只读段
```

## 三、`&'static str`

| 部分 | 含义 |
|------|------|
| `&` | 借用 |
| `'static` | 数据与程序同寿 |
| `str` | 被引用类型 |

```rust
let s: &'static str = "abc";

fn get_str() -> &'static str {
    "hello" // ✅
}

// fn bad() -> &str {
//     let st = String::from("hi");
//     &st // ❌
// }
```

## 四、`'a` 约束什么

```rust
fn borrow<'a>(data: &'a i32) -> &'a i32 {
    data
}
```

| 部分 | 含义 |
|------|------|
| `data: &'a i32` | 入参引用绑定 `'a` |
| `-> &'a i32` | 返回值与入参**同 `'a`** |

```rust
// fn bad<'a>() -> &'a i32 {
//     let tmp = 20;
//     &tmp // ❌
// }
```

**三条规则**：

1. `'a` = **时长分组**，不是变量绑定；
2. 同 `'a` = 在**本次推断的 `'a` 区间内**，被借数据都须存活；
3. 整组上限 = **最短**数据源作用域（红线）→ [10.3.2](./10.3.2-同a约束与红线.md)
