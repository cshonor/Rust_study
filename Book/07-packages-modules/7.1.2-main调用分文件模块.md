# 7.1.2 main 如何调用 `a.rs` 里的函数

← 主笔记：[7.1](./7.1-包和crate.md) · 前置：[7.1.1 bin/lib](./7.1.1-二进制与库crate.md) · demo：[7.1-packages-crates-demo](./7.1-packages-crates-demo/) · 分文件 → [7.5](./7.5-将模块分割进不同文件.md)

---

## 口诀

**有 lib 过 lib，无 lib 直接 mod 在 main。**

---

## 一、两种结构对照

| | 结构 1（推荐） | 结构 2（纯 bin） |
|---|----------------|------------------|
| 文件 | `lib.rs` + `main.rs` + `a.rs` | `main.rs` + `a.rs` |
| 注册模块 | **`lib.rs`** 里 `mod a;` | **`main.rs`** 里 `mod a;` |
| 导出 | `lib.rs` 里 `pub use a::hello;` | 不需要 |
| main 调用 | `use 包名::hello;` | `a::hello();` |
| demo | `via_lib/` | `via_main_only/` |

---

## 二、结构 1：`lib.rs` 中转（工业标准）

```
proj/
├ Cargo.toml
└ src/
   ├ a.rs
   ├ lib.rs
   └ main.rs
```

| 步骤 | 文件 | 代码 |
|------|------|------|
| ① 实现 | `a.rs` | `pub fn hello() { … }` |
| ② 注册+导出 | `lib.rs` | `mod a;` · `pub use a::hello;` |
| ③ 调用 | `main.rs` | `use proj::hello;` · `hello();` |

```rust
// a.rs
pub fn hello() {
    println!("来自 a.rs");
}

// lib.rs
mod a;
pub use a::hello;

// main.rs
use proj::hello;
fn main() { hello(); }
```

> 也可 `pub mod a;`，则 main 写 `use proj::a::hello;`。

**原因**：`main.rs` 是 bin crate root，**不能**直接 `mod` lib 侧的模块树；必须经 lib crate 的 `pub` 导出。

---

## 三、结构 2：纯 binary，main 直接注册

```
proj/
├ Cargo.toml
└ src/
   ├ a.rs
   └ main.rs
```

```rust
// main.rs — bin crate root
mod a;

fn main() {
    a::hello();
}
```

无 `lib.rs` → 只有 **一个 bin crate**，`mod a;` 写在 `main.rs` 即可。

---

## 四、三条铁则

| # | 规则 |
|---|------|
| 1 | **有 `lib.rs`** → lib 里 `mod` + `pub` 导出，main 从**包名**导入 |
| 2 | **无 `lib.rs`** → main 里 `mod a;`，路径 `a::hello()` |
| 3 | `a.rs` 函数无 `pub` → 无论哪种结构都访问不了 |

---

## 五、与 crate 的关系

```
结构1（两个 crate）                 结构2（一个 crate）
┌──────── bin: main.rs ────┐       ┌── bin: main.rs ──┐
│  use 包名::hello         │       │  mod a;          │
└──────────┬───────────────┘       │  a::hello()      │
           │ use                   └────────┬─────────┘
┌──────── lib: lib.rs ─────┐                │
│  mod a; pub use a::hello │       ┌────────▼─────────┐
└──────────┬───────────────┘       │  a.rs            │
           │                       └──────────────────┘
┌──────────▼───────────────┐
│  a.rs                    │
└──────────────────────────┘
```

`mod a;` 对应 `src/a.rs` 的文件规则 → [7.5](./7.5-将模块分割进不同文件.md)
