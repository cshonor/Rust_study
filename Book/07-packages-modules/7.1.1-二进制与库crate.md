## 一、Crate 二选一（用代码看）

> Crate 只有两类：**binary crate**（有 `main`，产出 exe）或 **library crate**（无 `main`，产出库，供 `use`）。

### 示例 1：Binary crate — 只有 `main.rs`

```
proj/
├ Cargo.toml
└ src/main.rs
```

```rust
// bin crate 必须有 fn main()
fn main() {
    println!("我是二进制程序，运行生成 exe");
}
```

`cargo build` → `target/` 生成 exe；`cargo run` 直接运行。

demo → `7.1-packages-crates-demo/only_bin/`

### 示例 2：Library crate — 只有 `lib.rs`

```
mylib/
├ Cargo.toml
└ src/lib.rs
```

```rust
// 没有 main = lib crate
pub fn add(a: i32, b: i32) -> i32 {
    a + b
}
```

`cargo build` → 编译成库文件；**没有 `main`，不能 `cargo run`**，只能被别的项目 `use`。

demo → `7.1-packages-crates-demo/only_lib/`

## 二、`lib.rs` = 库 crate 对外 API 的出口（重点）

**`lib.rs` 是整个 lib crate 的根模块 `crate::`。**  
所有想给外部 `use` 的代码，必须在模块树里用 **`pub` 导出**；不加 `pub` = 私有，外面用不了。

### 最常用结构：同一 Package 里 `lib.rs` + `main.rs` + 子模块

```
myproj/
├ Cargo.toml
└ src/
   ├ lib.rs      // 库 crate root：统一对外导出
   ├ main.rs     // 二进制 crate root：use 本包库
   └ math/
       └ mod.rs
```

#### `src/math/mod.rs`

```rust
pub fn add(a: i32, b: i32) -> i32 {
    a + b
}

fn sub(a: i32, b: i32) -> i32 {
    a - b // 无 pub → 仅 math 模块内部可调用
}
```

#### `src/lib.rs`（核心：收拢 + 导出）

```rust
pub mod math;

// 可选：再导出，外部路径更短
// pub use math::add;
```

| 写法 | 含义 |
|------|------|
| `mod math;` | 引入模块，但**私有** → 同包 `main`、外部项目都 `use` 不到 |
| `pub mod math;` | 模块**公开** → `use 包名::math::add` 可用 |

#### `src/main.rs`

```rust
use myproj::math::add; // 包名 = Cargo.toml 的 name

fn main() {
    println!("{}", add(1, 2));
}
```

demo → `7.1-packages-crates-demo/bin_plus_lib/`（含 `math` 子模块）

## 三、三种 `pub` 场景（速查）

| 写法 | 谁能用 |
|------|--------|
| `mod math;` | 仅 **lib crate 内部**（`lib.rs` 及其子模块） |
| `pub mod math;` | 同包 **bin crate**、**外部依赖** 均可 `use 包名::math` |
| 模块内 `pub fn add()` | 函数公开；无 `pub` 的函数仅模块内部 |

> bin crate（`main.rs`）和 lib crate（`lib.rs`）是**两个独立 crate**；`main` 只能访问 lib 里 **`pub` 出来的项**，不能访问 lib 内部的私有 `mod`。

## 四、反面例子：去掉 `pub` 编译失败

把 `lib.rs` 里的 `pub mod math;` 改成 `mod math;`，再在 `main.rs` 里 `use 包名::math::add`：

```text
error[E0603]: module `math` is private
```

验证：

```bash
cd Book/07-packages-modules/7.1-packages-crates-demo/bin_plus_lib/compile_fail/private_mod
cargo check   # 预期失败
```

## 五、外部项目引用本库

别的项目 `Cargo.toml`：

```toml
[dependencies]
myproj = { path = "../myproj" }
```

新项目代码：

```rust
use myproj::math::add;
```

**能跨项目导入，全靠 `lib.rs` 里的 `pub` 导出链**（`pub mod` → 模块内 `pub fn`）。

路径依赖与 workspace 细节 → 后续 crate 发布章节；模块树与路径 → [7.3](./7.3-路径用于引用模块树中的项.md)。
