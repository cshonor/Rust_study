# 7.1.8 Package（包）vs Module（模块）

← 主笔记：[7.1](./7.1-包和crate.md) · 模块语法 → [7.2](./7.2-定义模块来控制作用域与私有性.md) · 范例：[7.1.3 web 目录](./7.1.3-Actix-web项目目录范例.md)

---

## 一句话核心

| | Package | Module |
|---|---------|--------|
| **是什么** | Cargo **项目管理单位**（文件夹 + `Cargo.toml`） | Rust **源码里的代码组织**（命名空间 + 可见性） |
| **编译** | 产出 **crate**（库 / exe） | **不能单独编译**，依附某个 crate |

---

## ① Package（包，工程维度）

1. **标志**：根目录有 `Cargo.toml`
2. **一个 Package 产出**：
   - 最多 **1 个 Lib Crate**（`src/lib.rs`）
   - 若干 **Bin Crate**（`main.rs`、`src/bin/*.rs` → exe）
3. **作用**：依赖、版本、`Cargo.lock`、整个项目构建配置

> 例：`market_core/` 整个文件夹 = **1 个 Package**

→ 细则：[7.1](./7.1-包和crate.md) · [7.1.6 一 Package 一 lib.rs](./7.1.6-一个Package一个lib.rs.md)

---

## ② Module（模块，源码语法维度）

1. **标志**：`mod xxx;`、`xxx/mod.rs`、花括号 `mod { … }`
2. **永远隶属于某一个 Crate**，不能脱离 crate 独立存在
3. **作用**：拆分代码、`pub`/私有、避免命名冲突

> 例：`src/domain/mod.rs` 里 `pub mod email;` → `domain`、`email` 都是 **module**，装在当前 **lib crate** 里

→ 语法与私有性：[7.2](./7.2-定义模块来控制作用域与私有性.md) · 分文件：[7.5](./7.5-将模块分割进不同文件.md)

---

## 层级关系（从上往下）

```
Package（项目文件夹 + Cargo.toml）
└ Crate（编译单元：lib 或 bin）
   ├ mod A
   │   └ mod B
   └ mod C
```

- **1 个 Package ≠ 1 个 module**
- **1 个 crate 里可以有成百上千个 module**

```
Package ≠ Crate ≠ Module

Package  ⊃  Crate(s)  ⊃  Module(s)
```

---

## 最关键对比表

| | Package | Module |
|---|---------|--------|
| **管控方** | Cargo | Rust 语言 |
| **存在形式** | 磁盘目录 + `Cargo.toml` | `.rs` 源码逻辑分组 |
| **能否单独编译** | ✅ `cargo build` | ❌ 随 crate 一起编 |
| **对外引用** | `Cargo.toml` `[dependencies]` | `use xxx::yyy;`（crate 内 / 跨 crate 经 `pub`） |

---

## 经典实例

```
web/                    ← Package
├ Cargo.toml
└ src/
   ├ lib.rs             ← lib **Crate** 入口
   ├ domain/mod.rs       ← **Module** domain
   └ routes/mod.rs       ← **Module** routes
```

| 名字 | 层级 |
|------|------|
| `web/` | **Package** |
| `lib.rs` 整棵树 | **1 个 lib Crate** |
| `domain`、`routes` | **Module**（同 crate 内） |

## 两个需求怎么选

| 需求 | 做法 |
|------|------|
| 多个**独立库**（多个 `.rlib`） | 多个 **Package** + Workspace → [14.3](../14-cargo-crates/14.3-Cargo工作空间.md) |
| 代码**内部分组** | 同一 crate 里建多个 **module** → [7.2](../07-packages-modules/7.2-定义模块来控制作用域与私有性.md) · 总览 → [7.1.8 Package vs Module](../07-packages-modules/7.1.8-Package与Module.md) |

### 口诀

**要多盒 → 多 Package；要大盒里分格 → mod 模块。**

## 与 Crate 的三层对照（复习）

| 层级 | 工具/语法 | 比喻 |
|------|-----------|------|
| **Package** | Cargo | 项目文件夹 |
| **Crate** | `rustc` 编译单元 | 库箱 / exe 箱 |
| **Module** | `mod` / `pub` | 箱内格子 |
