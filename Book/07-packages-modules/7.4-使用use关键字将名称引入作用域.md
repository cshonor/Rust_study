## 一、`use` 核心作用

**`use` = 把完整路径引入当前作用域，生成路径别名，后续简写调用** —— 解决 `crate::xxx::yyy::func()` 全路径冗长的问题。

```rust
// 原始全路径
crate::front_of_house::hosting::add_to_waitlist();

// use 简写
use crate::front_of_house::hosting;
hosting::add_to_waitlist();
```

| 要点 | 说明 |
|------|------|
| **`use` 只改名字可见性** | 不编译代码、不引入文件；文件进模块树靠 **`mod`** |
| **权限** | 路径上任意节点非 `pub`，**无法 `use` 导入** |

---

## 二、Rust 惯用 `use` 编码规范

### 1. 函数：导入父模块，保留 `模块::函数()`

```rust
// ✅ 推荐：一眼看出不是本地函数
use crate::front_of_house::hosting;
hosting::add_to_waitlist();

// ❌ 不推荐：add_to_waitlist() 分不清来源
use crate::front_of_house::hosting::add_to_waitlist;
add_to_waitlist();
```

### 2. 结构体 / 枚举 / 类型：直接导入类型名

```rust
use std::collections::HashMap;
let map = HashMap::new();
```

---

## 三、同名冲突：`as` 别名

场景：不同模块同名类型（最常见 `std::io::Result` vs `std::fmt::Result`）。

**方案 1：导入父模块区分（优先）**

```rust
use std::fmt;
use std::io;

fn a() -> fmt::Result { Ok(()) }
fn b() -> io::Result<()> { Ok(()) }
```

**方案 2：`as` 自定义别名**

```rust
use std::io::Result as IoResult;
fn test() -> IoResult<()> { Ok(()) }
```

## 四、`pub use`：重导出（re-export，企业级 lib 必备）

| | 作用域 |
|---|--------|
| 普通 `use` | **仅当前模块**内可用 |
| **`pub use`** | 对外暴露路径，**外部 crate 可直接 `use`** |

```rust
// lib.rs 内部
pub use crate::front_of_house::hosting;

// 外部依赖本包时，不必知道 front_of_house 内部结构
use your_crate::hosting;
hosting::add_to_waitlist();
```

**跨 Package（A 调 B）**：B 的 `lib.rs` 用 `pub use` 导出 API → A 的 `Cargo.toml` 依赖 B → A 里 `use b_crate::xxx`。  
→ 完整链路 [7.3.1](./7.3.1-跨Package路径与Workspace依赖.md) · demo `7.3-cross-package-paths-demo/`

## 五、嵌套路径合并导入

同前缀路径合并；`{ self, … }` 中 **`self` = 父模块自身**：

```rust
// 拆分
use std::cmp::Ordering;
use std::io;
use std::io::Write;

// 合并
use std::{cmp::Ordering, io};
use std::io::{self, Write}; // self = std::io 模块本身
```

## 六、glob 通配符 `*`：全量导入公有项

```rust
use std::collections::*;
```

| | |
|---|---|
| **适用** | 单元测试、脚本、小作用域块 |
| **禁止** | 正式业务代码 — 易冲突、难溯源 |

demo 在 `main.rs` 里用 `{ … }` 块演示 glob，限制作用域。

## 七、跨 Package / crates.io 第三方

1. **`Cargo.toml` 依赖 Package**（不是 Crate）：

```toml
[dependencies]
serde = "1.0"
sqlx = "0.7"
```

2. **代码 `use` 对方 Lib-Crate 路径**：

```rust
use serde::Serialize;
use sqlx::postgres::PgPool;
```

## 八、易混关键字对照

| 关键字 | 作用 |
|--------|------|
| **`mod`** | 定义模块、关联 `.rs` 源码文件 |
| **`pub`** | 开启公有访问权限 |
| **`use`** | 引入已有路径、简写名称（仅当前模块） |
| **`pub use`** | 对外重导出 API（`lib.rs` 标配） |
