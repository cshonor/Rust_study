# 8.1 vector 用来储存一系列的值（Vec<T>）

对应 demo 项目：`Book/08-collections/8.1-vector-demo/`

`Vec<T>`（vector）允许在一段连续内存中存放多个值，**元素类型必须相同**，适合存“列表”数据。

---

## 新建 vector

### 空 vector

```rust
let v: Vec<i32> = Vec::new();
```

因为没有插入值，编译器无法推断 `T`，所以常需要类型标注。

### 用初值创建（vec! 宏）

```rust
let v = vec![1, 2, 3]; // Vec<i32>
```

---

## 更新 vector

向尾部追加元素：`push`。要修改变量必须 `mut`：

```rust
let mut v = Vec::new();
v.push(5);
v.push(6);
v.push(7);
v.push(8);
```

---

## 离开作用域会丢弃 vector 及其元素

```rust
{
    let v = vec![1, 2, 3, 4];
    // 使用 v
} // v 离开作用域并被丢弃
```

---

## 读取元素：索引 vs get

```rust
let v = vec![1, 2, 3, 4, 5];

let third: &i32 = &v[2];

match v.get(2) {
    Some(third) => println!("The third element is {}", third),
    None => println!("There is no third element."),
}
```

差异：
- `&v[index]`：越界会 **panic**
- `v.get(index)`：返回 `Option<&T>`，越界返回 `None`

---

## 借用规则：引用存在时不能 push

```rust
let mut v = vec![1, 2, 3, 4, 5];
let first = &v[0];
v.push(6); // ❌ 错误：同时存在不可变借用和可变借用
println!("{}", first);
```

原因：`push` 可能导致 reallocate（搬家），已有引用会悬垂；借用检查器在编译期阻止这种情况。

## 遍历

### 不可变遍历

```rust
let v = vec![100, 32, 57];
for i in &v {
    println!("{}", i);
}
```

### 可变遍历并修改

```rust
let mut v = vec![100, 32, 57];
for i in &mut v {
    *i += 50;
}
```

`*i` 是解引用（详见第 15 章）。

## 用枚举在 vector 中存多种类型

vector 只能存同一种类型；要混合存多类型值，可以用枚举把不同类型包装到同一个 enum 中：

```rust
enum SpreadsheetCell {
    Int(i32),
    Float(f64),
    Text(String),
}

let row = vec![
    SpreadsheetCell::Int(3),
    SpreadsheetCell::Text(String::from("blue")),
    SpreadsheetCell::Float(10.12),
];
```

如果无法在编译期确定类型集合，可考虑 trait 对象（第 17 章）。
