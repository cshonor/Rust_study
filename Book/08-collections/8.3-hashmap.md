## 一、基础前置

```rust
use std::collections::HashMap; // 不在 prelude，须手动导入
```

| 要点 | 说明 |
|------|------|
| **作用** | 键 `K` → 值 `V`，字典/哈希表（词频、配置、分数） |
| **存储** | 键值**全在堆**；同 Map 内所有 K 同类型、所有 V 同类型 |
| **键约束** | `K: Hash + Eq`；`i32`/`String`/`&str` 默认满足；自定义 struct 需 `#[derive(Hash, Eq, PartialEq)]` |
| **不能做键** | **`f32`/`f64`**（浮点相等有精度问题） |

## 二、四种创建方式

### 1. `new()` + 逐个 `insert`

```rust
let mut scores = HashMap::new();
scores.insert(String::from("Blue"), 10);
scores.insert(String::from("Yellow"), 50);
```

`insert(k, v) -> Option<V>`：

| 情况 | 返回值 |
|------|--------|
| key **不存在** | `None` |
| key **已存在** | `Some(旧值)`，新值覆盖 |

```rust
let old = scores.insert(String::from("Blue"), 99); // Some(10)
```

### 2. 两 `Vec` + `zip` + `collect`

→ 原理详解：[8.3.1 zip → collect → HashMap](./8.3.1-zip-collect与HashMap.md)

```rust
let teams = vec![String::from("Blue"), String::from("Yellow")];
let init_score = vec![10, 50];
let scores: HashMap<_, _> = teams.iter().zip(init_score.iter()).collect();
```

> `iter().zip()` → `(&String, &i32)`；**`into_iter().zip()`** → `(String, i32)` 所有权进 Map。

### 3. `HashMap::from([...])`（Rust 1.56+）

```rust
let map = HashMap::from([("Blue", 10), ("Yellow", 50)]);
```

### 4. `with_capacity` 预分配

```rust
let mut map = HashMap::with_capacity(10);
```

## 三、取值三种方式（对标 Vec `[]` / `get`）

| 写法 | 返回 | 无 key 时 | 场景 |
|------|------|-----------|------|
| **`map["key"]`** | `&V` | **panic** | key 一定存在 |
| **`map.get(&key)`** | `Option<&V>` | `None` | key 不确定 |
| **`map.get_mut(&key)`** | `Option<&mut V>` | `None` | 修改 value |

```rust
if let Some(v) = scores.get("Blue") {
    println!("{}", v);
}
if let Some(v) = scores.get_mut("Blue") {
    *v += 5;
}
```

**`get(&key)`** 传键的引用，不消耗 key 所有权。常用链：

```rust
let n = scores.get(&team_name).copied().unwrap_or(0); // Option<&i32> → i32
```

**`contains_key(&key) -> bool`**：只判断存在，不取值。

→ 一页速查：[8.3.2 四段考点](./8.3.2-HashMap四段考点.md)

## 四、Entry API（重难点：只哈希一次）

`map.entry(key) -> Entry`（`Occupied` / `Vacant`）

### `or_insert`

- 无 key → 插入并返回 `&mut V`
- 有 key → 返回现有值的可变引用

```rust
map.entry(String::from("Yellow")).or_insert(50);
map.entry(String::from("Blue")).or_insert(50); // 已存在则不改
```

### 词频统计（最常用）

```rust
let words = ["apple", "banana", "apple"];
let mut cnt = HashMap::new();
for w in words {
    *cnt.entry(w).or_insert(0) += 1;
}
// {"apple": 2, "banana": 1}
```

### 扩展

| 方法 | 作用 |
|------|------|
| **`or_insert_with(\|\| …)`** | 不存在时用闭包延迟生成值 |
| **`and_modify(\|v\| …)`** | 存在才改；常链式：`entry(k).and_modify(...).or_insert(v)` |

```rust
map.entry("A").and_modify(|v| *v *= 2).or_insert(100);
```

## 五、删除

```rust
let mut map = HashMap::from([("A", 1), ("B", 2)]);
let val = map.remove("A");           // Option<V> → Some(1)
let kv = map.remove_entry("B");      // Option<(K,V)> → Some(("B", 2))
```

## 六、遍历

```rust
for k in map.keys() { /* … */ }
for v in map.values() { /* … */ }
for (k, v) in &map { /* (&K,&V)，不消耗所有权 */ }
for (_k, v) in map.iter_mut() { *v += 1; }
```

| 写法 | Map 之后 |
|------|----------|
| **`&map` / `map.iter()`** | 仍可用 |
| **`for (k,v) in map`**（无 `&`） | **move 整个 Map**，遍历后失效 |

顺序**不保证**（哈希表无序）。

## 七、所有权（必考）

| 类型 | `insert` 行为 | 原变量 |
|------|---------------|--------|
| **无 `Copy`**（`String`） | **move** 进 Map | 失效 |
| **实现 `Copy`**（`i32`） | **拷贝** 进 Map | 仍可用 |
| **`insert(&s, v)`** | Map 存 **`&K`** | `s` 仍可用；生命周期须合规 |

```rust
let s = String::from("test");
map.insert(&s, 99); // s 仍可用

let n = 10;
map.insert("A", n);
println!("{}", n); // i32 拷贝，仍可用
```
