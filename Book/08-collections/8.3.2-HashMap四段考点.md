## 一、插入所有权：`insert` 自动 Move / 拷贝

```rust
let field_name = String::from("Favorite color");
let field_value = String::from("Blue");
let mut map = HashMap::new();
map.insert(field_name, field_value);
// field_name、field_value 所有权移入 map，后续不能使用 → 编译报错
```

### 规则

| 类型 | `insert` 行为 | 原变量 |
|------|---------------|--------|
| **无 `Copy`**（`String`、自定义 struct） | **move** 进 Map | 失效 |
| **实现 `Copy`**（`i32`、`bool`、`u8`） | **拷贝** 进 Map | 仍可用 |

```rust
let num = 10;
map.insert("A", num);
println!("{}", num); // ✅ i32 是拷贝
```

> 想保留原数据：`insert(&field_name, &field_value)`，Map 存**借用**（生命周期须合规）。

## 二、取值 `get`：返回 `Option<&V>`

```rust
let team_name = String::from("Blue");
let score = scores.get(&team_name); // Option<&i32>
```

1. **`get(&key)`** 传键的**引用**，不消耗 key 所有权  
2. 存在 → `Some(&V)`（借用 Map 内数据）；不存在 → `None`，**不 panic**  
3. **`get_mut(&key)`** → `Option<&mut V>`，可变修改 value  
4. 常用辅助：  
   - **`.copied()`**：`Option<&i32>` → `Option<i32>`（`V: Copy` 时）  
   - **`.unwrap_or(0)`**：取不到返回默认值  

```rust
let n = scores.get(&team_name).copied().unwrap_or(0);
```

## 三、遍历规则

```rust
// &scores：借用遍历，不转移 map 所有权
for (key, value) in &scores {
    println!("{key}: {value}");
}
```

| 写法 | 元素类型 | Map 之后 |
|------|----------|----------|
| **`&scores` / `scores.iter()`** | `(&K, &V)` | 仍可用 |
| **`scores`（不带引用）** | `(K, V)` | **move 整个 Map**，遍历后失效 |
| **`scores.iter_mut()`** | `(&K, &mut V)` | 可改 value |

- HashMap **无序**，打印顺序随机（哈希存储，不保证插入顺序）  
- 可变遍历：`for (_, v) in scores.iter_mut() { *v += 1; }`

## 四、四种更新策略（高频）

### 1. 直接 `insert`：旧值被覆盖

```rust
scores.insert(String::from("Blue"), 10);
scores.insert(String::from("Blue"), 25); // 覆盖 → 最终 25
```

`insert` 返回 `Option<V>`：覆盖 → `Some(旧值)`；新增 → `None`。

### 2. `entry` + `or_insert`：缺则插，有则不动

```rust
scores.entry(String::from("Yellow")).or_insert(50); // 无 Yellow → 插 50
scores.entry(String::from("Blue")).or_insert(50);   // 已有 Blue → 不改
```

- `entry(key)` **只哈希查找一次**，分 `Occupied` / `Vacant`  
- `or_insert(x)`：空缺 → 插入 x 并返回 `&mut V`；已存在 → 返回原值可变引用  

### 3. 经典：基于原值更新（词频统计）

```rust
let text = "apple pear apple banana apple";
let mut map = HashMap::new();
for word in text.split_whitespace() {
    let count = map.entry(word).or_insert(0);
    *count += 1;
}
```

`or_insert` 固定返回 `&mut usize`，`*` 解引用自增 — Rust 统计标配。

### 4. 链式 `entry`（拓展）

```rust
map.entry("k").and_modify(|v| *v *= 2).or_insert(10);
// 键存在 → 翻倍；不存在 → 插入 10
```

---

## 快速总结表

| 操作 | 写法 | 核心特点 |
|------|------|----------|
| 全覆盖更新 | `insert(k, v)` | 无脑覆盖旧值 |
| 缺则插入 | `entry(k).or_insert(v)` | 存在不动，缺失新增 |
| 基于原值修改 | `entry(k).or_insert(0); *v += 1` | 词频统计专用 |
| 只读查询 | `get(&k)` | `Option<&V>`，安全不 panic |
