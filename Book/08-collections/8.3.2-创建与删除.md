## 一、四种创建方式

### 1. `new()` + 逐个 `insert`

```rust
let mut scores = HashMap::new();
scores.insert(String::from("Blue"), 10);
scores.insert(String::from("Yellow"), 50);
```

`insert(k, v) -> Option<V>`：

| 情况 | 返回值 |
|------|--------|
| key **不存在** | `None` |
| key **已存在** | `Some(旧值)`，新值覆盖 |

```rust
let old = scores.insert(String::from("Blue"), 99); // Some(10)
```

> `insert` 的所有权规则、更新策略 → [8.3.3 §一、§四](./8.3.3-insert-get-遍历与Entry.md)

### 2. 两 `Vec` + `zip` + `collect`

→ 完整原理：[8.3.1 zip → collect → HashMap](./8.3.1-zip-collect与HashMap.md)

```rust
let teams = vec![String::from("Blue"), String::from("Yellow")];
let init_score = vec![10, 50];
let scores: HashMap<_, _> = teams.iter().zip(init_score.iter()).collect();
```

### 3. `HashMap::from([...])`（Rust 1.56+）

```rust
let map = HashMap::from([("Blue", 10), ("Yellow", 50)]);
```

### 4. `with_capacity` 预分配

```rust
let mut map = HashMap::with_capacity(10);
```

已知大约要插多少条时，少几次扩容。

## 二、删除

```rust
let mut map = HashMap::from([("A", 1), ("B", 2)]);
let val = map.remove("A");       // Option<V> → Some(1)，只删 value
let kv = map.remove_entry("B");  // Option<(K,V)> → Some(("B", 2))，K+V 一起拿走
```

| 方法 | 返回 | 用途 |
|------|------|------|
| **`remove(&K)`** | `Option<V>` | 只要 value |
| **`remove_entry(&K)`** | `Option<(K,V)>` | key 也要 reuse |
