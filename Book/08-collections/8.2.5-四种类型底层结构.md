# 8.2.5 四种类型底层：`String` / `&str` / `[T;N]` / `&[T]`

← [8.2.1 str 与 DST](./8.2.1-str与DST定长规则.md) · [8.2.2 不定长放哪](./8.2.2-不定长数据放哪.md) · 主笔记：[8.2](./8.2-string.md)

## 核心结论

| # | 规则 |
|---|------|
| 1 | **`&str`、 `[T;N]`、 `&[T]`** 不是 struct，是**编译器内置原生类型** |
| 2 | **`String`** 是标准库用 **`struct`** 实现的堆字符串，能写出源码结构 |
| 3 | **切片**（`&str` / `&[T]`）= **胖指针**（ptr + len），无专属 struct，编译器语法糖 |
| 4 | **定长数组 `[T; N]`** = 栈上连续内存块，编译期布局，不是 struct |

---

## 一、`String`（标准库 struct，简化源码）

```rust
pub struct String {
    ptr: *mut u8,   // 堆上 UTF-8 字节起始地址
    len: usize,     // 有效字节数（非字符数）
    cap: usize,     // 已分配容量，cap ≥ len
}
```

| 位置 | 内容 |
|------|------|
| **栈** | `String` 本身（64 位通常 **24 字节** = 3×usize） |
| **堆** | `ptr` 指向连续 `u8` UTF-8 字节 |

```rust
let s = String::from("abc");
// ptr → 堆 [97, 98, 99]；len = 3, cap ≥ 3
```

---

## 二、`&str`：不是 struct，胖指针

逻辑等价（**不能**在 Rust 里真这么定义 `&str`）：

```rust
// 仅理解用
struct StrFatPtr {
    data: *const u8,
    len: usize,
}
// &str ≈ (data, len)，64 位通常 16 字节
```

`&str` **永远借用别处内存**：

| 来源 | 数据在哪 |
|------|----------|
| `"abc"` 字面量 | **只读程序段**（`&'static str`） |
| `&s[..]` | **借 `String` 的堆** |

```rust
let s = String::from("hello");
let slice: &str = &s[..];
// slice 的 ptr/len 指向 s 的堆字节
```

---

## 三、`[T; N]`：编译器原生栈连续内存

```rust
let arr: [i32; 3] = [1, 2, 3];
// 栈上连续 3 个 i32，无 ptr/len/cap 字段
```

- 定长、编译期知 `N`；
- 不能扩容；
- **不是 struct 封装**。

---

## 四、`&[T]`：与 `&str` 同源的通用胖指针

```rust
// 逻辑等价
struct SliceFat<T> {
    start: *const T,
    len: usize,   // 元素个数（非字节）
}
```

```rust
let arr = [1, 2, 3, 4];
let slice: &[i32] = &arr[1..3]; // 元素 [2, 3]，len = 2
```

---

## 五、对照表

| 类型 | 底层实现 | 数据常见位置 | 栈上占多少（64 位典型） |
|------|----------|--------------|-------------------------|
| **`String`** | struct `{ptr,len,cap}` | 结构体在栈，**数据在堆** | 24 B |
| **`&str`** | 胖指针 `(ptr, len)` | 数据：静态段/堆；**切片在栈** | 16 B |
| **`[T; N]`** | 原生连续内存 | **全在栈**（或静态） | `N × size_of::<T>()` |
| **`&[T]`** | 胖指针 `(ptr, len)` | 依附原数组；**切片在栈** | 16 B |

---

## 补充：`Vec<T>` 与 `String` 同源

```rust
pub struct Vec<T> {
    ptr: *mut T,
    len: usize,
    cap: usize,
}
```

> **`String` ≈ `Vec<u8>`**，额外保证内容为合法 UTF-8。

→ 见 [8.1 Vector](../8.1-vector.md)

## 内存示意图

```text
String（栈 24B）          堆
┌ ptr ──────────────────► [ u8 u8 u8 ... ]
│ len
└ cap

&str（栈 16B）            只读段 或 堆
┌ data ─────────────────► [ u8 u8 u8 ]
└ len

[i32;3]（栈 12B）
[ 1 | 2 | 3 ]   连续平铺，无指针

&[i32]（栈 16B）          栈上的 arr
┌ start ────────────────► arr[1]
└ len = 2
```
