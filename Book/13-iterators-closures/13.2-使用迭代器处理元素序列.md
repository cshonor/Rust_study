## 速览

| 概念 | 要点 |
|------|------|
| **惰性** | `map`/`filter` 只存规则；`collect`/`sum`/`for` 才跑 |
| **`Iterator`** | 只实现 `next` + 关联类型 `Item` |
| **适配器** | 返回新迭代器，仍惰性 |
| **消费器** | 拿走迭代器，用完作废 |
| **iter / iter_mut / into_iter** | `&T` / `&mut T` / `T` |
| **返回迭代器** | 常见 `impl Iterator`；分支异构 → `Box<dyn Iterator>` |

→ 完整拆解：[13.2.1](./13.2.1-迭代器精讲.md)
