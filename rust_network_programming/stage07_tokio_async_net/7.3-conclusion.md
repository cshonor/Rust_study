## 五、新手易错（全章汇总）

1. **在 async 里阻塞**：`std::thread::sleep`、同步 `mpsc`、无 `spawn_blocking` 的 CPU 密集。
2. **跨 `.await` 持锁**：`std::sync::Mutex` → 换 tokio 版或 message passing。
3. **TCP 字节流无消息边界**：必须 Codec/长度前缀/应用协议分帧。
4. **Future 不 poll 就不执行**：忘记 `.await` 或 `spawn`。
5. **同时踩异步 + TLS**：建议先熟练 Ch7 再加 [Ch8](../stage08_network_security/)。
