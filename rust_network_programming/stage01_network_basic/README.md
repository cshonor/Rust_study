# Stage 01 — 网络理论基础

**对应书籍**：第 1 章 — *Introduction to Client/Server Networking*（客户端/服务器网络简介）

## 核心内容

1. 网络的简史  
2. 分层模型（OSI / TCP/IP）与封装  
3. 网络寻址（MAC、IP、ASN、端口）  
4. IP 路由（路由表、最长前缀匹配、TTL、OSPF/BGP）  
5. DNS 解析流程  
6. 面向连接（TCP）vs 无连接（UDP）  
7. Linux 套接字与系统调用、阻塞 vs 多路复用  

## 学习定位

- **纯理论打底**，补足「自顶向下」式工程视角。  
- 为 Stage 03（`std::net`）/ Stage 07（Tokio）的代码建立心智模型。
