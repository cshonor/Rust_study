# Item 8 · 重点结论

← [Item 8 目录](./README.md)

1. **安全 Rust 无悬垂引用**——生命周期在编译期拦截；裸指针隔离在 `unsafe`。
2. **复杂共享结构**——组合智能指针，例如：
   - `Rc<RefCell<Vec<T>>>`：共享列表、整体可变；
   - `Rc<Vec<RefCell<T>>>`：共享容器、**逐项**可变。
3. **`Rc` 单线程，`Arc` 多线程**；跨 `.await` 常需 `Arc` + `Send`（见 §6）。
4. **图/树回边**——父指针用 **`Weak`**，避免 `Rc` 环泄漏。
