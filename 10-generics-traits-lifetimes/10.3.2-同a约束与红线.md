## 一、铁律与红线

> 在 **`'a` 划定的有效区间内**，组内所有被引用数据**都必须存活**。  
> ≠ 两个原变量必须**同时 drop**。

| 对象 | 规则 |
|------|------|
| 原变量 | drop 由 `{}` 决定 |
| `&'a T` 引用 | 只能用到本次 `'a` 上界；**不能跨红线** |

同一组 `'a`：**引用使用上界** = 组内最短数据源在本次调用中的作用域末尾。

```text
main ─────────────────────────►
      │ 内层 { ───► } │  ← 红线（short 作用域末）
      │ 块内 long、short 都活 │  本次 &'a 截止
```

## 二、`use_both` 实测

**rustc / demo §0.8：编译通过。**

```rust
fn use_both<'a>(a: &'a i32, b: &'a i32) {}

fn main() {
    let long_live = 100;
    {
        let short_live = 200;
        use_both(&long_live, &short_live); // ✅
    }
}
```

| 步 | 编译器行为 |
|----|------------|
| 1 | `a`、`b` 同组 `'a` |
| 2 | `'a` **收缩到内层块** |
| 3 | 块内两者都活着；无 `&'a` 流出 |
| 4 | `long_live` **变量**出块仍可存在；本次 `&'a` 不能跟出 |

**无返回值** + 内层一长一短 + 同 `'a` → **永远不报错**。

## 三、三种场景

### 同块两实参 ✅

```rust
fn main() {
    let x = 1;
    let y = 2;
    use_both(&x, &y);
}
```

### 嵌套块 + 同 `'a`

| 情况 | 结果 |
|------|------|
| **无返回值**，仅内层调用 | ✅（§二） |
| **有 `&'a` 返回值**，`res` 在内层定义并使用 | ✅ |
| **有 `&'a` 返回值**，`res` 在外层或出块再用 | ❌ |

```rust
fn longest<'a>(x: &'a str, y: &'a str) -> &'a str {
    if x.len() > y.len() { x } else { y }
}

// ✅ 场景 A：res 仅内层用
// ❌ 场景 B：res 在外层或出块 println!
```

**唯一触发报错（同 `'a`）**：组内 `&'a` 引用（尤其**返回值**）被带出本次 `'a` 的最短作用域。

→ 详解 [10.3.4 longest](./10.3.4-longest与get_first.md)

### 不同标签 `'a` / `'b` ✅

```rust
fn two_group<'a, 'b>(x: &'a i32, y: &'b i32) {}

fn main() {
    let outer = 100;
    {
        let inner = 200;
        two_group(&outer, &inner);
    }
}
```

## 四、原变量 vs `&'a` 引用

| 层次 | 出内层 `}` 后 |
|------|---------------|
| 原变量 `outer` | ✅ 仍可用 |
| 本次调用的 `&'a` 引用 | ❌ 不可再使用 |
| 新建 `let r = &outer` | ✅ 新生命周期 |

```rust
fn demo<'a>(x: &'a i32, y: &'a i32) {}

fn main() {
    let outer = 100;
    {
        let inner = 200;
        demo(&outer, &inner); // ✅
    }
    println!("{}", outer); // ✅
}
```

## 五、多标签与返回值

```rust
fn pick_one<'a, 'b>(x: &'a i32, _y: &'b i32) -> &'a i32 { x }

fn pick_any<'a>(x: &'a i32, y: &'a i32) -> &'a i32 {
    if *x > *y { x } else { y }
}
```

返回 `&'a T` 须与来源入参的 `'a` 对齐。

## 六、一页纸速查

| 返回值 | 同 `'a` | 长短 | 用法 | 结果 |
|--------|---------|------|------|------|
| 无 | 是 | 一长一短 | 内层 `use_both` | ✅ |
| 无 | 是 | 一长一短 | `&'a` 存到块外再用 | ❌ |
| 有 | 是 | 一长一短 | `res` 仅内层用 | ✅ |
| 有 | 是 | 一长一短 | `res` 外层或出块用 | ❌ |
| 任意 | `'a`+'b` | 一长一短 | 内层调用 | ✅ |
| 有 | 是 | — | 返回局部变量引用 | ❌ |

**背三句**：`'a` = 引用边界；区间内数据都活 → ✅；引用逃出边界 → ❌。

## 七、`'a` 是函数局部标签

不同函数里的 `'a` **互不影响**。

```rust
fn demo1<'a>(x: &'a i32, y: &'a i32) {}
fn demo2<'a>(p: &'a i32) {}

fn main() {
    let outer = 100;
    {
        let inner = 200;
        demo1(&outer, &inner); // ✅
        demo2(&inner);
    }
    demo2(&outer);
}
```

```rust
fn return_ref<'a>(p: &'a i32) -> &'a i32 { p }

fn main() {
    let outer = 100;
    {
        let inner = 200;
        let r1 = return_ref(&inner);
        println!("{}", r1); // 块内用完 ✅
    }
    let r2 = return_ref(&outer);
}
```
