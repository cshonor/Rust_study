## 核心结论

生命周期标记（`'a`、`'static`）**不创建引用**，只约束：**引用在使用期间，指向的数据必须一直有效**。

> 入参、返回值都标 `'a` → 返回的引用**来自**带同一 `'a` 的入参，**不能比该组 `'a` 活得更久**。

---

## 最终三句（唯一结论，可直接复制）

> 以下与 **rustc / demo §0.8** 一致；无「同码不同结果」之类区分。

```
【Rust 生命周期 · 同 'a 三条】

1. 无返回值函数
   入参一长一短 + 共用 'a → 永远不报错。
   编译器把 'a 收缩到最短作用域；该块内被借数据都活着。

2. 有返回值函数（返回 &'a T）
   返回值留在最短作用域内 → 不报错；
   返回值跑到最短作用域外 → 报错（悬垂）。

3. 'a 的本质
   'a = 引用的使用边界，不是「两变量必须同时 drop」。
   只要求：在 'a 范围内，所有被引用的数据都活着。
```

**canonical 实测（无返回值，永远 ✅）**：

```rust
fn use_both<'a>(a: &'a i32, b: &'a i32) {}

fn main() {
    let long_live = 100;
    {
        let short_live = 200;
        use_both(&long_live, &short_live);
    }
}
```

详解：[§五](#五同-a-与红线) · 表：[§5.7](#57-一页纸速查)

## 一、变量寿命 vs `'a` 标注

| | 变量寿命 | `'a` 标注 |
|---|----------|-----------|
| 是什么 | 从声明到 `{}` 结束（客观） | 编译期给**引用**分的组 |
| 管什么 | 变量何时 drop | **引用**能安全用到何时 |

```rust
fn main() {
    let num = 10;
    let r = &num;
} // num、r 一起结束
```

> `'a`：**同组引用的使用边界**在推断区间内须与数据存活一致；**不改变**变量本身的 drop 时机。

---

## 二、标注 vs 作用域

| 规则 | 管什么 |
|------|--------|
| `'a` / `'static` | **被指向的数据**能活多久 |
| `{}` | **引用变量**何时销毁 |

```rust
fn main() {
    let s: &'static str = "test";
} // 变量 s 消失；"test" 仍在只读段
```

## 四、`'a` 约束什么

```rust
fn borrow<'a>(data: &'a i32) -> &'a i32 {
    data
}
```

| 部分 | 含义 |
|------|------|
| `data: &'a i32` | 入参引用绑定 `'a` |
| `-> &'a i32` | 返回值与入参**同 `'a`** |

```rust
// fn bad<'a>() -> &'a i32 {
//     let tmp = 20;
//     &tmp // ❌ 无外部数据源支撑 'a
// }
```

**三条规则**：

1. `'a` = **时长分组**，不是变量绑定；
2. 同 `'a` = 在**本次推断的 `'a` 区间内**，被借数据都须存活；
3. 整组上限 = **最短**数据源在本次调用中的作用域（红线）。

## 五、同 `'a` 与红线

### 5.1 铁律（正确理解）

> 在 **`'a` 划定的有效区间内**，组内所有被引用数据**都必须存活**。  
> ≠ 两个原变量必须**同时 drop**。

| 对象 | 规则 |
|------|------|
| 原变量 | drop 由 `{}` 决定 |
| `&'a T` 引用 | 只能用到本次 `'a` 上界；**不能跨红线** |

### 5.2 红线模型

同一组 `'a`：**引用使用上界** = 组内最短数据源在本次调用中的作用域末尾。

```text
main ─────────────────────────►
      │ 内层 { ───► } │  ← 红线（short 作用域末）
      │ 块内 long、short 都活 │  本次 &'a 截止
```

### 5.3 实测：`use_both` 内层一长一短 ✅

**rustc / demo §0.8 验证：编译通过。**

```rust
fn use_both<'a>(a: &'a i32, b: &'a i32) {}

fn main() {
    let long_live = 100;
    {
        let short_live = 200;
        use_both(&long_live, &short_live); // ✅
    }
}
```

| 步 | 编译器行为 |
|----|------------|
| 1 | `a`、`b` 同组 `'a` |
| 2 | `'a` **收缩到内层块**（最短作用域） |
| 3 | 块内 `long_live`、`short_live` 都活着；无 `&'a` 流出 |
| 4 | `long_live` **变量**出块仍可存在；本次 `&'a` 不能跟出 |

### 5.4 三种场景

#### 同块两实参 ✅

```rust
fn main() {
    let x = 1;
    let y = 2;
    use_both(&x, &y);
}
```

#### 嵌套块 + 同 `'a`

| 情况 | 结果 |
|------|------|
| **无返回值**，仅内层调用 | ✅（§5.3） |
| **有 `&'a` 返回值**，`res` 在内层定义并使用 | ✅ |
| **有 `&'a` 返回值**，`res` 在外层或出块再用 | ❌ |

```rust
fn longest<'a>(x: &'a str, y: &'a str) -> &'a str {
    if x.len() > y.len() { x } else { y }
}

// ✅ 场景 A
// fn main() {
//     let outer = String::from("aaa");
//     {
//         let inner = String::from("bb");
//         let res = longest(&outer, &inner);
//         println!("{}", res);
//     }
// }

// ❌ 场景 B
// fn main() {
//     let outer = String::from("aaa");
//     let res: &str;
//     {
//         let inner = String::from("bb");
//         res = longest(&outer, &inner);
//     }
//     println!("{}", res);
// }
```

**唯一触发报错（同 `'a`）**：组内 `&'a` 引用（尤其**返回值**）被带出本次 `'a` 的最短作用域。

#### 不同标签 `'a` / `'b` ✅

```rust
fn two_group<'a, 'b>(x: &'a i32, y: &'b i32) {}

fn main() {
    let outer = 100;
    {
        let inner = 200;
        two_group(&outer, &inner);
    }
}
```

### 5.5 原变量 vs `&'a` 引用

| 层次 | 出内层 `}` 后 |
|------|---------------|
| 原变量 `outer` | ✅ 仍可用 |
| 本次调用的 `&'a` 引用 | ❌ 不可再使用 |
| 新建 `let r = &outer` | ✅ 新生命周期，与旧 `'a` 无关 |

```rust
fn demo<'a>(x: &'a i32, y: &'a i32) {}

fn main() {
    let outer = 100;
    {
        let inner = 200;
        demo(&outer, &inner); // ✅
    }
    println!("{}", outer); // ✅
}
```

### 5.6 多标签与返回值模式

```rust
fn pick_one<'a, 'b>(x: &'a i32, _y: &'b i32) -> &'a i32 { x }

fn pick_any<'a>(x: &'a i32, y: &'a i32) -> &'a i32 {
    if *x > *y { x } else { y }
}
```

- 返回 `&'a T` 须与来源入参的 `'a` 对齐；
- 同函数内多参共用 `'a` 且**保存/返回**引用时，才受红线约束。

### 5.7 一页纸速查

| 返回值 | 同 `'a` | 长短 | 用法 | 结果 |
|--------|---------|------|------|------|
| 无 | 是 | 一长一短 | 内层 `use_both` | ✅ |
| 无 | 是 | 一长一短 | `&'a` 存到块外再用 | ❌ |
| 有 | 是 | 一长一短 | `res` 仅内层用 | ✅ |
| 有 | 是 | 一长一短 | `res` 外层或出块用 | ❌ |
| 任意 | `'a`+'b` | 一长一短 | 内层调用 | ✅ |
| 有 | 是 | — | 返回局部变量引用 | ❌ |

**背三句**：`'a` = 引用边界；区间内数据都活 → ✅；引用逃出边界 → ❌。

## 六、`'a` 是函数局部标签

不同函数里的 `'a` **互不影响**——只是名字碰巧相同。

```rust
fn demo1<'a>(x: &'a i32, y: &'a i32) {}
fn demo2<'a>(p: &'a i32) {}

fn main() {
    let outer = 100;
    {
        let inner = 200;
        demo1(&outer, &inner); // ✅ 本函数内 'a 缩到内层
        demo2(&inner);         // ✅ 单参，'a 绑 inner
    }
    demo2(&outer);
}
```

```rust
fn return_ref<'a>(p: &'a i32) -> &'a i32 { p }

fn main() {
    let outer = 100;
    {
        let inner = 200;
        let r1 = return_ref(&inner); // 'a = inner 区间；块内用完 ✅
        println!("{}", r1);
    }
    let r2 = return_ref(&outer);
}
```

```rust
// fn get_short<'a>(x: &'a i32, y: &'a i32) -> &'a i32 { x }

//         let inner = 200;

//     }
//     // println!("{}", ret); // ❌ ret 跨红线

```

## 八、`get_first` 与 `longest`

### 8.1 `get_first`

```rust
fn get_first<'a>(s: &'a str) -> &'a str {
    &s[0..1]
}
```

单入参 + 返回引用 → 可省略 `'a`（见 §十）。

### 8.2 `longest`

```rust
fn longest<'a>(x: &'a str, y: &'a str) -> &'a str {
    if x.len() > y.len() { x } else { y }
}
```

- 签名：`x`、`y`、返回值**共用 `'a`**；
- 每次调用：`'a` = 两实参中**较短**的作用域；
- **不是**「一长一短传参就报错」，看 **`res` 是否跨红线**（§5.4）。

### 8.3 同作用域两实参

```rust
fn main() {
    let s1 = String::from("abc");
    let s2 = String::from("abcd");
    let res = longest(&s1, &s2);
    println!("{}", res);
}
```

### 8.4 `'a` / `'b` 分组

```rust
fn two_str<'a, 'b>(x: &'a str, y: &'b str) {
    println!("{} {}", x, y);
}
```

## 十、显式与隐式生命周期

**核心**：不写 `'a` 时编译器**隐式划分区间**，校验逻辑相同；手写 `'a` = **手动命名分组**。

| | 隐式 | 显式 |
|---|------|------|
| 适用 | 单引用入参+返回、简单借用 | 多引用入参、结构体持引用 |
| 例子 | `fn f(x: &i32) -> &i32` | `fn longest<'a>(...) -> &'a str` |

**必须手写**：结构体/枚举含引用；多个引用入参需标明同组或分组。

**省略三条（elision）**：

1. 每个引用参数有输入生命周期；
2. 仅一个输入生命周期 → 赋给所有输出；
3. 方法有 `&self` → 输出默认跟 `self`。

> 多个引用入参 + 返回引用 → **必须手写**（如 `longest`）。

```rust
// 隐式跨块报错（无手写 'a，照样检查）
// fn main() {
//     let r: &i32;
//     { let x = 10; r = &x; }
//     println!("{}", r);
// }
```
