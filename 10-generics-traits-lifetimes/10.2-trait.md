# 10.2 trait：定义共享的行为

对应 demo 项目：`10-generics-traits-lifetimes/10.2-traits-demo/`

trait 用来抽象“共享的行为”（类似接口），并可通过 **trait bound** 约束泛型类型必须具备某些能力。

---

## 定义 trait

```rust
pub trait Summary {
    fn summarize(&self) -> String;
}
```

trait 里写方法签名（末尾分号），实现者必须提供同签名的方法体。

---

## 为类型实现 trait

```rust
impl Summary for Tweet {
    fn summarize(&self) -> String {
        format!("{}: {}", self.username, self.content)
    }
}
```

调用方式与普通方法一样：`tweet.summarize()`。

---

## 孤儿规则（orphan rule）

只能在满足其一时实现 trait：
- trait 在当前 crate 定义 **或**
- 类型在当前 crate 定义

因此：不能为外部类型实现外部 trait（例如给 `Vec<T>` 实现 `Display`）。

---

## 默认实现（default implementation）

trait 方法可以有默认实现，实现者可选择重写：

```rust
pub trait Summary {
    fn summarize(&self) -> String {
        String::from("(Read more...)")
    }
}
```

也可以让默认实现调用同 trait 的其他方法（只要求实现少数“核心方法”）。

> 注意：不能从同名的重写实现里调用默认实现。

---

## trait 作为参数

### `impl Trait`

```rust
pub fn notify(item: impl Summary) {
    println!("Breaking news! {}", item.summarize());
}
```

### trait bound（泛型）

```rust
pub fn notify<T: Summary>(item: T) { /* ... */ }
```

多个约束用 `+`：

```rust
pub fn notify(item: impl Summary + std::fmt::Display) { /* ... */ }
```

大量约束可用 `where`：

```rust
fn f<T, U>(t: T, u: U)
where
    T: std::fmt::Display + Clone,
    U: Clone + std::fmt::Debug,
{ }
```

---

## 返回实现了 trait 的类型

```rust
fn returns_summarizable() -> impl Summary { /* ... */ }
```

限制：`impl Trait` 返回值必须是**单一具体类型**（不能根据分支返回不同类型）。

---

## 用 trait bound 修复泛型 largest

```rust
fn largest<T: PartialOrd + Copy>(list: &[T]) -> T { /* ... */ }
```

如果不想要求 `Copy`：
- 可以用 `Clone`（可能有额外开销）
- 或返回引用 `&T`（最通用、避免 move）

---

## 条件实现（有条件的方法 / blanket impl）

可根据 trait bound 只给满足条件的类型实现方法：

```rust
impl<T: Display + PartialOrd> Pair<T> {
    fn cmp_display(&self) { /* ... */ }
}
```

对所有满足条件的类型实现 trait 称为 blanket implementation，例如标准库：

```rust
impl<T: Display> ToString for T { /* ... */ }
```
