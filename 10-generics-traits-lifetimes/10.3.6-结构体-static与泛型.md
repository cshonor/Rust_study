## 一、结构体中的引用

```rust
struct ImportantExcerpt<'a> {
    part: &'a str,
}
```

实例不能比 `part` 指向的数据活得更久。

```rust
// struct Wrapper { val: &i32 } // ❌ 必须写 'a

struct Wrapper<'a> {
    val: &'a i32,
}
```

→ 悬垂场景见 [10.3.1 §4.4](./10.3.1-悬垂引用.md#44-结构体容器持引用)

## 二、`'static`

```rust
let s: &'static str = "I have a static lifetime.";
```

- 字面量在**只读数据段**，程序全程存在；
- 栈上的引用变量 `s` 仍会出 `{}` 销毁；
- 不要用 `'static` 掩盖真实的生命周期不匹配。

类型拆解见 [10.3.3 §三](./10.3.3-生命周期基础.md#三static-str)。

## 三、泛型 + trait + `'a`

```rust
use std::fmt::Display;

fn longest_with_an_announcement<'a, T>(x: &'a str, y: &'a str, ann: T) -> &'a str
where
    T: Display,
{
    println!("Announcement! {}", ann);
    if x.len() > y.len() { x } else { y }
}
```

`'a` 与泛型参数 `T` 独立：`'a` 管引用边界，`T` 管额外入参类型。
