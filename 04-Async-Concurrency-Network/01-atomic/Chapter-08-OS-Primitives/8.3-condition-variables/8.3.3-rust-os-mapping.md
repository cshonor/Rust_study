## 总表

| Rust API | 典型底层（概念） |
|----------|------------------|
| `Mutex::lock` 阻塞 | futex WAIT / pthread mutex / critical section |
| `MutexGuard` drop | 用户态解锁 + futex WAKE |
| `Condvar::wait(guard)` | 释放 mutex + cond 上 WAIT（或 futex REQUEUE 路径） |
| `Condvar::notify_one` | signal / WakeConditionVariable / WakeByAddressSingle |
| `Condvar::notify_all` | broadcast / WakeAll / WakeByAddressAll |
| `thread::park` / `unpark` | 平台线程调度 + 可能 futex（实现细节） |

日常 **safe Rust 不直接 syscall** —— std 封装上述原语。
