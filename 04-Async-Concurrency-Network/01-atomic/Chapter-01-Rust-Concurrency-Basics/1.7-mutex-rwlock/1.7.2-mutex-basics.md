**一句话**：`lock()` 阻塞直到拿到锁 → 返回 **`MutexGuard<T>`** → 通过 **`DerefMut`** 当 `&mut T` 用 → **Drop 时自动解锁**（RAII）。

```rust
use std::sync::Mutex;

let m = Mutex::new(5);
{
    let mut guard = m.lock().unwrap();
    *guard += 10;
} // guard drop → 解锁
println!("{}", m.lock().unwrap());
```

### 多线程累加模板

```rust
use std::sync::{Arc, Mutex};
use std::thread;

let n = Arc::new(Mutex::new(0));
let mut handles = vec![];

for _ in 0..10 {
    let n = Arc::clone(&n);
    handles.push(thread::spawn(move || {
        for _ in 0..100 {
            *n.lock().unwrap() += 1;
        }
    }));
}
for h in handles {
    h.join().unwrap();
}
assert_eq!(*n.lock().unwrap(), 1000);
```
