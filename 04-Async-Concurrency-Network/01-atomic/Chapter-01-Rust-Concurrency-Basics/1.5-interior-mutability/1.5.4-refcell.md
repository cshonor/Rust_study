## 1.5.4 `RefCell<T>` 精读

> 书 §1.5 · 索引：[1.5-interior-mutability.md](./1.5-interior-mutability.md) · 前：[1.5.3 Cell](./1.5.3-cell.md) · 后：[1.5.5 UnsafeCell](./1.5.5-unsafecell-multithread.md)

---

**适用**：非 Copy 类型（`String`、`Vec` 等）。运行时跟踪借用：

- `borrow()` → `Ref<T>`（多个共享读 OK）
- `borrow_mut()` → `RefMut<T>`（独占写；与任何借用互斥）

```rust
use std::cell::RefCell;

let rc = RefCell::new(String::from("hello"));
{
    let mut s = rc.borrow_mut();
    s.push_str(" world");
}
println!("{}", rc.borrow());
```

违反规则（例如同时两个 `borrow_mut`）→ **panic**，不是 UB。

仓库对应：[1.5-interior-mutability-cell-refcell-demo.rs](./code/1.5-interior-mutability-cell-refcell-demo.rs)

### 对照：`Cell` vs `RefCell`

| | **`Cell<T>`** | **`RefCell<T>`** |
|---|---------------|------------------|
| `T` 约束 | `Copy` | 任意 |
| 访问方式 | 复制 / 整体替换 | 借 `Ref` / `RefMut` |
| 运行时开销 | 无 | 借用计数 |
| 跨线程 | ❌ 非 `Sync` | ❌ 非 `Sync` |
