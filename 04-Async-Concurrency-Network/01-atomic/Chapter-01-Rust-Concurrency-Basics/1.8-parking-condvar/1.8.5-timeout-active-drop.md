## 超时 API

| API | 作用 |
|-----|------|
| **`Condvar::wait_timeout`** | 条件不满足时最多等一段时间；返回 `(guard, WaitTimeoutResult)` |
| **`thread::park_timeout`** | 线程休眠上限；可被提前 `unpark` |

超时后应像正常唤醒一样**再查条件**，不要假设一定就绪。

仓库对应：`demo_wait_timeout`、`demo_park_timeout`

---

## 主动 `drop(guard)` 再干慢活

消费者从队列取出任务后，**处理任务可能很慢**——应 **`drop(q)` 释放锁**，再 `sleep` / 计算，否则生产者无法入队。

仓库对应：`demo_active_drop`
