## 9.3.2 写优先 RwLock — 状态机与代码骨架

---

## `state` 编码

| 编码 | 含义 |
|------|------|
| **读者数 × 2** 存入 `state` | 用 **偶/奇** 标记是否有写者排队 |
| **偶数** | 无写者排队 → **新读者可** `CAS +2` |
| **奇数** | 有写者排队 → **新读者必须 wait** |
| **`u32::MAX`** | 写者持写锁 |

另设 **`writer_wake_counter`**：写者在专用计数上 **wait**，由**最后一个读者** `Drop` 时 **`wake_one`**。

---

## 写者 `write()` 三步

1. 若 `s <= 1` → CAS 为 **`MAX`**，获得写锁。  
2. 若 **偶数**且仍有读者 → **`state += 1` 变奇数**，举起「写者等待」旗，**拦截新读者**。  
3. 在 **`writer_wake_counter`** 上 **wait**，直到最后读者唤醒。

---

## 读者 `read()` 两步

1. 仅当 **`s % 2 == 0`** 时 CAS **`s + 2`** 加入。  
2. 若 **奇数**或已是 **`MAX`** → **wait** 在 `state` 上。

---

## 运行逻辑（写者饥饿如何消除）

1. 写者发现仍有读者 → **偶数 state 加 1 变奇数**。  
2. 此后所有 **`read()`** 见奇数 → **不再加入**，进入 wait。  
3. 老读者陆续 Drop，`state` 减 2。  
4. **最后一个读者** wake 写者 → 写者 CAS 为 `MAX` 独占。  
5. 写者 Drop 后 **wake_all** 读者（书中完整实现见原书）。

---

## 代码骨架（与书一致，节选）

```rust
// 依赖：atomic_wait = "..."  （书中示例；仓库未默认加入 Cargo.toml）

use std::cell::UnsafeCell;
use std::sync::atomic::{AtomicU32, Ordering::{Acquire, Relaxed, Release}};
// use atomic_wait::{wait, wake_one, wake_all};

pub struct RwLock<T> {
    state: AtomicU32,
    writer_wake_counter: AtomicU32,
    value: UnsafeCell<T>,
}

impl<T> RwLock<T> {
    pub fn read(&self) -> ReadGuard<T> {
        let mut s = self.state.load(Relaxed);
        loop {
            if s % 2 == 0 {
                match self
                    .state
                    .compare_exchange_weak(s, s + 2, Acquire, Relaxed)
                {
                    Ok(_) => return ReadGuard { rwlock: self },
                    Err(e) => s = e,
                }
            }
            if s % 2 == 1 {
                // wait(&self.state, s);
                s = self.state.load(Relaxed);
            }
        }
    }

    pub fn write(&self) -> WriteGuard<T> {
        let mut s = self.state.load(Relaxed);
        loop {
            if s <= 1 {
                match self.state.compare_exchange(s, u32::MAX, Acquire, Relaxed) {
                    Ok(_) => return WriteGuard { rwlock: self },
                    Err(e) => {
                        s = e;
                        continue;
                    }
                }
            }
            if s % 2 == 0 {
                match self.state.compare_exchange(s, s + 1, Relaxed, Relaxed) {
                    Ok(_) => {}
                    Err(e) => {
                        s = e;
                        continue;
                    }
                }
            }
            let w = self.writer_wake_counter.load(Acquire);
            s = self.state.load(Relaxed);
            if s >= 2 {
                // wait(&self.writer_wake_counter, w);
                s = self.state.load(Relaxed);
            }
        }
    }
}

// ReadGuard / WriteGuard：Deref + Drop 中递减 state、必要时 wake（见全书完整版）
```

## 背诵

| # | 背这句 |
|---|--------|
| 1 | state 存读者数×2；偶=可进读者，奇=写者排队。 |
| 2 | 写者：偶数时 +1 拦新读者；wait 在 writer_wake_counter。 |
| 3 | 读者：仅偶数 CAS +2；奇数则 wait。 |
| 4 | MAX=写者持锁；最后读者 wake 写者。 |
