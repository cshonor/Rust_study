## 整体脉络

Rust 提供五种 **`std::sync::atomic::Ordering`**。每次 `load` / `store` / `fetch_*` / `compare_exchange` **都必须显式传入**。

| 强度 | Ordering | 一句话 |
|------|----------|--------|
| 最弱 | **Relaxed** | 只保单原子总序，不同步别的内存 |
| 常用 | **Release / Acquire** | 成对建立 HB，同步普通内存 |
| RMW | **AcqRel** | 读 Acquire + 写 Release |
| 最强 | **SeqCst** | 全局 SeqCst 时间线，极少用 |
