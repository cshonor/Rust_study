## 10.7.3 调试排错与并发陷阱

> 书 §10.7 · 索引：[10.7-teaching-materials.md](./10.7-teaching-materials.md) · 前：[10.7.2 性能](./10.7.2-performance-practices.md) · 后：[10.7.4 面试](./10.7.4-interview-cheatsheet.md)

---

| 手段 / 陷阱 | 说明 |
|-------------|------|
| **Miri** | MIR 层跑测试，抓 UB、非法指针、**数据竞争**（`cargo +nightly miri test`）。 |
| **虚假唤醒** | `park` / `Condvar::wait` 可无因而醒 → **`while !cond { wait }`**，勿单次 `if`（[1.8.4](../Chapter-01-Rust-Concurrency-Basics/1.8-parking-condvar/1.8.4-spurious-notify.md)）。 |
| **死锁 / 中毒** | 临界区要短；持锁 panic → **PoisonError**（[1.7.4](../Chapter-01-Rust-Concurrency-Basics/1.7-mutex-rwlock/1.7.4-lock-poisoning.md)）。 |
| **`if let` 锁作用域** | `if let Some(x) = mutex.lock().unwrap().pop()` 整段持锁 → 先 `let mut g = lock()`，**`drop(g)`** 再处理（[1.7.3](../Chapter-01-Rust-Concurrency-Basics/1.7-mutex-rwlock/1.7.3-mutex-guard.md)）。 |
| **无锁 ABA** | [10.3.2](./10.3.2-aba-recycling.md) — 用库勿徒手。 |
