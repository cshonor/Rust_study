## 1.4.2 数据竞争（Data Race）定义

### 定义（精简背诵）

> **无同步机制**下：**一个线程写**，**另一个线程同时读或写同一块内存** → **数据竞争** → **UB**。

```rust
// 伪代码：两个线程无同步地写/读同一 i32 —— UB
// 线程 A: *ptr = 1;
// 线程 B: let x = *ptr;
```

### 在安全 Rust 里

普通 **`&T` / `&mut T`** 的跨线程使用受 **`Sync`** 与借用规则约束，**很难**在 safe 代码里拼出经典数据竞争。

问题常出现在：

- **`unsafe`** 里手动拆引用；
- 误用 **`static mut`**；
- 与 **C / FFI** 互操作。
