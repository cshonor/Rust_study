## 四、学习建议

| 环境 | 做法 |
|------|------|
| 本地 dev | 自签名 + 手动信任，或 **mkcert** |
| 生产 | ACME 自动续期 + rustls + 正确 SAN 域名 |
| 学习顺序 | 先 Ch7 Tokio，再加 TLS |

---

## 五、新手易错（全章汇总）

1. **生产用自签名**或关闭证书校验。
2. **证书过期**无监控（LE 90 天）。
3. **native-tls 要 pfx、rustls 要 pem** — 混用配置失败。
4. **HTTP 明文传敏感数据**。
5. **用 ring 自造 TLS 替代 rustls**（除非明确知道在做什么）。
