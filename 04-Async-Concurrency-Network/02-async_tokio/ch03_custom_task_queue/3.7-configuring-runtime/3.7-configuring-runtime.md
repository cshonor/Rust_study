# 3.7 Configuring Our Runtime（配置运行时）

## 核心梳理

- **`Runtime` 结构体**：封装高/低优队列、worker 句柄、启动与关闭逻辑。
- **CPU 感知**：按 `std::thread::available_parallelism()` 等决定默认线程数。
- **环境变量**：如 `HIGH_NUM`、`LOW_NUM` 在启动时配置各队列 worker 数量，便于部署调优。

## 配置示例（概念）

```text
HIGH_NUM=4 LOW_NUM=2 ./my_async_app
```

启动时解析 env → 创建对应数量 Receiver clone + 线程。
