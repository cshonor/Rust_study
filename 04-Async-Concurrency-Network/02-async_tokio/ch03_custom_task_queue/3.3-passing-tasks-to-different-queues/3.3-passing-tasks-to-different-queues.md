# 3.3 Passing Tasks to Different Queues（任务分发到不同队列）

## 核心梳理

- **多优先级队列**：例如 `HIGH_QUEUE`（2 个 worker）与 `LOW_QUEUE`（1 个 worker）。
- **路由**：定义 `FutureType` / 优先级枚举 + 特型或 `spawn` 参数，将 Runnable 发送到对应 `Sender`。
- **效果**：高优任务更快获得线程资源；低优任务在独立通道排队。

## 设计表

| 队列 | Worker 数 | 典型任务 |
|------|-----------|----------|
| High | 多 | 延迟敏感、交互路径 |
| Low | 少 | 批处理、后台整理 |
