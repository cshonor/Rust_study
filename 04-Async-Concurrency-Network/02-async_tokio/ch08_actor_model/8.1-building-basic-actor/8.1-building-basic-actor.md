# 8.1 Building a Basic Actor（构建基础 Actor）

## Actor 形态

- 一个 **`tokio::spawn`** 的异步任务，**无限循环** `recv` 邮箱；
- 邮箱通常为 **`mpsc::Receiver<Message>`**；
- 状态**独占**在 Actor 内部，外部不直接共享 `&mut`。

## 请求—响应：`oneshot`

调用方在消息里携带 **`oneshot::Sender<T>`**，Actor 处理完后 `send` 结果，调用方 `await` **`oneshot::Receiver`**：

```rust
struct RespMessage {
    value: i64,
    responder: oneshot::Sender<i64>,
}
```

这样保持**异步**又不暴露 Actor 内部锁。

## 配套 Demo

`8.1-building-basic-actor/`：`mpsc` 邮箱 + `oneshot` 返回 `value + 1`。
