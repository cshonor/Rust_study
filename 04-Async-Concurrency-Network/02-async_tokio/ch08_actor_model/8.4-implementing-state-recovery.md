# 8.4 Implementing State Recovery for Actors（状态恢复）

## Writer Actor

- **独占**文件句柄，将 KV Actor 的变更**异步写入** `data.json`（或 WAL）；
- KV 处理事务时把指令**转发**给 Writer，避免多 Actor 争用同一文件。

## 启动恢复

Actor / 系统启动时 **读取 `data.json`**，重建内存 `HashMap`，实现崩溃或重启后的**状态恢复**。

## 注意

- 写盘频率 vs 性能：批量、定时刷盘；
- 与 §8.5 监督配合：Writer 卡死时心跳超时触发重启。
