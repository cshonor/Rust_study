# 10.1 Setting Up the Basics（基础设置）

## 目标

**零第三方异步库**（无 Tokio / smol / hyper），纯 **`std`** 理解 Future + Executor + Waker 如何拼成 TCP 服务。

## Cargo Workspace（书中）

拆分为独立 crate，例如：

- `client` / `server` / `data_layer` / `async_runtime`

## 序列化

手写 **Serialize / Deserialize** 字节流，满足纯标准库下的请求/响应协议（无 `serde` 依赖时的教学路径）。

## 配套 Demo

`10.1-setting-up-basics/`：占位；完整 workspace 按书搭建。
