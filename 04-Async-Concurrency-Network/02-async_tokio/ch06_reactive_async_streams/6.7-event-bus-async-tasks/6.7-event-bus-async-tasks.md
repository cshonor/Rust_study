# 6.7 Interacting with Our Event Bus via Async Tasks

> Demo：[6.7-event-bus-async-tasks/code/](./code/)

## Tokio 任务中消费

```rust
let handle = bus.subscribe().await;
tokio::spawn(async move {
    while let Some(ev) = handle.next().await {
        process(ev).await;
    }
});
```

句柄或任务结束时 **`Drop` 退订**（§6.6）。

## 垃圾回收 (GC) 任务

热路径上同步删除大队列可能拖慢 `publish`。书中用**后台 GC**：

- 批处理收集「已注销 id」；
- 在单独任务里 `remove` 并 `drop` 空队列；
- 降低 `publish` 持锁时间。

## 背压 (Backpressure)

生产快于消费时 `VecDeque` 膨胀：

| 策略 | 说明 |
|------|------|
| 有界队列 | 满则丢弃或阻塞 `publish` |
| 限速 | 令牌桶 |
| 监控 | 队列深度指标 |
