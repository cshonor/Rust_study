# 6.6 Building Our Event Bus Handle

> Demo：[6.6-event-bus-handle/code/](./code/)

## `EventHandle` 职责

订阅者拿到的句柄通常包含：

- **`subscriber_id: u64`**
- **`bus: Arc<EventBus>`**（或 `Weak` 避免与 runtime 循环引用）

对外 API：`async fn next(&mut self) -> Option<Event>`，内部从**自己的**队列 `pop_front`；`publish` 只由生产者调用。

## 内存泄漏（书中重点）

若句柄被 **drop 却不 unsubscribe**：

- `HashMap` 中条目仍在；
- 持续 `publish` → 该队列无限增长 → **OOM**。

## `Drop` 自动退订

```rust
impl Drop for EventHandle {
    fn drop(&mut self) {
        // 向总线发 Unsubscribe(self.id)，或 spawn 异步清理
    }
}
```

异步 `Drop` 不能直接 `.await`，常见模式：

- `spawn` 清理任务；
- 或同步从 map **移除 id** 并标记队列待 GC（§6.7）。
