# 4.3 Building an HTTP Connection（构建 HTTP 连接）

## 核心梳理

- **独立于 Tokio** 构建底层连接：实现 **`hyper::service::Service`**（或书中 Connector 类型）。
- **`CustomStream` 枚举**：区分 **TCP (HTTP)** 与 **TLS (HTTPS)** 封装。
- **职责**：接收 URI → 建连 → 把 **流** 交给 `hyper` 上层。

## 端口

HTTP **80** · HTTPS **443**（默认）。
