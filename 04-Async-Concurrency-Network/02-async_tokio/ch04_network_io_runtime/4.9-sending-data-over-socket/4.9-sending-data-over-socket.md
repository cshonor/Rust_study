# 4.9 Sending Data over the Socket（通过 Socket 发送数据）

> Demo：[4.9-sending-data-over-socket/4.9-sending-data-client-demo.rs](./code/4.9-sending-data-client-demo.rs)

## 核心梳理

- **客户端**：`TcpStream` + 注册 **`Interest::WRITABLE`**。
- **轮询**：端口可写前返回 `Pending`；就绪后 `write` / `write_all` 发送报文。
- **与 §4.8 对称**：监听侧 READABLE，连接侧 WRITABLE（也常需 READABLE 收响应）。

## 注意

教学 demo 需与 listener 端口配对；生产用 Tokio/mio 封装好的 `AsyncWrite` 更安全。
