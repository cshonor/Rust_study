# 9.3 The Decorator Pattern（装饰器模式）

## 核心

在不改核心业务的前提下，为 **Future** 动态加横切能力：日志、Fixture、会话等。

## 实现方式

- 包装 **Inner Future**，在 **`poll`** 前后插入行为；
- 涉及 **`Pin`** 投影：手写 `unsafe` 或 `Box::pin` / 组合子库；
- 结合 Cargo **`[features]`** 做零成本功能拔插（编译期裁剪）。
