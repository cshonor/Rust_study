# 9.7 Summary（总结）

## 核心梳理

| 模式 | 用途 |
|------|------|
| 隔离模块 | 低侵入接入遗留同步代码 |
| 瀑布 | 可复用的异步流水线 |
| 装饰器 | 横切能力拔插 |
| 状态机 | 限流与阶段控制 |
| 重试 + 退避 | 瞬时故障恢复 |
| 熔断 | 持续故障时快速失败 |

## 工程忠告

**过度设计**是最大雷区：遇实际问题再渐进引入，保持简单与可回滚。
