## 原书要点

### 本章三条主线（回顾）

| 线 | 节 | 核心 |
|----|:---:|------|
| **定义与边界** | 1.1 | 便携式二进制指令格式 · 宿主沙盒 · 非转译目标 · 非独立语言 |
| **架构机制** | 1.2 | 栈式机器 · `i32/i64/f32/f64` · 结构化控制流 · 线性内存（无 GC） |
| **动手验证** | 1.3 | wabt · WAT S-表达式 · 往返编译 · 安全/逆向启示 |

### 读完后应能回答

1. 为什么 Wasm 选**栈式**而不是寄存器架构？（体积、移植、实现成本）
2. 为什么浏览器里 Wasm 1.0 仍需要 **JavaScript 宿主**？
3. **线性内存**与宿主内存的隔离意味着什么？DOM 如何间接访问？
4. **wasm2wat** 对商业模块设计有什么警示？
