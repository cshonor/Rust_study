## 一句话

Wasm 用**栈式机器** + **四种数值类型** + **结构化控制流** + **线性内存** 换便携与安全——没有堆、没有 GC、没有原生字符串；有符号/无符号只在**具体操作**时区分。

---

## 原书要点

### 基于栈的机器 (Stack Machine)

与日常 **基于寄存器 (Register)** 的 CPU 不同，Wasm 用 **LIFO 栈** 做运算：

```text
i32.const 1    ;; 压栈 → [1]
i32.const 2    ;; 压栈 → [1, 2]
i32.add        ;; 弹出两数，结果压栈 → [3]
```

| 优势 | 说明 |
|------|------|
| **二进制小** | 指令编码紧凑 |
| **易移植** | 虚拟机实现简单，不绑定具体寄存器文件 |
| **与 LLVM 栈式 IR 有亲缘** | 对照 [06 Learn LLVM 17](../../../06_Compilers-and-LLVM-Learning/04_Learn-LLVM-17/README.md) 里 SSA/栈混合直觉 |

↔ [Crafting Interpreters · VM](../../../06_Compilers-and-LLVM-Learning/01_Crafting-Interpreters/part01_welcome/chapter02_map-of-the-territory/01-7-virtual-machine.md)

### 极其有限的数据类型 (Wasm 1.0 MVP)

**只有四种**：

| 类型 | 含义 |
|------|------|
| `i32` | 32 位整数 |
| `i64` | 64 位整数 |
| `f32` | 32 位浮点 |
| `f64` | 64 位浮点 |

- **没有**原生字符串、结构体、指针类型（指针在源码层是 `i32` **偏移**）。
- **有符号 / 无符号**不是类型属性，而是 **`i32.add` vs `i32.add_s` 等指令**在操作时才区分。

**HFT / Rust 对照**：Rust 编译到 Wasm 时，`String`、`Vec` 都在**线性内存 + 运行时辅助**里实现；热路径若要保持可预测，需像 [2.2 位布局](../chapter02_wasm_checkers/2.2-data-structure-constraints.md) 那样自己管布局。

### 安全的控制流

Wasm **没有**汇编式「盲跳 (blind jump)」到任意地址；控制流结构化：

| 指令 | 用途 |
|------|------|
| `block` / `loop` | 带标签的块与循环 |
| `if` / `else` | 条件分支 |
| `br` / `br_if` | 跳转到**已知**块标签 |

→ 验证器可证明控制流合法，利于沙箱安全（与 [附录 A2](../../appendix/A2-security.md) 衔接）。

### 线性内存 (Linear Memory)

| 特性 | 说明 |
|------|------|
| **模型** | 一块可按 **64KB 页 (page)** 增长的**连续字节** |
| **没有** | 传统 heap、`new`、**垃圾回收 (GC)**（1.0 MVP） |
| **安全边界** | 宿主可读写这块内存；模块**不能**访问宿主内存 |
| **「直接 DOM」** | 错觉——实际是共享线性内存 + **宿主 (JS) 代为操作 DOM** |

```text
Wasm 模块                    宿主
┌─────────────────┐         ┌──────────────┐
│ linear memory   │ ◄──────►│ JS / Rust    │
│ (i32 偏移寻址)   │  import │ 读写字节视图  │
└─────────────────┘         └──────────────┘
```

↔ Rust 侧：[Nomicon · 布局](../../../04-Rust-Nomicon/README.md) · [RFR Ch2 类型布局](../../../02-RFR/Chapter-02-Types/README.md)

## WAT / 工具 / 代码锚点

| 项 | 说明 |
|----|------|
| 最小加法 WAT | 见 [1.3](./1.3-building-wasm-app.md) 手写示例 |
| `memory` 段 | `(memory 1)` 声明至少 1 页；`(data (i32.const 0) "hi")` 初始化 |

示例片段（概念）：

```wat
(module
  (func $add (param i32 i32) (result i32)
    local.get 0
    local.get 1
    i32.add)
  (export "add" (func $add)))
```
