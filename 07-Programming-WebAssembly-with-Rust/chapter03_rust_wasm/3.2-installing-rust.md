## 一句话

用 **`rustup target add wasm32-unknown-unknown`** 为工具链添加 Wasm 编译目标，即可在本地把 Rust **直接编译成跨平台 `.wasm` 模块**。

---

## 原书要点

### 安装 WebAssembly 编译目标

```bash
rustup target add wasm32-unknown-unknown
rustup target list --installed   # 确认含 wasm32-unknown-unknown
```

| 项 | 说明 |
|----|------|
| **triple** | `wasm32-unknown-unknown` — 32 位 Wasm，无特定 OS（unknown-unknown） |
| **与 host 区别** | 默认 `x86_64-pc-windows-msvc` 等编译本机 exe；此 target 产出 **`.wasm`** |
| **std** | 仍链接 Rust **标准库**（除非 `#![no_std]`）→ 体积大于 Ch2 手写模块 |

### 常用编译命令（预览）

```bash
cargo build --target wasm32-unknown-unknown --release
# 产物通常在 target/wasm32-unknown-unknown/release/*.wasm
```

详细 Hello 流程 → [3.3](./3.3-hello-wasm-rust.md)

### 与 Layer 1 对齐

[Layer 1](../../layer01_rust-llvm-to-wasm/README.md) 在同一 target 上并排：

```bash
cargo rustc --target wasm32-unknown-unknown -- --emit=llvm-ir   # 可选：对照 06 IR
```
