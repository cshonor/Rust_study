## 一句话

本章用**纯 WAT/wast** 从零实现可玩的跳棋（Checkers / Draughts）模块——先理解棋盘规则与状态，再在 [1.2 线性内存](../chapter01_wasm_fundamentals/1.2-wasm-architecture.md) 上手工编码。

---

## 原书要点

### 本章目标

| 点 | 说明 |
|----|------|
| **语言** | **纯 WebAssembly 文本格式 (wast/wat)**，不用 Rust/C 编译器 |
| **产物** | 一个可导出给宿主调用的 **Checkers 游戏模块** |
| **难点** | Wasm **没有**数组、结构体、枚举、GC——一切状态靠**线性内存 + 位打包** |

### 跳棋（Checkers / Draughts）规则概要

| 要素 | 说明 |
|------|------|
| **棋盘** | 8×8，通常只用深色格走子 |
| **棋子** | 黑方 / 白方；到达对手底线 **加冕 (Crowned)**，可双向移动 |
| **移动** | 斜走；跳吃（本章实现中会涉及合法性与加冕判断） |
| **状态** | 当前回合、每格棋子类型与是否加冕 |
