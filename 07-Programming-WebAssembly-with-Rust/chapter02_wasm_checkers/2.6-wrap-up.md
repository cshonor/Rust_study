### 本章四条主线（回顾）

| 线 | 节 | 核心 |
|----|:---:|------|
| **数据结构** | 2.2 | `offset=(x+y*8)*4` · 位标志 0/1/2/4 |
| **棋盘与回合** | 2.3 | `$getPiece`/`$setPiece`/`$inRange` · `$currentTurn` |

| **测试与体积** | 2.5 | export 边界 · ~853B · 256B memory |

### 读完后应能回答

1. 为什么 8×8 棋盘要用 **`(x + y*8)*4`** 而不是「二维数组」？
2. 加冕黑棋的值 **5** 是怎么用位标志拼出来的？
3. 为什么 **`$setPiece` 不应 export**？
4. 853 字节模块 vs 未来 Rust 版本，你预期差在哪（runtime、allocator、panic）？
