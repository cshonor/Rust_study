# 6.1 How to Be a Good Host

> 第 6 章索引：[README](./README.md) · 后：[6.2-interpreting-wasm-rust](./6.2-interpreting-wasm-rust.md)

---

## 一句话

Wasm 可移植靠 **模块↔宿主契约**；任何宿主必须履行 **加载验证、暴露 export、满足 import、解释执行、模块隔离** 五项职责 — 浏览器只是宿主之一。

### 好宿主的 5 项核心职责

| # | 职责 | 说明 |
|:---:|------|------|
| **1** | **Load and Validate 加载与验证** | 读 `.wasm` 二进制 · 验证格式合法 — **防恶意模块**的第一道防线 |
| **2** | **Expose Exports 暴露导出** | 宿主侧 **glue**，安全调用模块 **export** 函数（如 `$move`、`add`） |
| **3** | **Satisfy Imports 满足导入** | 模块 **import** 的外部函数必须由宿主提供；缺实现则 **实例化失败** |
| **4** | **Interpret / Execute 解释执行** | 维护运行时状态 · 处理 trap/错误 · 执行 Wasm 指令 |
| **5** | **Module Isolation 模块隔离** | **沙盒**：一模块崩溃不拖垮其它模块 · 不能越权读宿主/他模块私有内存 |

```text
┌──────────── Host (Rust 控制台) ────────────┐
│ ① load/validate  ② call exports            │
│ ③ provide imports  ④ VM/runtime  ⑤ sandbox│
│         ▲ import          export ▼           │
│    ┌────┴──── .wasm 模块 ────┐              │
│    │  linear memory (隔离)    │              │
│    └─────────────────────────┘              │
└─────────────────────────────────────────────┘
```
