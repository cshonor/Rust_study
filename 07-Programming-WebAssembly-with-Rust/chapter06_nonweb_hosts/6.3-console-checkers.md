# 6.3 Building a Console Host Checkers Player

> 第 6 章索引：[README](./README.md) · 前：[6.2-interpreting-wasm-rust](./6.2-interpreting-wasm-rust.md) · 后：[6.4-wrap-up](./6.4-wrap-up.md)

---

## 一句话

把 **Ch2/Ch3 跳棋 `.wasm`** 原封不动搬进 **Rust 控制台宿主**：实现 **`ModuleImportResolver` + `Externals`** 满足 `notify_*` import，再封装 Game API，并**直接读模块 linear memory** 在终端画 **ASCII 棋盘**。

---

## 原书要点

### 实战目标

| 点 | 说明 |
|----|------|
| **同一份 `.wasm`** | 浏览器 JS 宿主 ↔ 本章 Rust 控制台 — **零修改** |
| **可移植性** | 证明 Wasm 二进制跨宿主切换 |

来源模块 → [Ch2 wast](../chapter02_wasm_checkers/README.md) 或 [Ch3 Rusty Checkers](../chapter03_rust_wasm/README.md)

### 宿主三层结构

```text
┌─────────────────────────────────────┐
│  Game API（Rust 惯用封装）            │  ← move_piece(), render_board()
├─────────────────────────────────────┤
│  Externals（中央调度）                │  ← 拦截 import 调用
├─────────────────────────────────────┤
│  ModuleImportResolver（解析导入）     │  ← 链接 notify_* 签名与索引
├─────────────────────────────────────┤
│  wasmi ModuleInstance + linear mem   │
└─────────────────────────────────────┘
```

#### 1. 解析导入 — `ModuleImportResolver`

| 任务 | 说明 |
|------|------|
| Wasm 跳棋 **import** | `$notify_piecemoved` · `$notify_piececrowned`（[Ch2/Ch3](../chapter03_rust_wasm/3.5-rusty-checkers-ffi.md)） |
| Resolver | 提供函数 **签名** · 分配 **调用索引** · **实例化时**满足 import 表 |
| 失败 | 缺 import → 模块 **无法** `ModuleInstance::new` |

#### 2. 满足外部调用 — `Externals` trait

| 角色 | 说明 |
|------|------|
| **中央调度器** | Wasm 执行 `call import` 时，runtime 转到宿主 Rust 函数 |
| **宿主侧状态** | 可单独记录胜负、生命值等 — **防止 Wasm 模块作弊**改全局结果 |
| 通知回调 | 在 `Externals` 里 `println!` 或更新 Game 状态，再触发重绘 |

↔ Layer 3：**宿主**持有实盘风控计数，**策略 wasm** 只能 export 信号 → [layer03](../../layer03_quant-ma-strategy/README.md)

#### 3. 封装 Game API + 读 linear memory

| 技巧 | 说明 |
|------|------|
| **Rust API 包装** | 避免每次 `invoke_export` 手写参数 |
| **直接读 linear memory** | 宿主控制内存视图 · 按 Ch2 公式 `offset=(x+y*8)*4` 读每格 `i32` |
| **ASCII 渲染** | 终端打印棋盘 — **不依赖** JS/DOM |

```text
宿主 read memory[offset] → 0/1/2/4/5/6 位标志
  → 映射为 '.', 'b', 'B', 'w', 'W' 等字符
  → println! 网格
```

这与 Layer 2「宿主 fetch JSON、Wasm 解析」相反：**宿主直接窥视 Wasm 内存**（仅当布局契约稳定且受控时安全）。

### 与浏览器宿主对比

| | JS 宿主 (Ch3) | Rust 控制台 (Ch6) |
|---|---------------|-------------------|
| import 实现 | `env.notify_*` 回调 | `Externals` |
| 棋盘 UI | DOM / canvas | **ASCII + memory 读取** |
| 网络 | 可选 fetch | 无 |
