## 一句话

编写 **`dumbotrs`**（标靶）与 **`Rook`**（巡逻战车）两个 **Wasm 机器人**，用 **`consolerunner`** 在终端跑 **10 万帧** 对战，经 **channel 录事件** 并支持 **控帧回放** — 验证引擎与 wasmi 多实例加载。

---

## 原书要点

### 机器人一：dumbotrs（标靶）

| 行为 | 目的 |
|------|------|
| **10% 速度向北 `drive`** | 简单运动 |
| 循环 **`damage` 等状态查询** | 压测宿主 **并发读** |
| 撞墙后不动 | 测试边界与停止条件 |

→ 引擎 **读多写少**、锁粒度是否足够的压力测试。

### 机器人二：Rook（战车）

| 行为 | 说明 |
|------|------|
| 移动到 **战场中心** | 几何 + drive |
| **东西向巡逻** | 固定模式 |
| **`scan` 四周** | 发现敌人 |
| **`cannon` 直到消灭** | 完整 **感知→决策→开火** 闭环 |

→ 调用全套 [8.2 WARoS API](./8.2-waros-api.md)。

### 编译与加载

```text
robot logic (Rust) → wasm32-unknown-unknown → dumbotrs.wasm / rook.wasm
  → wasmi 加载 N 个实例 → 同一 import 表指向引擎 ECS
```

### consolerunner 与回放

| 组件 | 作用 |
|------|------|
| **`consolerunner`** | CLI · 将多个 `.wasm` 投入竞技场 |
| **100,000 帧** | 长对战 · 压测性能 |
| **Channels 录事件** | 导弹 · 碰撞 · 死亡等 → 可序列化 |
| **控帧回放** | 调节 FPS 重放赛况（类似 demo 播放器） |

↔ [7.4 · mpsc 与 FPS](../chapter07_iot/7.4-raspberry-pi-host.md) · Layer 3 回测 **tick 回放**

### 与 Layer 3

| WARoS | Layer 3 |
|-------|---------|
| dumbotrs | 固定策略 smoke test |
| Rook | 简单 **MA 交叉** 策略原型 |
| consolerunner 100k 帧 | **CSV 回测** + 权益曲线 |
| 事件 channel | 成交/信号日志 |

→ [layer03/demo/fixtures/sample_ticks.csv](../../layer03_quant-ma-strategy/README.md)
