## 一句话

WARoS 可上 **云端**：开发者上传 **任意语言编译的 `.wasm` 机器人**，后台 **Serverless/FaaS** 跑匹配引擎，经 **Kafka / WebSocket** 把战况推到网页 — 与 [附录 A1 · Serverless](../../appendix/A1-serverless.md) 呼应。

---

## 原书要点

### 云端架构愿景

```text
开发者上传 robot.wasm (Rust/C/Go/… 编译)
        ↓
云端 WARoS Match Engine（多租户 · 沙箱）
        ↓
Kafka / WebSocket 事件流
        ↓
Web 观众页（Yew / JS · 实时回放）
```

| 组件 | 说明 |
|------|------|
| **上传门户** | 网站接收 `.wasm` · 校验签名 [A2](../../appendix/A2-security.md) |
| **后台引擎** | 同 [8.3 ECS + wasmi](./8.3-match-engine.md) · 水平扩展 |
| **Serverless / FaaS** | 单场对战 = 一次函数/容器 invocation |
| **Kafka** | 持久化事件流 · 多订阅者 |
| **WebSocket** | 前端 **实时** 观战 |

### 与 Ch5 Yew 的衔接

| 能力 | 组合 |
|------|------|
| [Yew 聊天/SPA](../chapter05_yew/5.2-live-chat-app.md) | 换成 **观战仪表盘** |
| ChatEngine 推送 | **WebSocket 战报** |

### Layer 3 / HFT 延伸（笔记向）

| WARoS 云 | 量化云（概念） |
|----------|----------------|
| 上传 robot.wasm | 上传 **策略 wasm**（验签） |
| 匹配对战 | **纸面撮合 / 回测队列** |
| 实时推送 | **监控面板** · 非实盘下单 |

实盘低延迟路径仍走 [6/7 边缘宿主](../chapter07_iot/README.md)，云侧偏 **研究与管理**。
