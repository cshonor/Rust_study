# 8.2 Designing the WARoS API

> 第 8 章索引：[README](./README.md) · 前：[8.1-homage-crobots](./8.1-homage-crobots.md) · 后：[8.3-match-engine](./8.3-match-engine.md)

## 一句话

**WARoS API** = 把 Crobots 内置函数（**`scan` / `cannon` / `drive` / `damage` / 三角函数** 等）变成机器人 Wasm 的 **`extern "C"` import** — 宿主实现、模块调用，形成标准 **FFI 契约**。

---

## 原书要点

### 机器人模块 ↔ 宿主：FFI 契约

| 方向 | 内容 |
|------|------|
| **Wasm import** | 机器人调用宿主提供的 **API**（感知战场、开火、移动） |
| **Wasm export** | 机器人入口（如每帧 **`robot()`** / `main` — 以书为准） |
| **ABI** | **`extern "C"`** · 参数多为 **`f64` / `i32`** 等 Wasm 原生类型 |

### 典型 import（概念，名称以原书为准）

| API | 用途 |
|-----|------|
| **`scan`** | 雷达扫描 · 角度 → 距离 |
| **`cannon`** | 指定角度与射程开火 |
| **`drive`** | 速度 + 方向 · 驱动马达 |
| **`damage`** | 查询自身受损 |
| **三角函数** | `sin` / `cos` / … — 瞄准与几何 |

```rust
// 机器人 Wasm 侧（概念）
#[link(wasm_import_module = "env")]
extern "C" {
    fn scan(angle: f64) -> f64;
    fn cannon(angle: f64, range: f64) -> i32;
    fn drive(speed: f64, direction: f64);
    fn damage() -> i32;
}
```

### 与前面章节的 FFI 演进

| 章节 | FFI 方式 |
|------|----------|
| Ch2 wast | 手写 import `notify_*` |
| Ch3 | `extern "C"` + wasmi/JS |
| Ch4 | **wasm-bindgen** |
| **Ch8 WARoS** | **大规模标准化 import 表** — 多模块共享同一 API |
