## 一句话

**WARoS** 致敬 1985 年 Tom Poindexter 的 **《Crobots》** — 在 **1000×1000m 竞技场** 里多坦克对战；本章用 **Rust 引擎宿主 + 多 Wasm 机器人** 复刻并升华这一经典。

---

## 原书要点

### 全书实战高潮

| 前面章节 | 第 8 章 |
|----------|---------|
| Ch1～2 手写 wast · Ch3～5 Rust/JS/Yew | **多 Wasm 模块 + 复杂 Rust 宿主** |
| Ch6 单模块控制台 · Ch7 IoT 插件 | **多机器人竞技场 + ECS 并发** |
| 单线程/简单并发 | **全书 Rust 最深、最复杂的一章** |

### Crobots / WARoS 游戏规则（概要）

| 要素 | 规则 |
|------|------|
| **竞技场** | **1000 × 1000** 米 |
| **机器人（坦克）** | 驱动马达 · **火炮**（最多 **2** 枚导弹同时在飞）· **雷达**（半径 **700m**） |
| **伤害** | 撞墙 · 被炮弹击中 → **扣生命值** |
| **目标** | 存活并消灭对手 |

### 架构预览

```text
┌──────────── WARoS Match Engine (Rust) ────────────┐
│  Game Loop · ECS · wasmi · consolerunner        │
│       ▲ import scan/cannon/drive/...             │
│  ┌────┴────┐  ┌─────────┐  ┌─────────┐         │
│  │ robot A │  │ robot B │  │ rook.wasm│  ...    │
│  │  .wasm  │  │  .wasm  │  │         │         │
│  └─────────┘  └─────────┘  └─────────┘         │
└─────────────────────────────────────────────────┘
```

### Layer 3 映射

| WARoS | Layer 3 量化 |
|-------|--------------|
| **竞技场 tick** | **行情 tick / 回测帧** |
| **Wasm 机器人** | **`ma_strategy.wasm`** |
| **比赛引擎** | **策略调度器**（喂价 · 收信号） |
