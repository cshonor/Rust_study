## 原书 Part / 章 → 三层映射

### Part I · 建立基础 (Building a Foundation)

| 章 | 核心子主题（原书） | 三层 |
|----|-------------------|------|
| **1** WebAssembly 基础 | 介绍 Wasm · 架构（栈式 VM、线性内存）· 构建首个 Wasm 应用 | **Layer 1** 全书主线 |

### Part II · 与 JavaScript 交互 (Interacting with JavaScript)

| 章 | 核心子主题（原书） | 三层 |
|----|-------------------|------|
| **3** Rust 涉足 Wasm | 安装 Rust · Hello Wasm · Rust 跳棋引擎 · Wasm 接口 · JS 中游玩 | **Layer 2**（Rust → Wasm + 导出 API） |
| **4** 与 JS 集成 | 更好的 Hello · **Rogue** 游戏 · 深入实验 | **Layer 2**（**wasm-bindgen** · 宿主 fetch 订单簿） |

### Part III · 非 Web 宿主 (Working with Non-Web Hosts)

| 章 | 核心子主题（原书） | 三层 |
|----|-------------------|------|

| **7** Wasm 物联网 | 通用指示器模块 · **ARM** · **树莓派** 托管 | **Layer 3**（边缘节点跑同一策略模块） |
