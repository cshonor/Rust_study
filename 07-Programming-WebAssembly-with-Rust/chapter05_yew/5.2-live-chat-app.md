# 5.2 Building a Live Chat Application

> 第 5 章索引：[README](./README.md) · 前：[5.1-getting-started-yew](./5.1-getting-started-yew.md) · 后：[5.3-wrap-up](./5.3-wrap-up.md)

## 一句话

本章实战：**消息区 + 在线用户 + 输入/连接** 的实时聊天 SPA；用 Yew **Services** 封装第三方 **PubNub ChatEngine (JS)**，以 **`js!` 宏 + 防腐层** 把异步 JS 隔离成 Rust 可调的强类型服务。

---

## 原书要点

### 应用结构

| UI 区域 | 功能 |
|---------|------|
| **消息展示区** | 聊天历史滚动列表 |
| **在线用户列表** |  presence / 上下线 |
| **输入 / 连接区** | 昵称、房间、发送消息 |

多个 **Component** 组成组件树，共享或订阅同一 **Chat Service**。

### 集成第三方 JavaScript：PubNub ChatEngine

| 决策 | 说明 |
|------|------|
| **不在 Rust 里重写全套网络** | 复用 **ChatEngine**（PubNub 生态 JS SDK） |
| **Yew Services** | 长生命周期、与 UI 解耦的后台任务/外部 I/O |
| 原则 | 与 Ch4 Rogue 相同 — **JS 做生态擅长的事**（此处：实时 Pub/Sub） |

### 防腐层 (Anti-Corruption Layer)

```text
┌─────────────────────────────────────────┐
│  Yew Components (Rust 强类型 Msg)        │
└─────────────────┬───────────────────────┘
                  │ 干净 API：send_chat, on_message
┌─────────────────▼───────────────────────┐
│  ChatService (Rust)                      │  ← 防腐层
│  js! 宏 · 回调绑定 · Promise → Msg       │
└─────────────────┬───────────────────────┘
                  │
┌─────────────────▼───────────────────────┐
│  ChatEngine (JavaScript)                 │
└─────────────────────────────────────────┘
```

| 技术 | 作用 |
|------|------|
| **`js!` 宏** | 内联调用 JS 表达式 / 注册回调 |
| **回调绑定** | JS 异步事件 → 转成 Yew **`Msg`** 投递回组件 |
| **类型隔离** | UI 只 `match Msg::ChatReceived { .. }`，不见裸 JS 对象 |

与 [4.3 · js_sys/web_sys](../chapter04_js_integration/4.3-experimenting-further.md) 对比：Yew Service 是**架构级**封装，而非散落的 `extern`。

### UI 消息与后端对接

典型 **`Msg`** 变体（概念）：

| Msg | 来源 |
|-----|------|
| `SendChat(String)` | 用户点击发送 |
| `ChatReceived { user, text }` | ChatEngine 回调 |
| `UserOnline` / `UserOffline` | presence 事件 |
| `Connect { nickname }` | 连接按钮 |

`update()` 中更新 **`Model`**（消息 Vec、用户 HashSet 等），子组件通过 props / 全局 service 刷新。

### Layer 3 映射：聊天 → 回测

| 原书聊天 | Layer 3 回测 SPA |
|----------|------------------|
| ChatEngine 推送消息 | **CSV 回放 / WebSocket 行情**（或本地 tick 定时器） |
| `Msg::ChatReceived` | `Msg::Tick(f64)` · `Msg::TradeSignal` |
| 消息列表 UI | **权益曲线 + 成交表 + MA 参数** |
| ChatService | **`BacktestService`** 驱动 [ma_core.wasm](../../layer03_quant-ma-strategy/README.md) |
