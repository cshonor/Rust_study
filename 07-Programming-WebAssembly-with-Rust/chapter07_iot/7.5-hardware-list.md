## 原书要点

### 所需硬件（书中）

| 设备 | 说明 |
|------|------|
| **Raspberry Pi 2 或更高** | 运行 **ARM Linux 宿主**（[7.3](./7.3-rust-arm.md) 交叉编译产物） |
| **Pimoroni Blinkt!** | GPIO 扩展板 · **8 个可编程 RGB LED** · 适合 indicator 进度条 / 脉冲动画 |
| 辅材 | SD 卡 · 电源 · （可选）网络 SSH 部署 |

### Blinkt 与 GIMS 的对应

| GIMS API | 硬件 |
|----------|------|
| **`set_led(index, r, g, b)`** | index 0..7 对应 Blinkt 8 灯 |
| Battery 模块 | 8 灯作 **电量条** |
| Pulsing 模块 | 8 灯作 **扫描动画** |

### 无硬件时的学习路径

```text
7.2 cargo test（Wasm 逻辑）
  → 7.4 PC 编译（println mock）
  → 7.3/7.4 有 Pi 再上 blinkt
```
