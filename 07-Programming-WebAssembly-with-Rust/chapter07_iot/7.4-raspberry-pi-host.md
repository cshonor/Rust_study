# 7.4 Hosting Indicator Modules on a Raspberry Pi

> 第 7 章索引：[README](./README.md) · 前：[7.3-rust-arm](./7.3-rust-arm.md) · 后：[7.5-hardware-list](./7.5-hardware-list.md)

## 一句话

树莓派 **Rust 宿主**：**mpsc + 超时** 锁 **10 FPS** 动画、`#[cfg]` 在 ARM 上链 **blinkt** / 在 PC 上 **println 调试**，并实现 **Wasm 模块热重载**（覆盖 `.wasm` 文件即切换指示器）。

---

## 原书要点

### 宿主并发架构

| 需求 | 实现 |
|------|------|
| 传感器输入 · 动画 · 硬件 IO 并发 | **多线程** |
| 线程间消息 | **`mpsc` channel** |
| 固定 **10 FPS** | channel **`recv_timeout`** — 超时则推进下一帧 / 调用 `apply()` |
| 避免锁死 | 用 **超时驱动** 而非复杂锁（对照 [05-atomic](../../../05-Async-Concurrency-Network/01-atomic/README-学习区.md) 并发模型） |

```text
Thread A: 读传感器 → sensor_update → 发 Msg
Thread B: 10 FPS tick → apply() → set_led → blinkt
Thread C: 文件监控 → 热重载 wasm
```

### 条件编译 `#[cfg(...)]`

| 编译目标 | 行为 |
|----------|------|
| **ARM + 特性 `hardware`** | 链接 **`blinkt`** · 真实 8 LED RGB |
| **PC 开发机** | **`println!` 模拟** LED 颜色 — 无需硬件即可跑宿主 |

```rust
#[cfg(all(target_arch = "arm", feature = "hardware"))]
fn set_led_hw(i: u8, r: u8, g: u8, b: u8) { /* blinkt */ }

#[cfg(not(all(target_arch = "arm", feature = "hardware")))]
fn set_led_hw(i: u8, r: u8, g: u8, b: u8) {
    println!("LED {} = ({},{},{})", i, r, g, b);
}
```

### 热重载 (Hot Reloading)

| 步骤 | 说明 |
|------|------|
| **文件系统监控** | 监视 indicator `.wasm` 路径 |
| **检测到新文件** | **安全暂停** 当前模块 |
| **加载新实例** | 如 **battery → pulsing** 切换 |
| **继续运行** | **无需重启** Pi 或宿主进程 |

```text
运行中: battery_indicator.wasm
  → 覆盖为 pulsing_indicator.wasm
  → 宿主 reload
  → Knight Rider 动画（进程不退出）
```

**Layer 3 映射**：边缘节点 **热换策略 wasm** 而不重启撮合/风控进程 — 须配合 [附录 A2 签名](../../appendix/A2-security.md)。

### 与 Ch6 宿主模式对比

| Ch6 跳棋控制台 | Ch7 树莓派 |
|----------------|------------|
| wasmi · Externals | 同类 runtime + **`set_led` import** |
| 读 memory 画 ASCII | **import 驱动硬件** |
| 静态 wasm | **热重载** |
