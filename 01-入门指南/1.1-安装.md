# 1.1 安装

## 安装 Rust

通过 **rustup** 下载 Rust，这是一个管理 Rust 版本和相关工具的命令行工具。需要互联网连接。

> 如果不想用 rustup，可参阅 [Rust 其他安装方法](https://forge.rust-lang.org/infra/other-installation-methods.html)。

---

## 命令行符号说明

- `$` — 命令行的开始，**不需输入**
- 不以 `$` 开头的行 — 表示命令的输出
- PowerShell 专用示例使用 `>` 而非 `$`

---

## Linux / macOS 安装

```bash
$ curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh
```

安装成功会显示：`Rust is installed now. Great!`

### 链接器

Rust 需要**链接器**。若遇链接器错误，可安装 C 编译器：

- **macOS**：`$ xcode-select --install`
- **Linux（如 Ubuntu）**：安装 `build-essential` 包

---

## Windows 安装

访问：https://www.rust-lang.org/zh-CN/tools/install

- 可能需安装 **Visual Studio C++ build tools**
- 推荐安装 VS 2019 构建工具
- 勾选：**C++ build tools**、**Windows 10 SDK**、**英文语言包**

---

## 更新与卸载

```bash
# 更新到最新版本
$ rustup update

# 卸载 Rust 和 rustup
$ rustup self uninstall
```

---

## 验证安装

```bash
$ rustc --version
```

输出示例：`rustc x.y.z (abcabcabc yyyy-mm-dd)`

> Windows 若无法识别命令，检查 Rust 是否已加入 `%PATH%`

---

## 本地文档

```bash
$ rustup doc
```

在浏览器中打开本地文档，便于离线查阅 API。
