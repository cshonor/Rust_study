# 1.2 Hello, World!

按照传统，学习新语言时先编写一个输出 "Hello, world!" 的简单程序。

> **注意**：本书假定你熟悉基本命令行。Rust 对编辑器、工具、代码位置无特殊要求，可自由选用 IDE。很多 IDE 已提供 Rust 支持。

---

## 创建项目目录

建议在**主目录**下创建 `projects` 目录存放项目。

### Linux / macOS / PowerShell

```bash
$ mkdir ~/projects
$ cd ~/projects
$ mkdir hello_world
$ cd hello_world
```

### Windows CMD

```cmd
> mkdir "%USERPROFILE%\projects"
> cd /d "%USERPROFILE%\projects"
> mkdir hello_world
> cd hello_world
```

---

## 编写和运行 Rust 程序

1. 创建源文件 `main.rs`（Rust 文件以 `.rs` 结尾）
2. 多词文件名用下划线：`hello_world.rs` ✓  `helloworld.rs` ✗

### main.rs

```rust
fn main() {
    println!("Hello, world!");
}
```

### 编译与运行

**Linux / macOS：**
```bash
$ rustc main.rs
$ ./main
Hello, world!
```

**Windows：**
```cmd
> rustc main.rs
> .\main.exe
Hello, world!
```

## Rust 程序剖析

### main 函数

```rust
fn main() {

}
```

- `main` 是**主函数**，可执行程序的入口，第一个运行的代码
- 无参数、无返回值
- 函数体用大括号 `{}` 括起
- 风格建议：左大括号与函数声明同一行，中间留空格

> 可用 `rustfmt` 工具自动格式化代码（已随 Rust 安装）

### println! 宏

```rust
    println!("Hello, world!");
```

| 要点 | 说明 |
|------|------|
| 缩进 | 使用 **4 个空格**，不用 Tab |
| `!` | 表示**宏**，不是普通函数（宏详见第 19 章） |
| 字符串 | `"Hello, world!"` 作为参数传给 `println!` |
| 分号 | `;` 表示表达式结束 |

## 编译和运行是独立的步骤

### 编译

```bash
$ rustc main.rs
```

- 类似 `gcc` / `clang`
- 生成二进制可执行文件

### 编译后的文件

| 平台 | 文件 |
|------|------|
| Linux / macOS | `main`、`main.rs` |
| Windows (PowerShell) | `main.exe`、`main.pdb`、`main.rs` |
| Windows (CMD) | `main.exe`、`main.pdb`、`main.rs` |

- `main.exe`（或 `main`）— 可执行文件
- `main.pdb` — 调试信息（仅 Windows）

### 运行

```bash
$ ./main          # Linux / macOS
$ .\main.exe      # Windows
```

---

## Rust 是预编译语言

- **预编译 (AOT)**：先编译成可执行文件，分发后他人**无需安装 Rust** 即可运行
- 与 Ruby、Python、JavaScript 不同：分发 `.rb`/`.py`/`.js` 需要对方安装对应运行环境
- 代价：需单独执行编译步骤，不能“一条命令”直接运行源码
