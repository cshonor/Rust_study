# 10.3.1 OS 相关适配层

> 章索引：[第 10 章](./README.md) · 前：[10.3](./10.3-process-mgmt.md) · 后：[10.3.2 · 对外 API](./10.3.2-process-public-api.md)

## 一句话

**`std::sys::*::process`** 是 **全 `unsafe` 的 OS 绑定** — Unix 走 **`fork → fd 重定向 → execvp → waitpid`**，Windows 走 **`CreateProcessW + 句柄继承 + WaitForSingleObject`**；通过 **`cfg` 静态分发** 对上暴露 **同名 `imp` 函数**，与 [9.2 · sys 分层](../chapter09_userspace_std_basics/9.2-engineering-trick.md) 同构。

---

## 原书要点

### 分层结构

```text
上层 std::process（Command / Child / ExitStatus）
        ↓ AsInner / FromInner / IntoInner
底层 sys/process/imp
    ├─ unix/   fork · exec · dup2 · waitpid
    └─ windows/ CreateProcess · 句柄继承 · Wait
```

### Unix 链路

| 步骤 | 说明 |
|------|------|
| **`fork()`** | 子进程副本 · 新 pid |
| **子进程内** | **dup2** 绑定 stdin/stdout/stderr · **关闭** 父侧多余 pipe 端 |
| **`execvp()`** | 替换程序镜像 · **保留** 已配置 fd 表 |
| **`waitpid()`** | 阻塞回收 · **`ExitStatus`** · 防 zombie |

### Windows 链路

| 步骤 | 说明 |
|------|------|
| **`CreateProcessW`** | **无 fork** · 一次创建 |
| **句柄继承** | 管道 **Handle** 复制到子进程 stdio · **`bInheritHandle`** |
| **等待 + 退出码** | **`WaitForSingleObject`** · 统一映射到 **`ExitStatus`** |

### 适配层职责清单

| 职责 | 不在此层 |
|------|----------|
| 平台 syscall / libc | **`Command::arg` 链式语义** |
| errno → **`io::Error`** | 用户文档 |
| **`pre_exec`（unix）** | — |
| fd **close-on-exec** | — |
| [10.1](./10.1-anonymous-pipe.md) **`pipe` + dup2** | — |

### 两层互通 Trait

```rust
// 安全层取出底层对象
impl AsInner<imp::Stdio> for Stdio {
    fn as_inner(&self) -> &imp::Stdio { &self.inner }
}
// 从裸 fd 构造 — 调用者保证句柄有效
unsafe impl FromRawFd for Stdio {
    unsafe fn from_raw_fd(fd: RawFd) -> Self { ... }
}
```

**凡直接操作裸系统资源的转换均 `unsafe`** — UB 风险显式交给调用者；业务代码应优先 **`Stdio::piped()` / `from_file`**。

### unsafe 集中区

spawn 后 **仅子进程可见的 fd 布局**、**exec 前单线程约束** — 原书在此章展开证明义务；对照 [10.4 · 分平台精简注释](./10.4-unsafe-source-walkthrough.md)。
