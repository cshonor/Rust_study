# 10.3.1 OS 相关适配层

> 章索引：[第 10 章](./README.md) · 前：[10.3](./10.3-process-mgmt.md) · 后：[10.3.2 · 对外 API](./10.3.2-process-public-api.md)

## 一句话

**`std::sys::unix::process` / `windows::process`** 等 **OS 适配层** 抹平 **`fork+execve+waitpid`** 与 **`CreateProcessW`** 差异 — 负责 **真正 SYSCALL**、**fd 继承表**、**环境块/cwd**；对上层只暴露 **`io::Result`** 与 **原始 handle**。

---

## 原书要点（目录脉络 · 源码细节待刷书 p.245+）

### 与 [9.2 · sys 分层](../chapter09_userspace_std_basics/9.2-engineering-trick.md) 同构

```text
#[cfg(unix)]  sys::unix::process::Command::exec
#[cfg(windows)] sys::windows::process::...
```

| Unix 典型步骤 | 说明 |
|---------------|------|
| **`fork`** | 复制进程 · 子进程 **新 pid** |
| **`exec`** | 替换镜像 · **fd 继承** 由 spawn 前配置 |
| **`pipe` + `dup2`** | [10.1](./10.1-anonymous-pipe.md) · [10.2](./10.2-redirection.md) |
| **`waitpid`** | 回收 **ExitStatus** · 防 zombie |

| Windows | 说明 |
|---------|------|
| **`CreateProcess`** | 一次创建 · 句柄而非 fd 号 |
| **Stdio 句柄继承** | 与 Unix 概念对应 · API 不同 |

### 适配层职责清单

| 职责 | 不在此层 |
|------|----------|
| 平台 syscall / libc | **`Command::arg` 链式 builder 语义** |
| errno → **`io::Error`** | 用户文档 |
| **`pre_exec` 安全闭包**（unix） | — |
| fd **close-on-exec** 策略 | — |

### unsafe 集中区

原书预期在此章展开 **unsafe 块** 如何证明：**spawn 后仅子进程可见的 fd 布局** — **待刷书补全**。
