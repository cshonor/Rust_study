## 一句话

 **`std::process::Command` / `Child` / `ExitStatus`** 是对开发者暴露的 **stable 安全 API** — 链式配置 **程序名、参数、env、cwd、stdio** 后 **`spawn`**，返回 **`Child`**；**`wait`/`output`/`status`** 封装 [10.3.1 · OS 层](./10.3.1-process-os-layer.md) 的 wait 与 I/O。

---

## 原书要点（目录脉络 · 细节待刷书）

### 核心类型

| 类型 | 作用 |
|------|------|
| **`Command`** | Builder · **`arg`/`env`/`current_dir`/`stdin/stdout/stderr`** |
| **`Child`** | 运行中子进程 · **`stdin()`/`stdout()`/`stderr()`** 管道端 |
| **`ExitStatus`** | 退出码 · 平台封装 |
| **`Output`** | **`output()`** 一次性捕获 stdout/stderr |

```rust
let mut child = Command::new("my_tool")
    .args(["--config", path])
    .stdout(Stdio::piped())
    .spawn()?;

let status = child.wait()?;
```

### 安全设计（相对 C `system()`）

| 特性 | 说明 |
|------|------|
| **无 shell 除非 `Command::new("sh")`** | 默认 **argv 数组** · 少 injection |
| **FD 所有权** | **`ChildStdout` Drop** · 不泄漏 pipe |
| **`io::Result`** | 失败可传播 |

### 与 10.1～10.2

| 配置 | 底层 |
|------|------|
| **`.stdin(Stdio::piped())`** | [10.1](./10.1-anonymous-pipe.md) + [10.2](./10.2-redirection.md) |
| **`.stdout(Stdio::from(file))`** | 文件 **fd** 作 stdout |

### HFT

**spawn 辅助进程**（日志归档、合规导出）— **`current_dir`/`env`** 隔离配置 · **勿** 在 tick 热路径频繁 spawn。
