## 一句话

**`str`/`String`** = **UTF-8 字节序列** + 标准库丰富 API（`split`、`replace`、`parse`…）；子串查找 **`find`/`match_indices`** 底层并非暴力扫描，而是 **TwoWay 算法** — 利用模式串 **周期性** 大幅跳跃，**极低额外堆内存**。

### 深度补充：TwoWay 子串查找（原书 · 性能天花板）

**`str.find(needle)`** 等路径使用 **TwoWay 算法**（Crochemore–Perrin 思想的标准库实现）。

| 要点 | 说明 |
|------|------|
| **非暴力** | 不匹配时 **非仅 +1** — 结合 **前缀/后缀** 与 **临界因子** 分析 |
| **周期性（Period）** | 若模式串 **自重复**（如 `"abab"`），算法识别 **周期** — 失配时可 **大步跳过** |
| **双向** | 从模式两端同步比较信息，压缩比较次数 |
| **空间** | **O(1) 额外堆**（预处理常数级表）— 适合嵌入式/热路径 |

```text
暴力: 失配 → i+1          O(n·m) 最坏
TwoWay: 失配 → i+period/k  近似 O(n) 级别 · 常数优秀
```

| 源码入口 | 说明 |
|----------|------|
| `library/core/src/str/pattern.rs` | Pattern API |
| `library/core/src/str/search.rs` | TwoWaySearcher |

### HFT

行情 **symbol / 合约代码** 查找 — **`&str`** 视图 + 预编译 needle；理解 TwoWay 有助于评估 **超长 pattern** 与 **短 haystack** 的相对成本。
