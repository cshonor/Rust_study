# 6.3 字符类型

## 一句话

**`char`** 是 **4 字节 Unicode 标量值**（`U+0000`～`U+10FFFF`，不含 surrogate）— 与 **`str` 的 UTF-8 字节序列** 相对；标准库提供 **大小写、数字属性、`encode_utf8`** 等，并 impl **`Step`** 供范围迭代（[5.4](../chapter05_iterators/5.4-range-iterators.md)）。

---

## 原书要点

| 对比 | `char` | `str` / `[u8]` |
|------|--------|----------------|
| 语义 | **单个 Unicode 标量** | **UTF-8 编码文本** |
| 宽度 | 固定 **4 字节** | **变长** 1～4 字节/字符 |
| 索引 | 标量值本身 | **字节偏移** |

| API | 作用 |
|-----|------|
| **`is_ascii()` / `is_numeric()`** | 分类 |
| **`to_uppercase()`** | 可能 yield **多个** `char` |
| **`encode_utf8`** | 写入 `&mut [u8; 4]` |
