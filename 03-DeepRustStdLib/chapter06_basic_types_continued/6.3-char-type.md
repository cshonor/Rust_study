## 一句话

**`char`** = **4 字节 Unicode 标量** — 原书系统梳理 **与 `u32`、UTF-8 字节、`&str` 的互转**，以及 **分类判断**、**`escape_unicode`** 等字符级 API；是理解 [6.4 · UTF-8 字符串](./6.4-string-type.md) 的前置。

---

## 原书要点

### 类型关系

| 方向 | API / 规则 |
|------|------------|
| **`char` ↔ `u32`** | **`as u32`** / **`char::from_u32`**（`Option` — 非法标量拒绝） |
| **→ UTF-8 字节** | **`encode_utf8(&mut [u8; 4])`** → `&str` 片段 |
| **← `&str`** | **`.chars().next()`** 或解析单个字符 |

```rust
let c = '中';
let mut buf = [0u8; 4];
let s: &str = c.encode_utf8(&mut buf);
```

### 分类与转义

| API | 作用 |
|-----|------|
| **`is_alphabetic` / `is_numeric` / `is_whitespace`** | Unicode 分类 |
| **`is_ascii()`** | ASCII 子集 |
| **`escape_unicode()`** | **`Iterator<Item=char>`** — `\u{4e2d}` 形式转义 |
| **`escape_debug` / `escape_default`** | 调试/默认转义 |

### 与字符串章节的边界

| `char` | `str` |
|--------|-------|
| 固定 4 字节标量 | 变长 UTF-8 **字节**串 |
| 可安全转 `u32` | 切片须 **`is_char_boundary`**（[6.4](./6.4-string-type.md)） |
