## 原书要点

### 非零整数（NonZero）— 本章核心

| 类型 | 语义 |
|------|------|
| **`NonZeroU8` … `NonZeroU128`** | 构造时 **禁止 0** · 内部保证 **`get() != 0`** |
| **`NonZeroUsize` / `NonZeroIsize`** | 指针长度、索引等常见 |

**内存优化（零成本抽象）**：

```text
Option<u32>        →  通常 8 字节（tag + padding）
Option<NonZeroU32> →  4 字节 — 用 0 表示 None，非零值表示 Some
```

| 机制 | 说明 |
|------|------|
| **Niche optimization** | `0` 在 `NonZero*` 中 **非法** → 编译器 **复用 0 作 `None`** |
| **与枚举** | 标准库大量 **`Option<NonZero*>`** 压缩智能指针/计数器布局 |

↔ [3.4 · NonNull niche](../chapter03_memory_model/3.4-nonnull.md)（指针非空同理）

```rust
let n = NonZeroU32::new(42).unwrap();
let opt: Option<NonZeroU32> = None; // 内存布局 ≡ 单个 u32
```
