## 一句话

**`[T]`** 胖指针视图 — 原书以 **`swap`** 与 **插入排序** 为例：**安全层对外**，内部在证明 **不越界、不重叠** 后大量使用 **`unsafe` + `ptr::copy_nonoverlapping`** 直拷内存，支撑与 C 级相当的吞吐。

↔ [5.5 · 双指针迭代](../chapter05_iterators/5.5-slice-iterators.md)

### 示例一：`swap` 与底层指针

**`slice.swap(i, j)`** 交换两元素 — 内部可在 **unsafe** 下 **指针读写的** 避免多余 temp move（尤其大 `T` 或优化路径）。

### 示例二：插入排序 + `copy_nonoverlapping`

向前挪动元素时，若逐步 **安全 move**：

```text
每步 → 可能 drop + 所有权转移  →  指令膨胀
```

标准库在 **unsafe 块** 内：

```rust
ptr::copy_nonoverlapping(src, dst, count);  // 等同 C memcpy
```

| 范式 | 说明 |
|------|------|
| **对外 safe** | `sort` / `sort_by` / `swap` — 用户 **`&mut [T]`** |
| **对内 unsafe** | std 作者承担 **边界/别名** 证明 |
| **哲学** | **高级安全封装 ← 高水平 unsafe 底座** |

↔ [3.1 · 裸指针](../chapter03_memory_model/3.1-raw-pointers.md) · [Nomicon](../../04-Rust-Nomicon/README.md)
