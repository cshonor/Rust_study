## 一句话

**`bool`** 除 **1 字节两值** 与 **`Option<bool>` niche** 外，标准库提供 **`then` / `then_some`** — 把条件判断 **嵌入迭代器/闭包链**，是函数式编程拼图，替代部分 **`if…else`** 分支。

---

## 原书要点

### 基础

| 项 | 说明 |
|----|------|
| **大小** | `size_of::<bool>() == 1` |
| **无 truthy** | `if n { … }` 非法 · 须显式比较 |
| **niche** | `Option<bool>` 等布局优化 |

### 函数式转换（原书重点）

| 方法 | 签名直觉 | 作用 |
|------|----------|------|
| **`then_some`** | `true → Some(v)` · `false → None` | 条件 **Optional 化** |
| **`then`** | `true → f()` · `false → None` | **惰性** 构造 · 闭包链 |

```rust
// 链式：条件成立才产生下一步
let opt = flag.then_some(42);
let mapped = flag.then(|| expensive());

// 融入 Iterator
items.into_iter()
    .filter(|x| x.valid)
    .flat_map(|x| x.is_top.then_some(x.price))
    .collect::<Vec<_>>();
```

```text
if flag { Some(x) } else { None }  ──►  flag.then_some(x)
```
