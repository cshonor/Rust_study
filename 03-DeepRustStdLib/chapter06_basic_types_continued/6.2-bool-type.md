## 一句话

**`bool`** 仅 **`true`/`false`** 两值 — 内存 **1 字节**（非 1 bit）；与 **`Option` niche 优化**、条件 **`match`** 及 **`if` 必须是 `bool`**（无 C 式 truthy）共同构成 Rust 逻辑分支底座。

---

## 原书要点

| 项 | 说明 |
|----|------|
| **大小** | `size_of::<bool>() == 1` |
| **转换** | 无隐式 int→bool · 须显式比较 |
| **`as` 转换** | `bool as u8` → 0/1 |
| ** niche** | `Option<bool>` 等可优化布局 |

```rust
// 错误：if n { … }  // E0308
if n != 0 { … }
```
