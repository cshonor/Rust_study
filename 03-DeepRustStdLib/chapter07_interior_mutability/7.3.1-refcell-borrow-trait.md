# 7.3.1 Borrow Trait 分析

> 章索引：[第 7 章](./README.md) · 前：[7.3](./7.3-refcell-overview.md) · 后：[7.3.2 · RefMut](./7.3.2-refcell-borrowmut-trait.md)

## 一句话

**`RefCell::borrow()`** 返回 **`Ref<'b, T>`** guard — 通过 **`Deref` → `&T`** 提供共享访问；guard **Drop** 时递减 **`BorrowFlag`**，实现 **运行期「不可变借用」** 语义（对应原书「Borrow 分析」）。

---

## 原书要点

### 工作流程

```rust
let r = refcell.borrow();   // flag: 0 → 1（或 n → n+1）
let x: &i32 = &*r;          // Deref
// … 多个 Ref 可共存 …
drop(r);                   // flag -1
```

| 组件 | 作用 |
|------|------|
| **`Ref<'b, T>`** | 借用 token · **lifetime `'b`** 绑定到 RefCell 借用周期 |
| **`Deref` / `DerefMut`（无）** | **`Ref` 仅 Deref 到 `&T`** |
| **`BorrowFlag`** | 多个 **`Ref`** 可同时存在 |

### 冲突 → panic

```rust
let _m = refcell.borrow_mut();
let _r = refcell.borrow();  // panic: already mutably borrowed
```

### `try_borrow`

返回 **`Result<Ref, BorrowError>`** — 热路径/库代码避免 panic。

↔ [7.1 · std::borrow::Borrow](./7.1-borrow-borrowmut.md)（Trait 名易混 · 此处指 **RefCell 借用 guard**）
