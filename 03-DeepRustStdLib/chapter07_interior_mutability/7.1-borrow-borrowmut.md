# 7.1 Borrow / BorrowMut 分析

> 章索引：[第 7 章](./README.md) · 后：[7.2 · Cell 总览](./7.2-cell-overview.md)

---

## 一句话

**`Borrow<T>` / `BorrowMut<T>`**（`core::borrow`）描述 **「等价键/等价视图」** 的借用规则 — 为 **`HashMap`/`HashSet` 查找** 与 **`RefCell` 系列 API 命名** 提供 Trait 基础；内部可变性章从这里过渡到 **`UnsafeCell` 栈**。

---

## 原书要点

| Trait | 契约 |
|-------|------|
| **`Borrow<B>`** | `&self` 可等价于 **`&B`**（如 `String` → `str`） |
| **`BorrowMut<B>`** | 可变等价视图（较少见） |

```rust
// HashMap 查找：k 不必与存储键同型
map.get("hello");  // String key · &str 查找 · Borrow
```

### 与内部可变性的关系

| 层 | 类型 |
|----|------|
| **编译期借用** | `&T` / `&mut T` |
| **Trait 等价** | `Borrow` — 集合与智能指针 |
| **运行期借用** | `RefCell::borrow` → **`Ref`** guard（[7.3](./7.3-refcell-overview.md)） |

**注意**：`RefCell` 的 **`borrow()`** 返回 **`Ref<'b, T>`**，与 **`std::borrow::Borrow`** 是 **不同概念** — 7.3.1 讨论的是 **RefCell 的运行期借用 guard**。
