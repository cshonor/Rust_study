# 7.3.2 BorrowMut Trait 分析

> 章索引：[第 7 章](./README.md) · 前：[7.3.1](./7.3.1-refcell-borrow-trait.md) · 后：[7.3.3 · 其他函数](./7.3.3-refcell-other-fns.md)

---

## 一句话

**`RefCell::borrow_mut()`** 返回 **`RefMut<'b, T>`** — **`DerefMut` → `&mut T`**，**独占** 内部；`BorrowFlag` 置 **`-1`**，与 **任何 `Ref`/`RefMut`** 互斥，违则 **panic**。

---

## 原书要点

| 规则 | 说明 |
|------|------|
| **至多一个 `RefMut`** | 等价编译期 **`&mut T`** |
| **与所有 `Ref` 互斥** | 读写不能并存 |
| **guard 存活期** | **`RefMut` 未 drop 前** 不能再 `borrow` |

```rust
let mut w = cell.borrow_mut();
*w += 1;
// drop(w) 后 flag 回 0
```

### `RefMut` 能力

| Trait | 效果 |
|-------|------|
| **`Deref`** | 也可 **`&*w`** |
| **`DerefMut`** | **`*w = …`** |
| **`Drop`** | 恢复 `BorrowFlag` |

### 典型模式

```rust
// Rc<RefCell<T>> — 单线程共享可变（多 owner + 内部 mut）
let shared = Rc::new(RefCell::new(vec![]));
shared.borrow_mut().push(1);
```

**跨线程** → **`Mutex`/`RwLock`**，**非 RefCell**。
