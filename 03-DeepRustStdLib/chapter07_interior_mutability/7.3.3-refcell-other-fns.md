# 7.3.3 RefCell<T> 的其他函数

> 章索引：[第 7 章](./README.md) · 前：[7.3.2](./7.3.2-refcell-borrowmut-trait.md) · 后：[7.4 · Pin](./7.4-pin-unpin.md)

---

## 一句话

除 **`borrow`/`borrow_mut`** 外，`RefCell` 还提供 **`try_borrow*`**（无 panic）、**`replace`/`swap`**（需独占时可 **不持 RefMut 长生命周期** 地交换值）等 — 减少 guard 嵌套与 panic 风险。

---

## 原书要点

| API | 作用 |
|-----|------|
| **`try_borrow()` / `try_borrow_mut()`** | `Result` · 冲突时 **Err** |
| **`replace(val)`** | **立即** 取旧值、写新值（内部短暂 mut · 类似 `mem::replace`） |
| **`swap(&other)`** | 与另一 `RefCell` 交换内容 |
| **`get_mut()`** | **唯一 `&mut T`** — 仅当 **无活跃 Ref/RefMut**（编译期+运行期） |
| **`as_ptr()`** | 底层 `*mut T`（unsafe 场景） |

### 使用建议

| 场景 | 推荐 |
|------|------|
| 回调可能重入 | **`try_borrow`** |
| 快速换整值 | **`replace`** 优于长期 **`borrow_mut`** |
| 确定无其他借用 | **`get_mut`** 零 guard 开销 |
