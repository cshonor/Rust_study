# 7.3 RefCell<T> 类型分析

> 章索引：[第 7 章](./README.md) · 前：[7.2.2](./7.2.2-cell.md) · 后：[7.3.1 · borrow guard](./7.3.1-refcell-borrow-trait.md)

---

## 一句话

**`RefCell<T>`** = **`UnsafeCell<T>` + `BorrowFlag` 运行期借用计数器** — 在 **`&RefCell<T>`** 上 **`borrow()`/`borrow_mut()`** 动态执行 **读写互斥**；违规则 **`panic!`**（非 UB），把部分 **编译期** 借用检查 **推迟到运行期**。

---

## 原书要点

### 不是编译器补丁

| 误解 | 原书立场 |
|------|----------|
| RefCell =  hack | **有意架构** — 树/图/观察者 **共享 &self 仍要改子节点** |
| 与 `&mut` 等价 | **运行期** 才检查 · 有 **panic 成本** |

### BorrowFlag（直觉）

| 状态 | 含义 |
|------|------|
| **`0`** | 未借用 |
| **`>0`** | **N 个** 共享 `borrow`（`Ref`） |
| **`-1`** | **唯一** `borrow_mut`（`RefMut`） |

```text
borrow()     → flag +1   → Ref  guard drop 时 -1
borrow_mut() → flag = -1  → 与任何 borrow 互斥
冲突         → panic!    （try_borrow 返回 Err）
```

### 与 `Cell`

需要 **`&T` 指向内部字段**（大结构、非 Copy）→ **`RefCell`**；仅改 **`Copy` 标量** → **`Cell`** 更轻。
