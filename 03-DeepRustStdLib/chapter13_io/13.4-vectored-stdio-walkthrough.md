## 一句话

对照 **`library/std`** 的 **readv/writev 与 stdio 全局锁** 概念骨架 — 非 rustc 逐字拷贝。

---

## A. writev（Unix）

```rust
// sys 层 — 概念
pub fn write_vectored(fd: RawFd, bufs: &[IoSlice]) -> io::Result<usize> {
    let mut iovec: Vec<libc::iovec> = bufs.iter().map(|s| libc::iovec {
        iov_base: s.as_ptr() as *mut _,
        iov_len: s.len(),
    }).collect();

    let ret = unsafe {
        cvt(libc::writev(fd, iovec.as_mut_ptr(), iovec.len() as _))?
    };
    Ok(ret as usize)
}
```

**`TcpStream` / `File`** 在 **`Write::write_vectored`** 中 override · 否则默认 **循环 write**。

---

## B. readv（Unix）

```rust
// 对称：readv 填充多个 IoSliceMut
unsafe {
    cvt(libc::readv(fd, iovec.as_mut_ptr(), count))?
}
```

## C. stdio 全局锁（概念）

```rust
// io/stdio.rs — 逻辑骨架
pub fn stdin() -> &'static Stdin {
    STDIN.get_or_init(|| Stdin { inner: sys_stdio::stdin() })
}

impl Stdin {
    pub fn lock(&self) -> StdinLock<'_> {
        StdinLock {
            inner: self.inner.lock(), // 全局 Mutex/ReentrantMutex
        }
    }
}

impl Read for StdinLock<'_> {
    fn read(&mut self, buf: &mut [u8]) -> io::Result<usize> {
        self.inner.read(buf) // 底层 fd 0 read
    }
}
```

| 点 | 说明 |
|----|------|
| **`lock()`** | 持锁期间 **独占** stdin |
| **`println!`** | **`stdout().lock()`** 同理 |

---

## D. 缓冲与 flush

```text
Stdout（TTY）:     行缓冲 → '\n' 或 flush() → write(fd 1)
Stdout（重定向）:  块缓冲 → 满块或 flush
Stderr:            无缓冲 → 每次 write 尽快到达 fd 2
```

实现分散在 **`sys::stdio`** · **`LineWriter`** 等 wrapper。
