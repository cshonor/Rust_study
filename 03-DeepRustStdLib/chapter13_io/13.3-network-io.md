## 一句话

**`std::net`** — **Socket 即 fd** · **`TcpStream: Read + Write`** · **Drop close** — 与 File/stdio **同一 IO 模型**；**Unix Berkeley Socket** · **Windows Winsock** 差异下沉 **sys::net**；**标准库仅同步阻塞 IO**。

---

## 原书要点

### 跨平台底层

| Unix | Windows |
|------|---------|
| **socket/bind/listen/accept/connect/send/recv** | **Winsock2** · **`WSAStartup`** |
| **fd 型 SOCKET** | **SOCKET 句柄** · 适配为 **OwnedFd/Handle** |

### 核心类型

| 类型 | 角色 |
|------|------|
| **`TcpListener`** | **`bind` + `accept()` → `TcpStream`** |
| **`TcpStream`** | 已连接 TCP · **`Read + Write`** · 与 File **通用读写** |
| **`UdpSocket`** | **`send_to` / `recv_from`** · 数据报 |
| **`IpAddr` / `SocketAddr`** | IPv4/IPv6 · 抹平 **`sockaddr`** 差异 |

```rust
let mut stream = TcpStream::connect("127.0.0.1:8080")?;
stream.write_all(request)?;
stream.read(&mut buf)?;
```

### 资源安全

| 机制 | 说明 |
|------|------|
| **Drop** | 自动 **close** fd/SOCKET · 防泄漏 |
| **`try_clone`** | dup 句柄 · 配合 **`Arc`** 多线程共享连接 |

### 阻塞 vs 异步

| std::net | Ch14 / tokio |
|----------|--------------|
| **同步阻塞** · 每 op 可能挂起线程 | **Future + epoll/io_uring/IOCP** · 非阻塞 |

```text
socket fd  ──►  OwnedFd  ──►  Drop → close
           ──►  impl Read/Write  ──►  与 File 相同 trait 世界
```
