## 一句话

**`std::net`** — **`TcpStream` / `TcpListener` / `UdpSocket`** 等 **Socket 即 fd** · 同样 **`Read + Write`**（TCP）· **Drop 关闭** — 本地文件与远程数据 **统一 I/O 模型**；阻塞 syscall · async 见 [第 14 章](../chapter14_async/README.md)。

---

## 原书要点（脉络 · 待刷书）

### 核心类型

| 类型 | 角色 |
|------|------|
| **`TcpListener`** | **`accept` → TcpStream** |
| **`TcpStream`** | 已连接 TCP · **`Read` + `Write`** |
| **`UdpSocket`** | **`send_to` / `recv_from`**（ datagram ） |

```rust
let mut stream = TcpStream::connect("127.0.0.1:8080")?;
stream.write_all(request)?;
stream.read(&mut buf)?;
```

### 与 Ch12/Ch9 一体

```text
socket fd  ──►  OwnedFd  ──►  Drop → close
           ──►  impl Read/Write  ──►  与 File 相同 trait 世界
```

底层：**`connect`/`send`/`recv` SYSCALL** · [9.4](../chapter09_userspace_std_basics/9.4-syscall-wrapper.md)

|----|------|
