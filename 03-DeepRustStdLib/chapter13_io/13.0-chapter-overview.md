## 三层架构

| 层 | 模块 | 特征 |
|----|------|------|
| **业务 API** | `io::stdin/stdout/stderr` · `std::net` | 全局标准流 · TCP/UDP |
| **通用抽象** | `std::io` · **Read/Write/Seek** · **ReadVectored/WriteVectored** | 文件/管道/终端/Socket **多态** |
| **sys 适配** | `sys::io` · `sys::net` · `sys::stdio` | **readv/writev** · Console · **Berkeley Socket / Winsock** · **`unsafe`** |

```text
copy_stream<R: Read, W: Write>  // 泛型一套逻辑
  ├── File（Ch12）
  ├── StdinLock / StdoutLock（fd 0/1）
  ├── 管道（Ch10）
  └── TcpStream（Socket fd）
        └── sys 层不同 syscall · 上层 trait 相同
```

## 统一 IO 示例

```rust
use std::io::{self, Read, Write};
use std::net::TcpStream;

fn copy_stream<R: Read, W: Write>(mut r: R, mut w: W) -> io::Result<()> {
    let mut buf = [0u8; 1024];
    loop {
        let n = r.read(&mut buf)?;
        if n == 0 { break; }
        w.write_all(&buf[..n])?;
    }
    w.flush()?;
    Ok(())
}

fn main() -> io::Result<()> {
    copy_stream(io::stdin().lock(), io::stdout().lock())?;
    let mut tcp = TcpStream::connect("example.com:80")?;
    tcp.write_all(b"GET / HTTP/1.1\r\nHost: example.com\r\n\r\n")?;
    copy_stream(tcp, io::stdout())?;
    Ok(())
}
```

## 核心设计亮点

| 亮点 | 说明 |
|------|------|
| **万物皆流** | 同一 Read/Write · 低学习成本 |
| **向量 I/O** | readv/writev · 少 syscall/拷贝 |
| **stdio 全局锁** | 多线程终端不交错 |
| **Socket=fd** | Drop close · 复用 RAII |
| **unsafe 下沉 sys** | 上层跨平台 safe |
