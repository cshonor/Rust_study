# 2.3.4 获取引用

> 章索引：[第 2 章](./README.md) · 前：[2.3.3-closures-for-inner-access](./2.3.3-closures-for-inner-access.md) · 后：[2.3.5-getting-ownership](./2.3.5-getting-ownership.md)

## 一句话

智能指针与封装类型提供 **`as_ref` / `as_mut` / `as_ptr` / `as_mut_ptr`** — 在 **不转移所有权** 的前提下取得内部 **`&T` / `&mut T` 或裸指针**（后者供 unsafe 边界使用）。

---

## 原书要点

### 标准函数族

| 方法 | 得到 | 消费外层？ |
|------|------|:----------:|
| **`as_ref()`** | `Option<&T>` → `&T` 视图 | ❌ |
| **`as_mut()`** | `Option<&mut T>` | ❌ |
| **`as_ptr()`** | `*const T` | ❌ |
| **`as_mut_ptr()`** | `*mut T` | ❌ |

```rust
let mut opt = Some(String::from("x"));
if let Some(s) = opt.as_mut() {
    s.push('y');
}
let p: *const String = opt.as_ref().map(|s| s as *const _).unwrap_or(std::ptr::null());
```

### 典型类型

| 类型 | 常用 API |
|------|----------|
| **`Option<T>`** | `as_ref` / `as_mut` |
| **`Box<T>`** | `as_ref` · **`as_mut`** · **`as_ptr`** |
| **`Vec<T>`** | **`as_ptr` / `as_mut_ptr`** · `as_slice` |
| **`String`** | `as_ptr` · `as_bytes` |

### 与 Deref 区别

| | `Deref` 自动 | `as_ref` 等 |
|---|-------------|-------------|
| 触发 | 方法解析时隐式 | **显式**调用 |
| `Option` | 不直接 Deref 到 `T` | **`as_ref`** 得 `Option<&T>` |

裸指针 → 第 3 章 [裸指针](../chapter03_memory_model/README.md) · **unsafe 解引用**。
