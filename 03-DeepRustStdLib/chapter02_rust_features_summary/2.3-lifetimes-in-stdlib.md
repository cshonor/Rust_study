# 2.3 生命周期在标准库中的落地

> 章索引：[第 2 章](./README.md) · 前：[2.2 Trait](./2.2-traits-in-stdlib.md) · 后：[2.4 闭包与 Iterator](./2.4-closures-iterator-in-stdlib.md)

---

## 一句话

读 `libstd` 时见到的 **`'a`、`'static`、`'env`** 不是装饰 — 它们标的是 **引用能活多久**，直接决定 `str` 切片、`Cow`、迭代器适配器能否合法编译。

---

## 三类高频生命周期

| 标注 | 在 std 里常见于 | 含义 |
|------|-----------------|------|
| **`'a`（泛型）** | `&'a str`、`Iter<'a, T>` | 引用不能超过被借数据 |
| **`'static`** | `&'static str`、线程闭包 `F: 'static` | 整个程序期有效，或 **owned 无借用** |
| **`'env`（闭包）** | `thread::scope` 闭包 | 绑在作用域，可借栈上局部 |

## 标准库中的落地例子

### 1. 字符串与切片

```rust
// &str 是 &[u8] 上的视图，生命周期 ≤ 底层 buffer
fn takes_str(s: &str) { ... }
```

`String::as_str()` → `&str` 借自 `String`；`String` drop 后引用失效 — 编译器用生命周期防止悬垂。

### 2. 集合迭代器 `Iter<'a, T>`

`vec.iter()` 产生 **`Iter<'a, T>`** — 迭代器持有 **`&'a Vec<T>`**（或内部指针），**不能在 `Vec` 可变借用期间** 同时 `iter_mut` 冲突（借用检查器）。

### 3. `Cow<'a, B>`

**Clone on write**：要么借 `'a` 的数据，要么拥有 `B` — 标准库在 **避免无谓分配** 时常用。

### 4. 线程与 `spawn` 的 `'static`

闭包若捕获 **`&局部`** → 不满足 `F: 'static`；须 **`move` owned** 或 **`scope`** — 这是 **生命周期规则在并发 API 上的投影**。

## 读源码提示

- 看到 **`struct Foo<'a>`** → 问：`'a` 从哪个参数 / 字段来？outlive 谁？
- 看到 **`fn foo() -> &str`** 无 `'a` → 实际是 **`'static`** 或 **编译错误**（省略 elision 规则）。
- **HRTB**（`for<'a>`）在 `Fn` trait、闭包 trait 对象里出现 — 进阶见 [RFR 08-2 HRTB](../../02-RFR/Chapter-02-Types/08-2-hrtb.md)。
