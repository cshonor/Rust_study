## 一句话

Rust 是 **以泛型为基础的语言** — 离开泛型几乎无法写程序；直接对泛型定义函数与 trait，换来更好抽象、可扩展性与函数式风格。

---

## 原书要点

### 泛型为本

| 点 | 说明 |
|----|------|
| **地位** | 泛型不是「高级特性」，而是 **默认写法** |
| **收益** | 抽象清晰 · 逻辑复用 · 易扩展 · 配合 **Iterator / 闭包** 的函数式编程 |
| **实现** | 单态化（monomorphization）— 编译期展开，零运行时多态开销（与 `dyn` 对比见 RFR） |
