## 一句话

**整个 async 体系的根** — **`poll(self: Pin<&mut Self>, cx: &mut Context) -> Poll<Output>`** · **惰性 · 非阻塞 Pending · Waker 唤醒再 poll**；**Pin 联动 Ch7** · **阻塞 IO 联动 Ch13**。

## 原书要点

### 标准库定义（core）

```rust
pub trait Future {
    type Output;
    fn poll(self: Pin<&mut Self>, cx: &mut Context<'_>) -> Poll<Self::Output>;
}

pub enum Poll<T> {
    Ready(T),
    Pending,
}
```

### 两个关键约束

| 约束 | 联动 | 含义 |
|------|------|------|
| **`Pin<&mut Self>`** | [Ch7 · Pin](../chapter07_interior_mutability/7.4-pin-unpin.md) | **自引用状态机** 不可 move |
| **仅 `poll` 无默认实现** | 编译器/手写 IO | 所有 async 任务 **必须实现 poll** |

### Pin 闭环（全书最重要串联之一）

```text
async 状态机 enum 内字段 A 的引用指向同 struct 内字段 B
  → move 整个 enum → 内部指针悬垂
  → 编译器生成 !Unpin
  → poll 签名强制 Pin
  → Box::pin / pin_mut! 固定地址
```

↔ [Ch8 · Box::pin](../chapter08_smart_pointers/) · [14.4 · 走读](./14.4-async-desugar-pin-walkthrough.md)

### 与阻塞 I/O（Ch13）

```text
阻塞: thread 在 read() 内核 sleep（占线程）
async:  poll → Pending → epoll 等 fd → Waker::wake → 再 poll（不占线程空等）
```

### HFT

**少 poll、少 wake** — 批处理就绪事件 · 避免 executor 抖动；超低延时可能 **不用 async**。
