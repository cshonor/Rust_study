# 14.1.1 协程概述

> 章索引：[第 14 章](./README.md) · 前：[14.1](./14.1-coroutine-framework.md) · 后：[14.1.2 · 多路复用](./14.1.2-io-multiplexing.md)

## 一句话

**Rust 无内置协程调度器** — 仅 **`async fn` / `async {}` 语法** · 编译器 **desugar 为 `Future` 状态机** · **惰性**（不 poll 不执行）· **无运行时绑定**。

---

## 原书要点

### 1. async/await 编译变换

```rust
async fn demo() -> u32 {
    let a = read_io().await;
    a + 1
}
// 编译器 ≈ 匿名 enum DemoFuture { State0, State1, … }
// 每个 .await → 一个变体 · 保存跨 await 的局部变量/引用
```

| 点 | 说明 |
|----|------|
| **协程 ≈ Future** | 语言 coroutine **落地为 trait Future** |
| **`.await`** | poll 子 Future · 未完成 → **yield Pending** |
| **惰性** | 调用 `demo()` **不跑 IO** · 须 executor **poll** |
| **`Send`** | 跨线程 spawn 须状态机 **`Send`** |

→ 脱糖示例：[14.4 · 状态机走读](./14.4-async-desugar-pin-walkthrough.md)

### 2. 协程 vs OS 线程

| | **线程（Ch11）** | **async 协程（Ch14）** |
|---|------------------|------------------------|
| 调度 | **OS 内核** · 切换成本高 | **用户态 executor** · poll |
| 栈 | 固定 MB 级 | 状态机 **按需** · 可大量并发 |
| 阻塞 IO | 占满线程 | **Pending** · 线程服务其他 Future |
| 典型规模 | 数百～数千线程 | **成千上万** Future · 少量工作线程 |

### 3. Pin 闭环（Ch7 · 全书关键串联）

| 问题 | 对策 |
|------|------|
| 状态机 **自引用**（await 间字段引用自身） | move → **悬垂指针** |
| **`Pin<&mut Self>`** on `poll` | 禁止 **非 Unpin** 被 move |
| 编译器生成 **`!Unpin`** 状态机 | **`Box::pin`** / **`pin_mut!`** · [Ch8 Box](../chapter08_smart_pointers/) |

↔ [7.4 · Pin/Unpin](../chapter07_interior_mutability/7.4-pin-unpin.md)
