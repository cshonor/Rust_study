## 原书要点（脉络）

```text
Executor                    Future (async 状态机)
  loop {                      poll(Pin<&mut Self>, &mut Context)
    ready = task.poll()           ├─ Ready(T)  → 完成
    if Pending → park             └─ Pending   → 注册 Waker
  }                           I/O 就绪 → reactor 调用 waker.wake()
```

| 类型 | 角色 |
|------|------|
| **`Poll<T>`** | **`Ready(t)` / `Pending`** |
| **`Context`** | 携带 **`Waker`** 传入 `poll` |
| **`Waker`** | I/O 就绪时 **唤醒** executor 再 poll |

↔ 原书前言：**揭示 Rust 异步最基础的思考与实现**
