# 12.1.3 目录操作分析

> 章索引：[第 12 章](./README.md) · 前：[12.1.2](./12.1.2-file-ops.md) · 后：[12.2 · 对外 API](./12.2-fs-public-api.md)

---

## 一句话

**目录 OS 层** — **`mkdir`/`remove_dir`/`read_dir`** 等 · Unix **`readdir`/`getdents`** · Windows **`FindFirstFile`** 系列 — 遍历结果仍为 **平台路径字节** → **`DirEntry` → `Path`**。

---

## 原书要点（脉络 · 待刷书）

| 操作 | 对外 API（12.2） | OS 层 |
|------|------------------|-------|
| 创建 | **`create_dir` / `create_dir_all`** | `mkdir` |
| 删除 | **`remove_dir` / `remove_file`** | `unlink` / `rmdir` |
| 遍历 | **`read_dir`** | `readdir` 迭代 |
| 元数据 | **`metadata` / `symlink_metadata`** | `stat` / `lstat` |

```rust
for entry in fs::read_dir(path)? {
    let entry = entry?;
    let p: PathBuf = entry.path();
}
```

### 权限与差异

| 点 | 说明 |
|----|------|
| **权限位** | Unix **`mode`** · Windows **ACL** — `sys` 层映射 |
| **符号链接** | **`symlink_metadata`** vs **`metadata`**（是否跟随） |
