# 12.0 本章总览 · 分层架构与跨平台对照

## 一句话

**第 12 章** = **`std::path` + `std::fs`** — 延续 Ch9～11 **双层（+ 平台扩展）架构**，在 **OsStr 编码隔离** 之上抹平 **Unix 字节路径 vs Windows UTF-16** 差异；**`File` RAII Drop** 与 [Ch10 管道 fd](../chapter10_process_management/10.1-anonymous-pipe.md) 同源。

---

## 三层结构（同进程/并发模块）

| 层 | 模块 | 特征 |
|----|------|------|
| **上层 safe API** | `std::path` · `std::fs` | **`Path`/`File`/`read_dir`** · RAII · **`io::Result`** |
| **OS 底层适配** | `sys::{unix,windows}::fs` · `sys::path` | **`unsafe`** · fd/HANDLE · open/stat/readdir |
| **平台扩展** | `std::os::unix::fs` · `std::os::windows::fs` | **扩展 Trait** · Unix mode · Windows 共享/属性 |

```text
业务  File::open(path) / fs::read_dir()
  └── std::fs / path（stable）
        └── sys::fs / sys::path（SYSCALL）
        └── os::unix::fs（按需 OpenOptionsExt 等）
```

---

## 跨平台对照

| 特性 | Unix | Windows | std 抹平 |
|------|------|---------|----------|
| 分隔符 | `/` | `\` | **`Path::join`/`push`** |
| 路径编码 | 原生 **字节**（常 UTF-8） | **UTF-16 WCHAR** | **`OsString`/`OsStr`** |
| 权限 | rwx **mode** | ACL / 只读隐藏 | **`Metadata`** + unix 扩展 |
| 软链接 | `symlink` | 文件/目录链区分 | **`fs::symlink`** 适配 |
| 句柄 | **fd** | **HANDLE** | **`File.inner`** |
| 目录遍历 | `opendir/readdir` | `FindFirstFileW` | **`ReadDir`/`DirEntry`** |

---

## 前置 / 后置

| 方向 | 关联 |
|------|------|
| **Ch9** | Read/Write/Seek · FD 所有权 · OsString |
| **Ch10** | 管道 = 特殊 fd · Drop 关闭 |
| **Ch11** | 多线程文件竞态 · **TOCTOU** |
| **Ch13** | **`File: Read + Write`** · stdin/网络 I/O 同 IO 体系 |

## 实战示例

```rust
use std::fs::{self, File, OpenOptions};
use std::path::PathBuf;
use std::io::{Read, Write};

fn main() -> std::io::Result<()> {
    let mut p = PathBuf::from("./data");
    p.push("test.txt");

    let mut f = OpenOptions::new()
        .create_new(true)  // 原子创建，规避 TOCTOU
        .write(true)
        .open(&p)?;
    f.write_all(b"rust fs test")?;

    let mut buf = Vec::new();
    File::open(&p)?.read_to_end(&mut buf)?;

    for entry in fs::read_dir("./")? {
        println!("{:?}", entry?.path());
    }
    println!("size: {}", fs::metadata(&p)?.len());
    Ok(())
}
```

## 核心设计亮点

| 亮点 | 说明 |
|------|------|
| **OsStr 隔离编码** | 禁止无声 UTF-8 假设 |
| **分层下沉差异** | 业务零 `cfg` |
| **RAII** | `File`/`ReadDir` Drop 释 fd/句柄 |
| **unsafe 收敛 sys** | 上层全 safe |
| **os 扩展 Trait** | 通用 API + 按需平台能力 |
| **TOCTOU 文档** | **`create_new(true)`** 原子创建 |
