## 一句话

**`sys::fs::File`** — Unix **`libc::open/read/write/close` + fd** · Windows **`CreateFileW/ReadFile/WriteFile + HANDLE`**；全部 **`unsafe`** · **`Drop` close** · 供上层 **`std::fs::File`** 安全包装。

---

## 原书要点

### Unix 底层（概念骨架）

```rust
pub struct File {
    fd: RawFd,
}
impl Drop for File {
    fn drop(&mut self) {
        unsafe { libc::close(self.fd); }
    }
}
```

| 操作 | SYSCALL |
|------|---------|
| **打开** | `open` / `openat` · path 字节 + flags + **mode** |
| **读/写** | `read` / `write` |
| **定位** | `lseek` |
| **锁** | `flock`（平台扩展） |

### Windows 底层

```rust
pub struct File {
    handle: HANDLE,
}
impl Drop for File {
    fn drop(&mut self) {
        unsafe { CloseHandle(self.handle); }
    }
}
```

| 操作 | API |
|------|-----|
| **打开** | **`CreateFileW`** · UTF-16 路径 · 访问/共享/属性 |
| **读/写** | `ReadFile` / `WriteFile` |

### OpenOptions 跨平台适配

上层 **`read()/write()/create()/truncate()`** → **`cfg` 分发**：

| 平台 | 内部存储 |
|------|----------|
| **Unix** | **`mode_t`** · libc **`O_*` flags** |
| **Windows** | **`dwDesiredAccess`** · **`dwShareMode`** · 属性标志 |

**`OpenOptionsExt`**（`std::os::unix::fs` / `windows::fs`）注入平台独有参数。
