## 一句话

**OS 层文件操作** — **`open`/`read`/`write`/`fsync`** 等 **SYSCALL**（或 libc 封装）· 路径经 [12.1.1](./12.1.1-path-types.md) 编码 · 返回 **原始 fd/Handle** 供上层 **`File`** 包装。

---

## 原书要点（脉络 · 待刷书）

```text
sys::fs::File::open(path, opts)
  └── openat / CreateFileW
  └── OwnedFd / Handle
```

| 操作 | 典型 SYSCALL |
|------|--------------|
| **打开** | `open` · `openat` · `CreateFile` |
| **读/写** | `read` / `write` · `ReadFile` / `WriteFile` |
| **元数据** | `stat` / `GetFileInformationByHandle` |
| **截断/同步** | `ftruncate` · `fsync` |
