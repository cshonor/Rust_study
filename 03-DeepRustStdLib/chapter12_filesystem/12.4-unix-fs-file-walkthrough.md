## 一句话

对照 **`library/std/src/sys/fs/unix.rs`** 的 **概念骨架** — **open → OwnedFd → read/write → Drop close**；非 rustc 逐字拷贝，便于与 [12.1.2](./12.1.2-file-ops.md) 并排刷源码。

---

## A. open 路径

```rust
// sys/unix/fs.rs — 逻辑骨架
pub fn open(path: &Path, opts: &OpenOptions) -> io::Result<File> {
    let path = cstr_path(path)?; // Path → OsStr → CString/CStr 字节
    let flags = opts.get_flags(); // O_RDONLY | O_CREAT | ...
    let mode = opts.get_mode();   // 仅 create 时有效

    let fd = unsafe {
        // SAFETY: path 指针有效 NUL 结尾；flags/mode 合法
        cvt(libc::open(path.as_ptr(), flags, mode))?
    };
    Ok(File { fd: fd as RawFd })
}
```

| 步骤 | 说明 |
|------|------|
| **Path → C 字符串** | Unix syscall 要 **`const char*`** |
| **`cvt`** | errno → **`io::Error`** |
| **返回 fd** | 上层 **`std::fs::File`** 再包一层 |

---

## B. read / write

```rust
impl File {
    pub fn read(&self, buf: &mut [u8]) -> io::Result<usize> {
        let ret = unsafe {
            cvt(libc::read(self.fd, buf.as_mut_ptr() as *mut _, buf.len()))?
        };
        Ok(ret as usize)
    }

    pub fn write(&self, buf: &[u8]) -> io::Result<usize> {
        let ret = unsafe {
            cvt(libc::write(self.fd, buf.as_ptr() as *const _, buf.len()))?
        };
        Ok(ret as usize)
    }
}
```

**`Read`/`Write` trait** 在 **`std::fs::File`** 层调用 **`inner`** 并处理 **WouldBlock** 等（依平台）。

## C. Drop 关闭 fd

```rust
impl Drop for File {
    fn drop(&mut self) {
        let _ = unsafe { libc::close(self.fd) };
        // 与 pipe/Child stdout 相同：fd 生命周期由 Rust Drop 管理
    }
}
```

## D. 上层 `std::fs::File` 包装

```rust
// library/std/src/fs/file.rs — 示意
pub struct File {
    inner: sys::fs::File,
}

impl Drop for File {
    fn drop(&mut self) {
        // 委托 inner Drop → close
    }
}

impl Read for File {
    fn read(&mut self, buf: &mut [u8]) -> io::Result<usize> {
        self.inner.read(buf) // 或经 Read 适配层
    }
}
```

开发者 **仅见 safe API** · **`unsafe` 仅在 sys**。

// 概念：上层 builder
