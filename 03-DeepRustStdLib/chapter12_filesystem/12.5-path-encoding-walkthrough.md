# 12.5 Path / PathBuf 编码转换极简走读

> 章索引：[第 12 章](./README.md) · 前：[12.1.1 · 路径类型](./12.1.1-path-types.md)

---

## 一句话

**Path 不存 UTF-8 假设** — 所有 fs syscall 前 **`Path → OsStr → 平台原生序列`**；Unix **`&[u8]`** · Windows **`&[u16]`** → 宽字符 API。

---

## 类型栈（拥有 → 借用）

```text
PathBuf  ──Deref──►  Path
   │                    │
   └─ inner: OsString ──┴─ inner: OsStr
            │                      │
         Unix: Vec<u8>         &[u8]
         Win:  Vec<u16>        &[u16]
```

```rust
let mut pb = PathBuf::from("/tmp");
pb.push("日志");           // join 自动分隔符
let p: &Path = pb.as_path();
```

---

## A. join / push（分隔符抹平）

```rust
// path.rs — 概念
impl PathBuf {
    pub fn push<P: AsRef<Path>>(&mut self, path: P) {
        self.inner.push(path.as_ref().inner); // OsString::push
        // sys::path 或 path 模块：Unix 补 '/' · Windows 补 '\\'（规则更细）
    }
}
```

**业务禁止** `"dir" + "\\" + "file"` · 统一 **`Path::new("dir").join("file")`**。

---

## B. Path → syscall 字节（Unix）

```rust
fn path_to_cstring(path: &Path) -> io::Result<CString> {
    let os = path.as_os_str();
    // OsStr → 字节切片（可能非 UTF-8）
    let bytes = os.as_encoded_bytes(); // 示意 API 名因版本而异
    CString::new(bytes).map_err(|_| io::Error::new(InvalidInput, "path contains NUL"))
}

// open 时：
libc::open(c_path.as_ptr(), flags, mode);
```

**`OsStr::to_str()`** 仅当 **合法 UTF-8** 才 **`Some`** — 用于 **打印/日志**，不用于 syscall。

---

## C. Path → UTF-16（Windows）

```rust
fn path_to_wide(path: &Path) -> io::Result<Vec<u16>> {
    let os = path.as_os_str();
    let wide: Vec<u16> = os.encode_wide().chain(Some(0)).collect(); // NUL 结尾
    Ok(wide)
}

// CreateFileW(wide.as_ptr(), ...);
```

Windows **`OsString`** 内部即 **`Vec<u16>`** · 与 Unix **`Vec<u8>`** 在 **`ffi/os_str`** 用 **`cfg` 分派**。

---

## D. 与 String 互转（显式、可能失败）

```rust
// String → Path（UTF-8 路径）
let p = Path::new("/home/user/文件.txt");

// Path → String（仅 UTF-8 路径）
if let Some(s) = p.to_str() {
    println!("{}", s);
} else {
    // 非 UTF-8 文件名：用 OsStr 或 lossy 显示
    println!("{:?}", p);
}
```

| 转换 | 安全性 |
|------|--------|
| **`&str` → `&Path`** | UTF-8 **`str` 必是合法 Path 字节**（Unix） |
| **`Path` → `&str`** | **`Option`** · 非 UTF-8 → **`None`** |
| **`OsString` from 字节** | 保留 **任意** 系统文件名 |

---

## E. `AsRef<Path>` 作为 fs API 边界

```rust
pub fn metadata<P: AsRef<Path>>(path: P) -> io::Result<Metadata> {
    imp::metadata(path.as_ref())
}
// 故 &str、String、PathBuf、&Path 均可传入
```
