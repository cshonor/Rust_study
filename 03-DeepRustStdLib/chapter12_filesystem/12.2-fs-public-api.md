## 一句话

**`std::fs`** 与 **`std::path`** — 开发者面对的 **统一 safe API**；**`File`** 包装 **OwnedFd** · **`Drop` 自动 close** — 将 OS 文件资源纳入 Rust **所有权模型**（原书 12.2 核心结论）。

---

## 原书要点（脉络 · 待刷书）

### 常用 API

| API | 作用 |
|-----|------|
| **`File::open` / `create` / `options()`** | 打开/创建 |
| **`read` / `write` / `copy`** | 便捷函数 |
| **`read_dir` / `create_dir_all`** | 目录 |
| **`metadata` / `set_permissions`** | 元数据 |

```rust
let mut f = File::create(path)?;
f.write_all(b"log line\n")?;
// drop(f) → close(fd)
```

### 与第 9 章 FD 设计

```text
C:   fopen → 必须 fclose
Rust: File  ──►  Drop 唯一 close  ──►  无句柄泄漏/双重 close
```
