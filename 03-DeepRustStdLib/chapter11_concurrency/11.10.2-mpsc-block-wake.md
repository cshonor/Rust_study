# 11.10.2 阻塞及唤醒信号机制

> 章索引：[第 11 章](./README.md) · 前：[11.10.1](./11.10.1-mpsc-queue.md) · 后：[11.10.3 · Oneshot](./11.10.3-mpsc-oneshot.md)

## 一句话

**`recv()` 空队列** → Receiver **阻塞**（**park / Condvar / futex**）；**`send()` 入队** → **唤醒** 等待中的 Receiver — MPSC **调度与背压** 核心。

---

## 原书要点

```text
Receiver 空:
  park / condvar.wait / futex_wait  ──►  不占 CPU

Sender 投递:
  push 消息  ──►  unpark / notify / futex_wake Receiver
```

| API | 阻塞行为 |
|-----|----------|
| **`recv()`** | 阻塞至有消息或通道关闭 |
| **`try_recv()`** | 不阻塞 |
| **`recv_timeout`** | 限时 |
