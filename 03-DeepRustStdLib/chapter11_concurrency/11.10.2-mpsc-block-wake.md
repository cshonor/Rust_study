## 一句话

**`recv()` 空队列** → 接收线程 **阻塞**（**Condvar / park / futex**）；**`send()` 入队** → **唤醒** 等待中的 Receiver — MPSC **背压与调度** 的核心（本节为你当前打开的文件）。

---

## 原书要点（脉络 · 硬核细节待刷书）

```text
Receiver 空:
  park / condvar.wait  ──►  不占 CPU

Sender 投递:
  push 消息  ──►  signal / unpark / futex_wake Receiver
```

| API | 阻塞行为 |
|-----|----------|
| **`recv()`** | 阻塞至有消息 |
| **`try_recv()`** | 不阻塞 |
| **`recv_timeout`** | 限时 |

↔ [11.3 · Condvar](./11.3-condvar-overview.md) · [11.1 · Futex](./11.1-futex.md)

### HFT

阻塞 recv **增加 jitter** · 热路径用 **`try_recv` + spin** 或 **无锁 ring**。
