## 一句话

**`Condvar::wait` / `wait_timeout` / `notify_*`** — 与 **`MutexGuard`** 组合实现 **生产者-消费者**、**线程池任务就绪** 等模式。

---

## 原书要点

```rust
while !ready {
    guard = cvar.wait(guard).unwrap();
}
```

| API | 作用 |
|-----|------|
| **`wait_timeout`** | 限时等待 |
| **`notify_all`** | 广播（多消费者） |
