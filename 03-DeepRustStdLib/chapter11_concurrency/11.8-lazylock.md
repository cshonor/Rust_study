# 11.8 LazyLock<T> 类型分析

## 一句话

**`LazyLock<T>`** = **`OnceLock<T>` + 闭包** — **首次 `Deref`** 触发 **`F: FnOnce() -> T`** · 并发安全全局 lazy static · 替代第三方 **`lazy_static`**。

---

## 原书要点

```rust
static RE: LazyLock<Regex> = LazyLock::new(|| Regex::new(r"...").unwrap());
fn parse(s: &str) { RE.is_match(s); }
```

| 点 | 说明 |
|----|------|
| **语法糖** | 封装 OnceLock · 首次访问才 init |
| **底层** | 同 [11.7 · OnceLock](./11.7-oncelock.md) · 原子 + futex |
