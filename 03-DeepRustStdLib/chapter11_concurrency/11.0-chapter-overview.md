## 三层架构（全章统一，同 Ch10）

| 层 | 模块 | 特征 |
|----|------|------|
| **上层安全 API** | `std::sync` · `std::thread` | **全 safe** · RAII · **Send/Sync** 编译期约束 |
| **OS 无关中间层** | `sync::poison` · `wait` · MPSC 队列逻辑 | 中毒 · 等待队列 · 屏蔽平台差异 |
| **OS 底层适配** | `sys::sync` · `sys::thread` | **`unsafe` + futex** · Unix **`WaitOnAddress`** · **pthread/CreateThread** |

```text
业务  Mutex::lock() / thread::spawn() / mpsc::channel()
  └── std::sync / std::thread（stable）
        └── poison / queue / parking（OS 无关）
              └── sys::sync / sys::thread（futex · SRWLock · Event）
```

## 典型实操

```rust
use std::sync::{Arc, Mutex, mpsc};
use std::thread;

// Mutex 共享内存
let data = Arc::new(Mutex::new(0));
let mut handles = vec![];
for _ in 0..4 {
    let d = Arc::clone(&data);
    handles.push(thread::spawn(move || {
        let mut val = d.lock().unwrap();
        *val += 1;
    }));
}
for h in handles { h.join().unwrap(); }

// MPSC 通信
let (tx, rx) = mpsc::channel();
let tx2 = tx.clone();
thread::spawn(move || { tx.send(1).unwrap(); });
thread::spawn(move || { tx2.send(2).unwrap(); });
println!("{} {}", rx.recv().unwrap(), rx.recv().unwrap());
```

## 核心安全设计

| 亮点 | 说明 |
|------|------|
| **Send/Sync** | 编译期禁止跨线程传递不安全类型 |
| **RAII** | `MutexGuard` · `JoinHandle` · `Sender` Drop 自动清理 |
| **unsafe 收敛** | 裸 futex/syscall 仅在 **sys** |
| **futex 快路径** | 无竞争 **零 syscall** |
| **双范式** | **共享内存锁** + **消息通道** 按需选型 |
