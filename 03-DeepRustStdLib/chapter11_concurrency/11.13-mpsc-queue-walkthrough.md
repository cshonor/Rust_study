## 一句话

**`std::sync::mpsc`** 内部 = **FIFO 队列 + Sender 原子计数 + recv 空时 park + send 后 unpark** — 下列为 **教学用极简注释版**（真实 std 实现更复杂，含 list/queue 分文件）。

---

## 核心组件关系

```text
channel() 创建:
  Inner { queue: Mutex<VecDeque<T>>, receiver_count, sender_count, on_empty: Condvar/park }

Sender::send(msg):
  lock queue → push_back(msg) → signal receiver

Receiver::recv():
  lock queue → 若空且 senders>0 → wait
              → pop_front 或 Err(Disconnected)
```

---

## A. 队列 + send

```rust
struct Inner<T> {
    queue: Mutex<VecDeque<T>>,
    senders: AtomicUsize,   // 存活 Sender 数
    receivers: AtomicUsize, // 通常 1
    // 阻塞：Condvar 或 thread::park 的 Packet
}

impl<T> Sender<T> {
    fn send(&self, msg: T) -> Result<(), SendError<T>> {
        let inner = &self.inner;
        if inner.receivers.load(Acquire) == 0 {
            return Err(SendError(msg)); // 无人收
        }
        {
            let mut q = inner.queue.lock().unwrap();
            q.push_back(msg);
        }
        inner.wake_receiver(); // unpark / condvar.notify
        Ok(())
    }
}
```

## B. recv 阻塞与唤醒

```rust
impl<T> Receiver<T> {
    fn recv(&self) -> Result<T, RecvError> {
        loop {
            {
                let mut q = self.inner.queue.lock().unwrap();
                if let Some(msg) = q.pop_front() {
                    return Ok(msg);
                }
                if self.inner.senders.load(Acquire) == 0 {
                    return Err(RecvError); // 通道关闭且空
                }
                // 空且有 sender：释放锁并阻塞
                self.inner.wait_while_empty(&mut q); // park / condvar.wait
            }
        }
    }
}

// wake_receiver（send 侧调用）:
fn wake_receiver(&self) {
    // 若有线程在 park/condvar 上睡 → unpark_one / notify_one
}
```

## C. Sender Drop → 通道关闭

```rust
impl<T> Drop for Sender<T> {
    fn drop(&mut self) {
        if self.inner.senders.fetch_sub(1, AcqRel) == 1 {
            // 最后一个 Sender 消失 → 唤醒 Receiver 使其 recv 返回 Err
            self.inner.wake_receiver();
        }
    }
}
```

## 与 Mutex/futex 对照

| 机制 | MPSC | Mutex |
|------|------|-------|
| **争用** | 队列锁 + 消息 move | 保护同一 `T` |
| **阻塞** | recv 空 · park/futex | lock 失败 · futex |
| **范式** | **通信** | **共享内存** |
