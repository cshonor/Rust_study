# 11.7 OnceLock<T> 类型分析

> 章索引：[第 11 章](./README.md) · 前：[11.6](./11.6-once.md) · 后：[11.8 · LazyLock](./11.8-lazylock.md)

---

## 一句话

**`OnceLock<T>`** — **线程安全单次初始化容器** · **`get_or_init(f)`** / **`set`** · 替代 **`lazy_static`** 宏方案 · 与 [7.5 · OnceCell](../chapter07_interior_mutability/7.5-lazy.md) 对应但 **`Sync`**。

---

## 原书要点

```rust
static CONFIG: OnceLock<HashMap<String, Rule>> = OnceLock::new();
CONFIG.get_or_init(|| load_rules());
```

| 保证 | 说明 |
|------|------|
| **只 init 一次** | 多线程并发 `get_or_init` 安全 |
| **`Sync`** | 全局 static 友好 |
