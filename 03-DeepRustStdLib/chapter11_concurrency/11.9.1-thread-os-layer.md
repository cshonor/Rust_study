## 一句话

**`pthread_create` / `pthread_join`**（Unix）与 **`CreateThread` / `WaitForSingleObject`**（Windows）— 在 **`sys::thread`** 创建 **原生线程** 并传入 **Rust trampoline**；屏蔽句柄与调度 API 差异。
