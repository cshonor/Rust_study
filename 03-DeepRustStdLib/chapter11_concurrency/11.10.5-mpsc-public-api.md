## 一句话

**`channel()` / `sync_channel(n)`** → **`(Sender<T>, Receiver<T>)`** · **`send` move · `recv` 阻塞** — 标准 **无畏并发** 消息 API；**`Sender: Clone`** 支持多生产者。

---

## 原书要点

```rust
let (tx, rx) = mpsc::channel();
let tx2 = tx.clone();
thread::spawn(move || { tx.send(1).unwrap(); });
thread::spawn(move || { tx2.send(2).unwrap(); });
println!("{} {}", rx.recv().unwrap(), rx.recv().unwrap());
```

| 错误 | 含义 |
|------|------|
| **`SendError`** | 所有 Receiver drop · 无人收 |
| **`RecvError`** | 所有 Sender drop · 通道空且关闭 |

### 与 Mutex 选型

| 模式 | 适用 |
|------|------|
| **Channel** | 流水线 · **所有权转移** · 少锁竞争 |
| **Mutex<T>** | 多线程 **反复读写同一结构** |
