# 11.10.5 MPSC — 对外接口层

## 一句话

**`channel()`** → **`(Sender<T>, Receiver<T>)`** · **`sync_channel(n)`** 有界 · **`send` move · `recv` 阻塞** — 标准 **无畏并发** 消息 API。

---

## 原书要点

```rust
let (tx, rx) = mpsc::channel();
thread::spawn(move || { tx.send(tick).unwrap(); });
while let Ok(t) = rx.recv() { process(t); }
```

| 错误 | 含义 |
|------|------|
| **`SendError`** | 所有 Receiver drop · 无人收 |
| **`RecvError`** | 所有 Sender drop · 通道空且关闭 |

### 与 Mutex 选型

| 模式 | 适用 |
|------|------|
| **Channel** | 流水线 · 所有权转移 |
| **Mutex<T>** | 多线程 **反复读写同一结构** |
