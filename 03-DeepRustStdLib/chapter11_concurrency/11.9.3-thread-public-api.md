## 一句话

**`thread::spawn` · `JoinHandle` · `sleep` · `yield_now` · `current` · `scope`** — stable 线程 API；闭包 **`Send`** 强制编译期校验跨线程安全。

---

## 原书要点

```rust
let h = thread::spawn(|| { compute() });
let v = h.join().unwrap();
```

| API | 作用 |
|-----|------|
| **`JoinHandle`** | RAII · **`join()`** 等待结束 · 取返回值 |
| **`sleep` / `sleep_ms`** | 线程休眠 |
| **`yield_now`** | 让出时间片 |
| **`available_parallelism`** | 核心数 hint |
| **`thread::scope`** | 栈上 borrow 安全 spawn 子线程 |

### HFT

**行情线程 + 策略线程** 分离 · **`join` 处理 panic** 避免 silent failure。
