## 一句话

**`thread::spawn` · `JoinHandle` · `sleep` · `yield_now` · `current`** — 开发者使用的 **stable 线程 API**；**`thread::scope`**（新版）允许 **栈上 borrow _spawn**。

---

## 原书要点

```rust
let h = thread::spawn(|| { compute() });
let v = h.join().unwrap();
```

| API | 作用 |
|-----|------|
| **`sleep`** | 线程休眠 |
| **`available_parallelism`** | 核心数 hint |

### HFT

**行情线程 + 策略线程** 分离 · **`join` 处理 panic** 避免 silent failure。
