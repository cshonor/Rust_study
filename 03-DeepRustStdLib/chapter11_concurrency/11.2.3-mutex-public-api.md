## 一句话

**`std::sync::Mutex`** — **`lock()` → `MutexGuard`** · **`Deref`/`DerefMut` 访问 `T`** · **`Sync` 当 `T: Send`**；开发者只见 **safe API**，不见 futex。

---

## 原书要点

| API | 说明 |
|-----|------|
| **`lock()`** | 阻塞至获得锁 |
| **`try_lock()`** | 非阻塞 |
| **`MutexGuard`** | 作用域结束 **自动 unlock** |
| **`into_inner`** | 拆 Mutex 取 `T`（需无 poison 或处理） |

```rust
let m = Mutex::new(OrderBook::new());
let mut book = m.lock().unwrap();
book.apply_tick(tick);
// drop guard → unlock
```

### HFT

**订单簿全局 Mutex** — 简单正确 · 竞争大时延迟高 · 常改 **RwLock 读多写少** 或 **分片**。
