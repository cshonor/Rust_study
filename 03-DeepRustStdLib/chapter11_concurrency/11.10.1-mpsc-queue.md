# 11.10.1 消息队列类型——Queue<T>

> 章索引：[第 11 章](./README.md) · 前：[11.10](./11.10-mpsc-overview.md) · 后：[11.10.2 · 阻塞唤醒](./11.10.2-mpsc-block-wake.md)

---

## 一句话

MPSC 内部 **FIFO 队列** 存 **`T`** — 多 **`Sender` push** · 唯一 **`Receiver` pop**；队列段通常 **Mutex + 链表/VecDeque**（或分段结构），**`send` move 所有权** 无共享 alias。

---

## 原书要点

```text
Sender::send(msg) ──► Queue.push_back(msg)
Receiver::recv()  ──► Queue.pop_front() 或 block
```

| 点 | 说明 |
|----|------|
| **`channel()`** | **无界** 队列 |
| **`sync_channel(n)`** | **有界** · 满则 **send 阻塞** |
| **Sender 计数** | 原子跟踪 · 全 drop → 通道关闭 |
| **SPSC 特化** | 单 producer 可更轻 · API 仍允许多 producer |

→ 极简实现注释：[11.13 · MPSC 走读](./11.13-mpsc-queue-walkthrough.md)
