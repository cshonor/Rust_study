# 11.10.1 消息队列类型——Queue<T>

> 章索引：[第 11 章](./README.md) · 前：[11.10](./11.10-mpsc-overview.md) · 后：[11.10.2 · 阻塞唤醒](./11.10.2-mpsc-block-wake.md)

---

## 一句话

MPSC 内部 **FIFO 队列** 存放 **`T` 消息** — 多 **`Sender`** 并发 **push** · 唯一 **`Receiver` pop`**；队列段通常由 **Mutex + 链表/VecDeque** 或 **无锁段+锁** 实现（原书 p.264+ 源码待刷书）。

---

## 原书要点（脉络）

```text
Sender::send(msg) ──► Queue.push_back(msg)
Receiver::recv()  ──► Queue.pop_front() 或 block
```

| 点 | 说明 |
|----|------|
| **所有权** | **`send` move `T`** 进队列 · 无共享 alias |
| **SPSC 特化** | 单生产者可更轻 · std API 仍允许多 producer |
