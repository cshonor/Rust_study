## 一句话

**`std::thread`** 封装 **OS 原生线程**（**1:1 模型**）— **`spawn`/`JoinHandle`** · **`LocalKey`** · **sleep/yield**；同样 **OS 层 → OS 无关 → 对外 API**（[11.9.1～11.9.3](./11.9.1-thread-os-layer.md)）。

---

## 原书要点

| 概念 | 说明 |
|------|------|
| **1:1 模型** | Rust 线程 = **内核线程** · 调度交 OS · 非 Go 式 green thread |
| **`spawn(F)`** | **`F: FnOnce() + Send + 'static`** |
| **`JoinHandle`** | RAII · **`join()`** 阻塞等待 · 取返回值 |
| **`LocalKey`** | 线程本地存储 **`#[thread_local]`** |

↔ [4.3.1 · Send/Sync](../chapter04_primitive_types/4.3.1-marker-traits.md)

与 **进程** [Ch10](../chapter10_process_management/README.md)：**共享地址空间** · 须 **Sync/Mutex/channel** 防数据竞争。
