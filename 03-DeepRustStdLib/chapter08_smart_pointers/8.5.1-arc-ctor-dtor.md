## 一句话

**`Arc::new`** 堆分配 **`ArcInner { strong: AtomicUsize, weak: AtomicUsize, data }`** · **`clone`** 用 **`fetch_add`** · **最后一个 Arc drop** 时 **`fetch_sub`** 至 0 → 释放 **`T`** — 全程 **原子序** 保证多线程正确。

---

## 原书要点

### 原子计数

```rust
// 概念
strong.fetch_add(1, Ordering::Relaxed);  // clone
strong.fetch_sub(1, Ordering::Release);  // drop — 可能 AcqRel fence
```

| 与 Rc 相同 | 与 Rc 不同 |
|------------|------------|
| strong/weak 双计数语义 | **原子** RMW |
| strong→0 drop T | **跨线程** clone/drop 安全 |

### 内存序（直觉）

| 场景 | 说明 |
|------|------|
| **Relaxed 计数** | 仅保证计数正确 |
| **Release/Acquire** | drop 侧与 **data 初始化** 可见性协作 |
