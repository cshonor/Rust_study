## 一句话

**`Rc`** 还提供 **`get_mut`（唯一 strong 时）**、**`try_unwrap`/`into_inner`**、**`weak_count`/`strong_count`** 等 — 在共享模型下 **尽量恢复独占** 或 ** introspect 计数**。

---

## 原书要点

| API | 条件 | 作用 |
|-----|------|------|
| **`get_mut`** | **strong == 1** 且无 active Weak 借用冲突 | **`&mut T`** 原地改 |
| **`try_unwrap`** | strong==1 | **`Ok(T)`** 取出 · 否则 **`Err(Rc)`** |
| **`Rc::downgrade`** | — | → **`Weak`** |
| **`strong_count` / `weak_count`** | — | 调试 · 检测泄漏 |

### 与 RefCell 组合

```rust
let g = Rc::new(RefCell::new(Graph::new()));
// 多 Rc 共享 + 运行期 borrow_mut 改内部
```

**单线程** 图结构标准模式 — **勿** 与 `Arc` 混用线程。
