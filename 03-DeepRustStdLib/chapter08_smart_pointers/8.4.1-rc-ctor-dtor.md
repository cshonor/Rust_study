# 8.4.1 Rc<T> 类型的构造函数及析构函数

> 章索引：[第 8 章](./README.md) · 前：[8.4](./8.4-rc-overview.md) · 后：[8.4.2 · Weak](./8.4.2-rc-weak.md)

---

## 一句话

**`Rc::new`** 堆分配 **`RcBox { strong:1, weak:0, value }`** · **`clone`** 仅 **strong+1**（浅拷贝指针）· **最后一个 `Rc` drop** 时 strong→0 → **drop `T` + 释放 box**（weak 仍可能存活）。

---

## 原书要点

### 构造

```rust
let a = Rc::new(data);  // strong = 1, weak = 0
let b = Rc::clone(&a);  // strong = 2, 同一块堆
```

| 操作 | strong | weak | data |
|------|--------|------|------|
| **`Rc::new`** | 1 | 0 | 初始化 |
| **`clone` (Rc)** | +1 | — | — |
| **`drop` 最后一个 Rc** | →0 | — | **`drop(T)`** + 释放（若无 weak 或 weak 亦清理） |

### 析构链

```text
strong 归零  ──►  drop value T
              ──►  若 weak 也为 0 → 释放 RcBox 壳
weak 最后归零 ──►  仅释放控制块（value 已不存在）
```

### 非线程安全

计数器为 **普通 usize** — **非原子** · 多线程 `clone` **数据竞争** → 须 **`Arc`**。
