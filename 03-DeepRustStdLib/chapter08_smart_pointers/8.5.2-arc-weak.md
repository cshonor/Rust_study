## 一句话

**`Arc::downgrade` → `Weak<T>`** 与 **`Rc::Weak`** 同语义 — **原子 weak 计数** · **`upgrade()` → `Option<Arc<T>>`**；用于 **多线程** 下打破 **Arc 环** 或 **缓存不延长生命周期** 的引用。

---

## 原书要点

| 操作 | 说明 |
|------|------|
| **`Arc::downgrade`** | weak **原子 +1** · 不增 strong |
| **`Weak::upgrade`** | strong>0 → **`Some(Arc)`**（原子增 strong）· 否则 `None` |
| **`Weak::clone`** | weak 原子 +1 |

### 多线程环

```text
Arc 环  ──►  与 Rc 环相同泄漏
一侧 Weak  ──►  strong 可归零  ──►  data 释放  ──►  upgrade 得 None
```

### 典型模式

| 模式 | 用途 |
|------|------|
| **观察者 / 回调注册表** | `Weak` 订阅 · 主体 drop 后自动失效 |
| **父子双向** | 父 `Arc` 子 · 子 `Weak` 父 |
