# 8.3 Vec<T> 类型分析

> 章索引：[第 8 章](./README.md) · 前：[8.2](./8.2-rawvec.md) · 后：[8.3.1 · 基础](./8.3.1-vec-basics.md)

---

## 一句话

**`Vec<T>`** = **`RawVec<T>` + `len`** — 动态连续数组的上层封装；**insert/remove/append** 等热路径用 **unsafe + ptr::copy** 直拷元素，对外仍 **safe**；并 impl 完整 **迭代器族**（[8.3.2](./8.3.2-vec-iterator.md)）。

---

## 原书要点

### 两层架构

```rust
// 概念布局
pub struct Vec<T, A: Allocator = Global> {
    buf: RawVec<T, A>,
    len: usize,
}
```

| 子节 | 内容 |
|:--:|------|
| [8.3.1](./8.3.1-vec-basics.md) | push/insert/remove · **mem copy** |
| [8.3.2](./8.3.2-vec-iterator.md) | `IntoIter` · `iter` · `drain` |

### 函数式与性能

| 点 | 说明 |
|----|------|
| **迭代器丰富** | 与 [第 5 章](../chapter05_iterators/README.md) 深度结合 |
| **reserve** | 预分配避免热路径 realloc — HFT 关键 |
| **Deref → `[T]`** | 切片 API 全可用 |

### 本章在 ALLOC 中的位置

```text
Box  ── 固定大小独占堆
RawVec + Vec  ── 动态连续堆
Rc/Arc  ── 共享所有权堆对象
```
