# 8.5.3 Arc<T> 的其他函数

> 章索引：[第 8 章](./README.md) · 前：[8.5.2](./8.5.2-arc-weak.md) · 后：[8.6 · Cow](./8.6-cow-overview.md)

## 一句话

**`Arc`** 同样提供 **`get_mut` / `try_unwrap`（唯一 strong）**、**`strong_count`** 等 — 跨线程广播后 **仍可在「唯一 owner」时** 取回 **`&mut T` 或 T 本身**；并发改共享数据 **必须** 另加锁。

---

## 原书要点

| API | 说明 |
|-----|------|
| **`Arc::get_mut`** | **唯一 Arc**（且无 Weak 竞态）→ **`&mut T`** |
| **`Arc::try_unwrap`** | strong==1 → **`Ok(T)`** |
| **`Arc::make_mut`** | COW：多 owner 时 **clone 内部 T** 再 mut |
| **`strong_count` / `weak_count`** | 调试 |

### Arc + 锁（原书强调）

```rust
let data = Arc::new(Mutex::new(OrderBook::new()));
let book = data.lock().unwrap();  // 改 T 的正确方式
```

| 反模式 | 后果 |
|--------|------|
| **`Arc<Cell<_>>`** | Cell !Sync · 通常 **不可 Arc** |
| **无锁改 `Arc<T>` 内部** | **数据竞争** · UB |

### HFT

**`Arc` clone** 每条策略线程 —  profiling **Relaxed fetch_add** 成本；**只读** 用 **`Arc<Snapshot>`** 无 Mutex。
