## 一句话

**`Rc<T>`**（Reference Counted）实现 **单线程多 owner 共享堆对象** — 内部 **`RcBox<T>`** 含 **data + strong/weak 双计数**；纯 **`Rc` 环** 会泄漏 → **`Weak<T>`** 打破（[8.4.2](./8.4.2-rc-weak.md)）。

---

## 原书要点

### 双计数器模型

```text
RcBox<T> {
  strong: usize,   // Rc clone +1 · drop -1 → 0 则释放 data
  weak:   usize,   // Weak clone +1 · 不影响 data 存活
  value:  T,
}
```

| 指针 | strong | weak | 持有 data? |
|------|--------|------|------------|
| **`Rc<T>`** | +1 | — | 是 |
| **`Weak<T>`** | — | +1 | **否**（须 `upgrade`） |

### 适用场景

| 适合 | 不适合 |
|------|--------|
| **图/树** 多入边 · 单线程 | **跨线程** → **`Arc`** |
| **`Rc<RefCell<T>>`** | 多线程 mutate → **`Arc<Mutex<T>>`** |
