## 一句话

**`GlobalAlloc`**（unsafe trait）是底层 **alloc/dealloc/realloc**；**`Allocator`** 是更高层分配器抽象 — **`Global`**（alloc 默认）与 **`System`**（std · OS）实现堆分配。

---

## 原书要点

| 接口 | 层级 | 说明 |
|------|------|------|
| **`GlobalAlloc`** | 底层 unsafe | `unsafe fn alloc(layout) -> *mut u8` 等 |
| **`Allocator`** | 较新抽象 | 返回 **`NonNull` + 错误** 等安全封装方向 |
| **`Global`** | alloc 默认单元 | 链接 **`#[global_allocator]`** 静态 |
| **`System`** | std | 封装 **libc malloc / 平台 SYSCALL** |

```text
Vec::push  →  RawVec  →  GlobalAlloc::alloc(Layout)
                              ↑
                    默认 System（用户态程序）
```

### 自定义分配器

```rust
// #![global_allocator]
// static MY: MyPoolAllocator = MyPoolAllocator;
```

→ 内存池 · bump allocator · HFT 预分配 arena
