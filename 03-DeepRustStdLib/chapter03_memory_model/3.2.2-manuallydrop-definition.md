## 一句话

**`ManuallyDrop<T>`** 告诉编译器：**生命周期结束时不要自动 `drop`** — 防止对 **未初始化** 内容误 drop（UB），或在 **`take`/`replace` 浅拷贝转移** 后 **双重释放**。

---

## 原书要点

| 问题 | ManuallyDrop 作用 |
|------|-------------------|
| 未初始化内存被当作 `T` drop | 包在 `MaybeUninit` + **`ManuallyDrop`** 流程中控制 drop 时机 |
| **`mem::replace` / 所有权转移** | 原槽位 **不应** 再 drop 已移走的值 |
| 手动释放 | `ManuallyDrop::drop()` — **显式**触发 |

### 与 `mem::forget`

| | `ManuallyDrop` | `mem::forget` |
|---|----------------|---------------|
| 语义 | 包装类型 · **仍拥有** `T` 直到手动 drop | **泄漏** — 永不 drop |
| 典型 | 自定义容器内部 | 极少数 intentional leak |
