# 3.7 动态内存申请及释放

> 章索引：[第 3 章](./README.md) · 前：[3.6.3-mem-other-fns](./3.6.3-mem-other-fns.md) · 后：[3.7.1-memory-layout](./3.7.1-memory-layout.md)

## 一句话

动态堆内存靠 **`Layout` 描述布局** + **`Allocator` / `GlobalAlloc` 接口** 分配释放 — **`Global`**（alloc/core）与 **`System`**（std · **SYSCALL**）是两条默认路径。

---

## 原书要点

### 子节

| 节 | 主题 |
|:---:|------|
| **3.7.1** | **`Layout`** — size · align |
| **3.7.2** | **`Allocator` / `GlobalAlloc` / `Global` / `System`** |

### 三层回顾

| 层 | 分配 |
|----|------|
| **`core::alloc::Layout`** | 只 **描述** 布局 |
| **`alloc` + `Global`** | 默认 **堆**（`#[global_allocator]` 可换） |
| **`std` + `System`** | 经 **操作系统** SYSCALL（malloc 等） |
