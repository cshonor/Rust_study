## 一句话

**`NonNull<T>`** = **保证地址非 null** 的裸指针封装 — 便于异常逻辑与 **空值优化**（`Option<NonNull<T>>` 与单指针同大）。

---

## 原书要点

### 相对 `*mut T`

| | `*mut T` | `NonNull<T>` |
|---|----------|--------------|
| **null** | 允许 | **Invariant：非 0 地址** |
| 优化 | — | **`Option<NonNull<T>>`** 可 **niche optimize** 为单字宽 |
| 转换 | `NonNull::new(ptr)` 返回 `Option` | 从 `&T` 构造常见 |

### 用途

- **`Vec` / `Box` 内部** 常存 `NonNull` 而非裸指针
- 链表、自定义 allocator 返回 **非空** 块
