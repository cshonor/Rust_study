## 一句话

**`mem::forget(t)`** 阻止 **`t` 的 `drop`** —  intentional leak 或配合 **手动内存管理**；误用会导致资源泄漏，与 **`ManuallyDrop`** 场景相关。

---

## 原书要点

| API | 作用 | 风险 |
|-----|------|------|
| **`mem::forget(t)`** | **永不** drop `t` | **泄漏**（内存/文件句柄若未另释） |
| **`mem::drop(t)`** | 显式提前 drop | 一般直接 `drop(x)` 或作用域结束 |
| **`mem::swap`** | 交换两可变占位 | 安全 · 常用于 `Option` 式状态机 |

### 与 ManuallyDrop

| 场景 | 更常用 |
|------|--------|
| 容器内部精确控制 | **`ManuallyDrop`** |
| 一次性 intentional leak | **`forget`**（极少） |
