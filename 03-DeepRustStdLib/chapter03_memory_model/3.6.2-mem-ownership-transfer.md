## 一句话

**`mem::take`** 与 **`mem::replace`** 在 **不能用 `=` 转移** 的场景（如 **`&mut` 结构体字段**）用 **浅拷贝/替换** 安全 **移出或交换** `T` 的所有权。

---

## 原书要点

| API | 行为 |
|-----|------|
| **`mem::take(&mut place)`** | `T: Default` — 用 **默认值替换** `place`，**返回** 旧值 |
| **`mem::replace(&mut place, new)`** | 存入 `new`，**返回** 被替换掉的旧 `T` |

```rust
fn reset(opt: &mut Option<Heavy>) {
    let old = opt.take(); // Option 专属；通用场景用 take/replace
}
```

### 为何需要

- **`&mut T` 不能 move 出 `T`**（否则留下未初始化位）
- **浅拷贝语义** — 对 `String`/`Vec` 是 **指针所有权转移**，非 deep clone
