## 一句话

**`Layout`** 严格描述一块内存的 **size + align** — 分配器 **`alloc` / `dealloc`** 的输入；与 **`mem::size_of` / `align_of`** 及 **#[repr(C)]** 订单簿布局同源。

---

## 原书要点

| 类型/API | 作用 |
|----------|------|
| **`Layout::new::<T>()`** | 单类型布局 |
| **`Layout::from_size_align(size, align)`** | 原始字节块（须满足 align 规则） |
| **`Layout::array<T>(n)`** | `n` 个 `T` 连续布局 |

### 对齐为何重要

- CPU **未对齐访问** 可能 fault 或性能损失
- **false sharing** — HFT 结构体常 **`align(64)`** 隔离缓存行
