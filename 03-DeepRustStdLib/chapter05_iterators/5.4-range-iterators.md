## 一句话

**`for i in 0..10`** 底层是 **`Range<T>: Iterator`** — 步进依赖 **`Step` Trait**（`forward`/`backward` 计算下一元素），与 [4.3.4 · RangeBounds](../chapter04_primitive_types/4.3.4-range-traits.md) 衔接。

---

## 原书要点

### 语法 → 迭代

```rust
for i in 0..10 { … }   // Range<i32> implements Iterator
for i in 0..=9 { … }   // RangeInclusive — 单独 impl
```

| 类型 | 迭代方式 |
|------|----------|
| **`Range<T>`** | `start` 每次 **`Step::forward`** 直到 `end`（**左闭右开**） |
| **`RangeInclusive<T>`** | 含末端 · 用 **`bool exhausted` 等** 状态 |

### Step Trait

| 方法 | 作用 |
|------|------|
| **`steps_between`** | 两值间步数（若可计算） |
| **`forward` / `backward`** | 从当前值 **安全** 步进 |

**实现 `Step` 的类型**：整数、`char` 等 — 自定义类型需自行 impl 才能 `a..b` 迭代。

### 性能

无容器分配 · **寄存器级计数** — 热路径 `for` 常完全内联。
