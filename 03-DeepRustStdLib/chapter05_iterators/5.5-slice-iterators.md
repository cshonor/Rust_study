# 5.5 切片类型迭代器

## 一句话

切片 **`Iter` / `IterMut`** 用 **两个裸指针**（当前元素 + 尾后边界）实现 **C 式指针步进** — 在 **safe Rust** 中封装，边界由 **尾指针比较** 保证，效率接近手写循环。

---

## 原书要点

### 结构（原书）

```text
Iter<'a, T> {
  ptr: NonNull<T>,      // 当前元素
  end: NonNull<T>,      // one-past-last（尾后）
  _marker: PhantomData<&'a T>,
}
```

| 步骤 | 行为 |
|------|------|
| **`next`** | `ptr == end` → `None`；否则取 `*ptr`，`ptr = ptr.add(1)` |
| **`IterMut`** | 同上 + **`&mut`** 别名规则由借用检查保证 |

### 设计要点

| 项 | 说明 |
|----|------|
| **双指针** | 避免维护 index + len 两次边界检查（可优化为单次比较） |
| **`NonNull`** | 非空 invariant · [3.4 · NonNull](../chapter03_memory_model/3.4-nonnull.md) |
| **零分配** | 仅两个指针宽度的状态机 |

### HFT

订单簿 **深度档 `[PriceLevel; N]` 或 slice** 扫描 — `iter()` 常编译为 **与索引 for 等价的机器码**。
