## 一句话

**`array::Iter` / `IterMut`** 与 **切片迭代器同构** — 双指针（或指针+长度）在 **栈上定长布局** 上步进，产出 **`&T` / `&mut T`**，**不 move** 元素，无 `MaybeUninit` 负担。

---

## 原书要点

| 类型 | `Item` | 容器 afterward |
|------|--------|----------------|
| **`Iter<'a, T, N>`** | `&'a T` | 数组 **完整保留** |
| **`IterMut<'a, T, N>`** | `&'a mut T` | 可 **原地修改** 各槽 |

```rust
let mut a = [1, 2, 3];
for x in a.iter_mut() {
    *x *= 2;
}
// a == [2, 4, 6]
```

### 与切片关系

**`&array`** 可 **coerce** 为 **`&[T]`** — 许多 API 统一用 **slice iter**；数组专用 iter 保留 **const N** 在类型中（泛型常量）。

### HFT

**`[PriceLevel; 10]`** 档位数固定 — **`iter()`/`iter_mut()`** 零堆分配遍历。
