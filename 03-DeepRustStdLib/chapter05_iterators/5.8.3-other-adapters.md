# 5.8.3 其他适配器

## 一句话

除 **Map/Chain** 外，标准库还提供 **`Filter`/`FlatMap`/`Zip`/`Take`/`Skip`/`Enumerate`/`Peekable`** 等 — 均属 **包装迭代器 + 惰性 next**，覆盖筛选、展平、配对、截断与窥视等常见模式。

---

## 原书要点

| 适配器 | 作用 |
|--------|------|
| **`Filter`** | 谓词为真才 yield |
| **`FilterMap`** | 映射为 `Option` — `Some` 才 yield |
| **`FlatMap` / `flatten`** | 嵌套迭代器展平 |
| **`Zip`** | 两迭代器 **同步步进** — 短者结束即停 |
| **`Take` / `Skip`** | 取前 n / 跳过前 n |
| **`Enumerate`** | `(index, item)` |
| **`Peekable`** | 可 **`peek()`** 不看消费 |
| **`Fuse`** | 首次 `None` 后永久 `None` |
| **`Rev`**（DoubleEnded） | 反向 |

### 组合示例

```rust
ticks.iter()
    .enumerate()
    .filter(|(_, t)| t.valid)
    .map(|(i, t)| (i, t.price))
    .take(100)
    .collect::<Vec<_>>();
```

### HFT 提示

| 适配器 | 实盘 |
|--------|------|
| **`take`** | 限深档 · 限 tick 批大小 |
| **`filter`** | 注意闭包 **分支预测** |
| **`zip`** | 对齐两路序列（时间戳+价格） |
