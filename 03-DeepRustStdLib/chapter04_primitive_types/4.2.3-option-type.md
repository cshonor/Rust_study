# 4.2.3 Option<T> 类型

## 一句话

**`Option<T>` 是标准库定义的 enum**（`Some(T)` | `None`），**不是** 编译器内置 magic 类型 — 用 **`map` / `and_then` / `unwrap_or_else`** 等闭包 API 表达「值是否存在」，鼓励 **函数式、无 panic** 的解封装。

---

## 原书要点

### 定义（core）

```rust
pub enum Option<T> {
    Some(T),
    None,
}
```

| 点 | 说明 |
|----|------|
| **编译器协作** | `match` 穷尽检查 · **`?` 对 Option**（需 `Try`）· **niche 优化**（`None` 常不占额外 tag） |
| **非 magic** | 源码在 **`library/core/src/option.rs`**，与自定义 enum 同一套规则 |

### 常用 combinator

| 方法 | 语义 |
|------|------|
| `map(f)` | `Some(x)` → `Some(f(x))` |
| `and_then(f)` | 链式 flatMap |
| `unwrap_or_else(f)` | 缺省值 **惰性** 计算 |
| `ok_or(e)` | → `Result` |

### 设计意图

```text
C: NULL 指针     ──►  Rust: Option<T> 显式分支 + 类型系统强制处理 None
```
