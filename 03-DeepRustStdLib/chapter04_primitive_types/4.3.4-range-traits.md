## 一句话

**`a..b` / `a..=b` / `..` / `..=n`** 等范围语法 desugar 为 **`Range` / `RangeInclusive` / `RangeFrom` / …** 结构体 — 它们实现 **`RangeBounds`**，供 **`slice::get`、迭代器、`BTreeMap` 区间查询** 等统一描述 **左闭右开 / 闭区间**。

---

## 原书要点

### 语法 → 类型

| 语法 | 标准库类型 | 区间语义 |
|------|------------|----------|
| `a..b` | `Range<T>` | **[a, b)** 左闭右开 |
| `a..=b` | `RangeInclusive<T>` | **[a, b]** 两端闭 |
| `a..` | `RangeFrom<T>` | **[a, ∞)** |
| `..b` | `RangeTo<T>` | **(-∞, b)** |
| `..=b` | `RangeToInclusive<T>` | **(-∞, b]** |
| `..` | `RangeFull` | 全范围 |

### RangeBounds Trait

| 作用 | 说明 |
|------|------|
| **`start_bound` / `end_bound`** | 返回 `Bound::Included/Excluded/Unbounded` |
| **消费者** | `[T]::get(range)` · `(a..b).contains(&x)` · `for i in 0..n` |

```rust
let s = &data[2..5];   // Range<usize> + Index<Range<usize>>
```
