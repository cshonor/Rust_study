## 一句话

**Trait 是 Rust 的灵魂** — 本章的 **`?`、`+`、`<`、`..`、`[]`** 等语法表象，底层都是 **标准库 Trait + 宏 + 泛型**；读懂 Marker / ops / Try / Range / Index，即读懂 **「语法糖从哪来」**。
