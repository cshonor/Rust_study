# 4.3.5 索引运算符 Trait

> 章索引：[第 4 章](./README.md) · 前：[4.3.4](./4.3.4-range-traits.md) · 后：[第 5 章 · 迭代器](../chapter05_iterators/README.md)

## 一句话

**`arr[i]` / `s[i]`** 语法糖来自 **`Index` / `IndexMut` Trait** — 数组、切片、`Vec`、字符串、`HashMap` 等 **各自 impl**，统一「下标访问」表象，底层可以是 **边界检查、O(1) 偏移或 hash 查找**。

---

## 原书要点

### Trait 签名（简化）

```rust
pub trait Index<Idx: ?Sized> {
    type Output: ?Sized;
    fn index(&self, index: Idx) -> &Self::Output;
}

pub trait IndexMut<Idx: ?Sized>: Index<Idx> {
    fn index_mut(&mut self, index: Idx) -> &mut Self::Output;
}
```

| 类型 | `Idx` 常见形式 |
|------|----------------|
| `[T]` / `Vec<T>` | `usize` · **`Range<usize>`** 等 |
| `str` | `Range<usize>`（字节区间，非 char） |
| `[T; N]` | `usize` |
| `HashMap<K,V>` | `&K` |

### 与 panic

| API | 行为 |
|-----|------|
| **`arr[i]`** | 越界 → **panic**（debug 友好） |
| **`arr.get(i)`** | 返回 **`Option<&T>`** — 安全 |

### 组合 Range

```rust
let mid = &v[1..4];  // Index<Range<usize>> → 切片引用
```
