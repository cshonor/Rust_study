## 一句话

标准库为 **`i8`～`i128` / `u8`～`u128` / `isize` / `usize`** 提供 **显式溢出策略**（`wrapping_*`、`checked_*`、`saturating_*`、`overflowing_*`）与 **字节序转换** — 体现 Rust 对 **系统级数值安全** 的严谨默认（debug 溢出 panic，release 可 `wrapping`）。

---

## 原书要点

### 溢出策略（同一运算，四种语义）

| 方法前缀 | 溢出行为 | 典型场景 |
|----------|----------|----------|
| **`wrapping_*`** | 按位环绕（模 2ⁿ） | hash 混合、序列号回绕 |
| **`checked_*`** | 返回 `Option` | 金融数量 **禁止 silent wrap** |
| **`saturating_*`** | 钳制到 MIN/MAX | 音量、百分比 |
| **`overflowing_*`** | `(T, bool)` 带溢出标志 | 需同时知道结果与是否溢出 |

### 字节序 /  wire format

| API | 作用 |
|-----|------|
| `to_be()` / `to_le()` | 主机 → 大/小端 |
| `from_be_bytes` / `from_le_bytes` | 字节数组 → 整数 |
| `swap_bytes` | 交换字节序 |

### 与 intrinsic

位计数、rotate 等 → [4.1.2](./4.1.2-math-bit-fns.md)；**`+` 运算符** → [4.3.2 · Add](./4.3.2-arithmetic-traits.md)。
