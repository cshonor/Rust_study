## 一句话

**`core::ops`** 下的 **`Add`/`Sub`/`Mul`/…** 重载 **算术运算符**；**`PartialEq`/`Eq`/`PartialOrd`/`Ord`** 拆分 **相等与比较** — 浮点 **`NaN`** 只能 **偏序（PartialOrd）**，不能 **全序（Ord）**。

---

## 原书要点

### 算术 Trait

| Trait | 运算符 | 关联类型 |
|-------|--------|----------|
| `Add` | `+` | `type Output` |
| `Sub` | `-` | |
| `Mul` | `*` | |
| `Div` | `/` | |
| `Rem` | `%` | |
| `Neg` | `-x` | |
| `Not` | `!x`（位非） | |

```rust
impl Add for MyTick {
    type Output = Self;
    fn add(self, rhs: Self) -> Self { … }
}
```

### 关系 Trait（严谨拆分）

| Trait | 语法 | 约束 |
|-------|------|------|
| **`PartialEq`** | `==` / `!=` | 允许 **不可比**（浮点 NaN） |
| **`Eq`** | 等价关系公理 | 需 **`PartialEq` + 自反/对称/传递** |
| **`PartialOrd`** | `<` `>` `<=` `>=` | **偏序** — 部分元素不可比 |
| **`Ord`** | 全序 | 需 **`Eq` + **`PartialOrd`** + 全序公理 |

```text
f64: PartialEq + PartialOrd  ✓
f64: Ord                     ✗  （NaN 与任何值 including NaN 无序）
```
