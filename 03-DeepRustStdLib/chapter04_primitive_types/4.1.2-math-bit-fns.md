# 4.1.2 数学函数及位操作函数

## 一句话

整数/浮点的 **硬件级** 运算与 **位操作**（leading/trailing zeros、rotate、popcount 等）通过 **intrinsic** 暴露 — 标准库在 `num` 模块为各宽度类型提供 **安全、可移植** 的方法封装。

---

## 原书要点

### 数学（数值 intrinsic）

| 方向 | 示例 API |
|------|----------|
| 浮点 | `sin`/`cos`/`sqrt` 等 — 常委托 **libm** 或目标内置指令 |
| 整数 | 乘除、溢出检测 — 部分走 **单条 CPU 指令** |

### 位操作（整数）

| 方法族 | 作用 |
|--------|------|
| `leading_zeros` / `trailing_zeros` | 前导/尾随零个数 |
| `rotate_left` / `rotate_right` | 循环移位 |
| `count_ones` / `count_zeros` | 位计数 |
| `swap_bytes` / `to_be` / `to_le` | **字节序** 转换（网络协议、二进制 wire format） |

↔ [4.2.1 · 整数类型](./4.2.1-integer-types.md) · `wrapping_*` / `checked_*` 等 **安全策略** 与 intrinsic **分层**

### HFT 关联

| 场景 | 用法 |
|------|------|
| 价格 tick 位域 | `&` / `rotate` / mask |
| 协议解析 | `from_be_bytes` / `to_le` |
| 快速 hash 混合 | `rotate_left` + xor |
