# 9.1.3 C 语言字符串类型适配

> 章索引：[第 9 章](./README.md) · 前：[9.1.2](./9.1.2-c-va-list.md) · 后：[9.1.4 · OsString](./9.1.4-osstring.md)

---

## 一句话

C 字符串 = **以 `\0` 结尾的字节序列、无长度前缀**；Rust 用 **`CStr` / `CString`** 适配 — 前者 **借用** NUL 结尾缓冲，后者 **拥有** 并在末尾 **写入 `\0`** 供 FFI 传出。

---

## 原书要点

| 类型 | 所有权 | 用途 |
|------|--------|------|
| **`&CStr`** | 借用 | 来自 **`CStr::from_ptr`**（unsafe）或静态 |
| **`CString`** | 拥有 | **`into_raw` → FFI** · **`from_raw` 收回** |
| **`str` / `String`** | UTF-8 有长度 | **不能** 直接当 C 串 |

```rust
let c = CString::new("hello")?;  // 内部含 \0
let ptr = c.as_ptr();             // *const c_char
```

### 鸿沟

| C | Rust |
|---|------|
| 任意字节到 `\0` | **UTF-8 验证**（`CString::new` 拒绝内部 `\0`） |
| 无编码约定 | **`str` 必 UTF-8** |

**路径/系统串** 往往 **非 UTF-8** → [9.1.4 · `OsString`](./9.1.4-osstring.md)
