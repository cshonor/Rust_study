# 9.5 文件描述符及句柄

> 章索引：[第 9 章](./README.md) · 前：[9.4](./9.4-syscall-wrapper.md) · 后：[9.5.1 · FD 所有权](./9.5.1-fd-ownership.md)

## 一句话

Unix **一切皆文件** — **socket / pipe / 普通文件** 在底层均为 **整数 FD**；Windows 用 **HANDLE**。**`std`** 把这些 OS 句柄 **包装进 Rust 类型**，交给 **所有权 + Drop** 自动 **close**，避免 C 式 **泄漏/双重 close**。

---

## 原书要点

### 统一抽象

| OS | 内核对象标识 |
|----|--------------|
| **Linux 等** | **`fd: RawFd`（i32）** |
| **Windows** | **`HANDLE`** |

```text
File · TcpStream · UnixStream · Stdio
  └── 内部 OwnedFd / Handle（平台类型）
        └── Drop → close(2) / CloseHandle
```
