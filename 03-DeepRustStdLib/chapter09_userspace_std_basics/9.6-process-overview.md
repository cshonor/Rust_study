## 原书要点

### 与本章链条

```text
FFI + OsString  ──►  System 堆  ──►  SYSCALL  ──►  FD Drop
                                              └──►  进程/子进程 FD 继承
```

| 概念 | std 侧 |
|------|--------|
| **创建进程** | `Command::spawn` → `fork`+`exec` / `CreateProcess` |
| **标准 IO** | 子进程 **FD 0/1/2** · **`Stdio::piped()`** |
| **等待退出** | `Child::wait` → `waitpid` |

### 资源安全

**`Child`** 与 **`File`** 同类 — **Drop 时回收**（避免 zombie 须 **`wait`** 或 **`kill`** 策略）。
