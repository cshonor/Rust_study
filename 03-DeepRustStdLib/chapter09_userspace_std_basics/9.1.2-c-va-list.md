# 9.1.2 C 语言的 va_list 类型适配

## 一句话

C 的 **`printf` 族** 依赖 **`va_list` 可变参数** — 标准库在 **`core::ffi`** 侧提供 **平台相关的 `VaList`/`VaListImpl` 封装**，使 Rust 能 **安全调用** 或 **实现** 带 `...` 的 C 接口（实现细节 **高度平台相关**）。

---

## 原书要点

| 概念 | 说明 |
|------|------|
| **`va_list`** | C 中遍历 `...` 参数的游标 |
| **`va_start` / `va_arg` / `va_end`** | 宏/内建 · 须成对使用 |
| **Rust 侧** | 封装为 **`VaList`** · **`unsafe`** 提取下一参数 |

```text
OS C API（如 syslog 风格）
  └── 需要 va_list 互操作
        └── std 按 arch/OS 提供不同 impl
```

| 注意 | |
|------|---|
| **ABI  fragile** | 错误 `va_arg` 类型 → UB |
| **少用** | 业务 FFI 优先 **固定签名** |
