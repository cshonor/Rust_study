# 4.1 Manage Unsafe Boundaries（管理 unsafe 边界）

> 所属：**Coping with Fear** · [← 章索引](./README.md)

1. **缩小边界** — `unsafe` 封进私有模块，暴露安全 API。
2. **同一信任域** — 凡能修改与 unsafe 假设相关状态的代码，都在边界内。
3. **优先安全替代** — 标准库 / crate 生态常已有封装。
