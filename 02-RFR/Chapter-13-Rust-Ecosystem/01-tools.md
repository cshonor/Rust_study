# 1.1 Tools（工具）

> 所属：**What's Out There?** · [← 章索引](./README.md)

Cargo / 编译周边 — 依赖卫生与 CI 门禁。

| 工具 | 作用 |
|------|------|
| **`cargo-udeps`** | 发现声明但未使用的依赖 |
| **`cargo-outdated`** | 查看可升级的 semver 范围 |
| **`cargo tree -i crate`** | 反向查谁引入了某依赖 |
| **`cargo-deny`** | 许可证、advisory、重复 crate |
| **`cargo-expand`** | 展开宏，对照排错 |
