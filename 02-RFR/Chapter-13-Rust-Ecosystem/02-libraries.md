# 1.2 Libraries（库）

> 所属：**What's Out There?** · [← 章索引](./README.md)

高质量生态 crate 是「野生模式」的教材：

| 领域 | 示例 | 学什么 |
|------|------|--------|
| 异步 | **Tokio** | runtime、IO、同步原语 |
| 图 | **petgraph** | 索引指针、Arena 思路 |
| 迭代器 | **itertools** | 扩展 trait |
| 服务抽象 | **tower** | `Service` / `ServiceExt` |
| 序列化 | **serde** | derive、feature 设计 |

读源码时对照 [第 3 章 API 设计](../Chapter-03-Designing-Interfaces/README.md) 与 [第 5 章 项目结构](../Chapter-05-Project-Structure/README.md)。
