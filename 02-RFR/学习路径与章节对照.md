# 《Rust for Rustaceans》— 学习定位与章节对照

> **本仓库主线**：以**看书 + 整理笔记**为主；目录按 **`00`～`05`** 编号：Rust 三书 → Nomicon → 并发/async/网络 → 编译器/LLVM。  
> **RFR 定位**：原理向进阶 — 建立「内存、类型、trait、unsafe」在机器上的图景；不是语法第一课。  
> **全书笔记入口**：[`RFR-本书目录.md`](./RFR-本书目录.md) · **纯阅读顺序**：[`docs/纯阅读路线.md`](../docs/纯阅读路线.md)

## 二、推荐学习顺序（与 README 一致）

### 主线

```text
（并行或先行）00-Book 1～10 章语法底座
      ↓
① 02-RFR 完整通读          ← 懂原理、内存、trait、async 模型
      ↓
② 01-ER 完整通读           ← 35 条工程习惯
      ↓
③ 03-Rust_Nomicon          ← unsafe 边界与内存契约
      ↓
④ 04-Async-Concurrency-Network   ← 01-atomic → 02-async_tokio → 03-rust_network
      ↓
⑤ 05_Compilers-and-LLVM-Learning/04_Learn-LLVM-17（按需）  ← IR / 优化对照
```

详见 [`docs/纯阅读路线.md`](../docs/纯阅读路线.md) · [`04-Async-Concurrency-Network/README.md`](../../04-Async-Concurrency-Network/README.md)。

### 阶段速记

| 阶段 | 做什么 | 别做什么 |
|------|--------|----------|
| **底座** | `00-Book` 所有权、trait、错误 | 不必刷完全部 20 章再开 RFR |
| **RFR 通读** | `02-RFR/Chapter-01`～`13` 按章 README 顺序 | 不必为每章搭大项目 |
| **ER 通读** | `01-ER` 35 Item；有 `demo/` 可 `cargo run` | 不必与 RFR 逐章 rigid 对齐 |
| **Nomicon** | `03-Rust_Nomicon/Stable` 为主 | Nightly 分支按需 |
| **04 实战** | 按 hub **01 → 02 → 03** 顺序 | 不宜跳过 01 直接读 03 异步网络 |
| **LLVM 对照** | 改 `05_Compilers-and-LLVM-Learning/04_Learn-LLVM-17/src/lib.rs` 导出 `.ll` | 不必啃 LLVM C++ 源码 |
