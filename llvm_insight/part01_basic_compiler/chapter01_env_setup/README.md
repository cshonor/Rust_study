## 本目录用途

- 记录你本机 **LLVM/Clang 版本**、PATH、以及「能跑 `opt`/`llc` 或读 `.ll`」即可。  
- **不要求** 从源码编译 LLVM；若做过，可仅在 `notes/` 写两行备忘。
