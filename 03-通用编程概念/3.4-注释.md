# 3.4 注释

编译器会忽略注释，但能帮助阅读源码的人理解代码。

---

## 行注释

以 `//` 开头，到行尾结束：

```rust
// Hello, world.
```

---

## 多行注释

每行都以 `//` 开头：

```rust
// 我们在这里处理一些复杂事情，需要足够长的解释，使用
// 多行注释可做到这点。我们希望这个注释将解释接下来
// 要实现的内容。
```

---

## 行尾注释

注释放到代码同一行后面：

```rust
fn main() {
    let lucky_number = 7; // I'm feeling lucky today
}
```

---

## 推荐写法

把注释放到**上一行**，单独成行，更清晰：

```rust
fn main() {
    // I'm feeling lucky today
    let lucky_number = 7;
}
```

---

## 文档注释

Rust 还支持**文档注释** `///`，用于生成 API 文档，在第 14 章会详细说明。

```rust
/// 计算两数之和
fn add(a: i32, b: i32) -> i32 {
    a + b
}
```
