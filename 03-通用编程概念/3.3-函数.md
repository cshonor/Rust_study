# 3.3 函数

Rust 用 `fn` 声明函数。函数名和变量名使用**蛇形命名法**（snake_case）：全小写，单词间用下划线。

---

## 函数定义与调用

```rust
fn main() {
    println!("Hello, world!");
    another_function();
}

fn another_function() {
    println!("Another function.");
}
```

- 定义顺序无所谓，只要在作用域内定义即可
- 调用：`函数名()` 

---

## 参数 (Parameters)

函数签名中**必须**为每个参数标注类型。

```rust
fn another_function(x: i32) {
    println!("The value of x is: {}", x);
}

// 多参数用逗号分隔
fn print_labeled_measurement(value: i32, unit_label: char) {
    println!("The measurement is: {}{}", value, unit_label);
}
```

- 形参 (parameter)：函数定义中的变量
- 实参 (argument)：调用时传入的具体值
- Rust 要求在定义处显式标注类型，方便推导

---

## 语句与表达式

| 类型 | 说明 | 返回值 |
|------|------|--------|
| **语句** (statement) | 执行操作，不返回值 | 无 |
| **表达式** (expression) | 计算并产生值 | 有 |

- `let y = 6;` 是语句，不能写 `let x = (let y = 6);` 赋值给别的变量
- `5 + 6`、函数调用、宏调用都是表达式
- 代码块 `{}` 也是表达式，其值为**最后一个表达式**的结果

```rust
let y = {
    let x = 3;
    x + 1   // 注意：末尾无分号！有分号就变成语句，不返回值
};
// y = 4
```

**要点**：表达式末尾**不加分号**；加分号会变成语句，不再返回值。

---

## 带返回值的函数

- 用 `-> 类型` 声明返回类型
- 返回值 = 函数体**最后一个表达式**的值（末尾无分号）
- 也可用 `return` 提前返回

```rust
fn five() -> i32 {
    5   // 无分号，作为返回值
}

fn plus_one(x: i32) -> i32 {
    x + 1   // 无分号
}
```

### 常见错误：末尾加分号

```rust
fn plus_one(x: i32) -> i32 {
    x + 1;   // ❌ 加了分号变成语句，返回 ()
}
// 错误: expected `i32`, found `()`
// 建议: consider removing this semicolon
```

- 语句返回单元类型 `()`
- 与声明的 `-> i32` 不一致，导致类型不匹配

---

## 小结

| 概念 | 要点 |
|------|------|
| 命名 | snake_case |
| 参数 | 必须标注类型 |
| 语句 | 不返回值，如 `let x = 6;` |
| 表达式 | 返回值，末尾无分号 |
| 返回值 | 最后表达式的值，或 `return` |
