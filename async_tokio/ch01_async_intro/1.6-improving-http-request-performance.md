# 1.6 Improving HTTP Request Performance（异步提升 HTTP 性能）

> Demo：[1.6-improving-http-request-performance/1.6-improving-http-request-performance-demo.rs](./1.6-improving-http-request-performance/1.6-improving-http-request-performance-demo.rs)

## 核心梳理

- **`reqwest`** 向延迟 API 发请求时，异步可在**网络空窗期**执行其它逻辑（如模拟 DB 查最后登录时间）。
- **`tokio::join!`**：四路请求并发，墙钟时间 ≈ **max(RTT)**，而非四次相加（书中常给 ~4.7× 量级，以实测为准）。
- **乱序完成**：各请求完成先后不确定；业务若依赖顺序需额外队列/序号。

## 示例要点

```rust
let (_, _, _, _) = tokio::join!(
    reqwest::get(url),
    reqwest::get(url),
    reqwest::get(url),
    reqwest::get(url),
);
```

需外网；教学 URL 可用 `jsonplaceholder.typicode.com`。
