# 10.3 Building Our Server（构建服务器）

## Accepting Requests

- 主线程 **listen + accept**，作路由器；
- **`spawn_worker!`** 宏将连接分给工作线程池；
- **`AtomicBool`** 控制 worker **park / unpark**，空闲时挂起，省 CPU。

## Handling Requests

- 各线程自有 **Executor**；
- 异步读字节 → **反序列化** → **async Sleep**（模拟业务延迟）→ 写回响应。

完整多线程服务器按书在 workspace `server` crate 实现。
