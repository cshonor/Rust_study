# 10.2 Building Our std Async Runtime（构建 std 异步运行时）

本章最硬核部分：五大组件（书中）：

## Waker

- **`RawWaker` + `RawWakerVTable`**：`clone` / `wake` / `wake_by_ref` / `drop`；
- **`Waker::from_raw`** 构造；`wake` 将任务挂回就绪队列。

## Executor

- **`VecDeque`** 任务队列；
- 将任意输出类型的 Future 擦除为 **`Pin<Box<dyn Future<Output = ()>>>`**，真实结果经 **std channel** 回传，避免泛型爆炸。

## 异步 TcpStream I/O

- **`set_nonblocking(true)`**；
- `WouldBlock` → **`Poll::Pending`** + 注册 Waker。

## Sleep

- 基于 **`Instant::now()`** 的非阻塞休眠 Future：未到点则 Pending。
