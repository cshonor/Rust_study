# 3.1 Building Our Own Async Queue（构建异步队列）

> Demo：[3.1-building-async-queue/3.1-building-async-queue-spawn-demo.rs](./3.1-building-async-queue/3.1-building-async-queue-spawn-demo.rs)

## 核心梳理

- **目标**：将第 2 章 Task/Future 理论落地为可运行的**轻量级运行时骨架**。
- **基础库**：
  - **`async-task`**：把 Future 包装成 **Runnable** + **Task** 句柄；
  - **`futures-lite`**：`block_on` 等轻量 Future 工具；
  - **`flume`**：MPMC 通道，**Receiver 可 clone**，适合多 worker 抢任务。
- **`spawn_task`**：生成 Runnable → 送入队列；后台线程循环 `recv` → `run()` 推进任务。

## 干洗店类比（书中）

| 概念 | 类比 |
|------|------|
| Future | 大衣里的清洗说明 |
| Runnable | 套袋贴编号、可上架的大衣 |
| Task | 取衣凭证，最终取回结果 |

## 本仓库

最小 demo 用 `flume` 模拟「入队 / worker 取出」；完整 `async-task` + `poll` 循环请按书在独立 crate 抄写。
