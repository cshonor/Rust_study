# 7.5 Summary（总结）

## 核心梳理

- **`Builder`**：掌控工作线程、阻塞池、队列与栈，必要时多运行时模拟优先级。
- **本地池**：控制任务**在哪条线程**跑，承载 `!Send` 与线程亲和优化。
- **`UnsafeCell`**：仅在 pinned + 不 `await` 前提下无锁访问线程数据。
- **优雅停机**：信号 → 收尾任务 → 汇总持久化，再退出。

## 工程忠告

本地池**不做窃取**，单 worker 过载需架构规避；`spawn_blocking` 仍要配合 **`max_blocking_threads`** 限流。
