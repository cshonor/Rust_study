# 4.8 Polling Sockets in Futures（在 Future 中轮询 Socket）

> Demo：[4.8-polling-sockets-in-futures/4.8-polling-sockets-listener-demo.rs](./4.8-polling-sockets-in-futures/4.8-polling-sockets-listener-demo.rs)

## 核心梳理

- **`Token`**：在 `Poll` 中区分成千上万个 socket 的就绪事件。
- **流程**：`register(socket, token, Interest::READABLE)` → `poll(&mut events)` → 按 token 分发。
- **ServerFuture**：手写 Future，在 `poll` 里 `accept` / `read`，遇 `WouldBlock` 返回 `Pending` 并登记 Waker。

## 本仓库 demo

绑定 `127.0.0.1:0`，一次 `poll` 打印事件数；可用 `nc` 触发可读事件。
