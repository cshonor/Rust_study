# 4.7 Introducing mio（mio 简介）

## 核心梳理

- **`mio`**：跨平台**非阻塞 I/O** 原语（epoll / kqueue / IOCP）；Tokio 底层基石之一。
- **何时用**：需对 I/O 路径**极致控制**、自建协议栈、教育性理解「事件从哪来」时。
- **非**「Metal I/O」正式全称；crate 名即 **`mio`**。

## 关键依赖（本章）

| crate | 作用 |
|-------|------|
| `mio` | Poll + Token + Socket |
| `hyper` | HTTP（需 Executor/IO 桥接） |
| `async-native-tls` | 异步 TLS（可选） |
