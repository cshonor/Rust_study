# 4.6 Connecting and Running Our Client（连接并运行客户端）

## 核心梳理

- **实战组合**：自定义 **Connector** + **Executor** + `hyper::Client`。
- **流程**：在自建运行时 `block_on` / `spawn` 中发起对 rust-lang.org 等站点的 **HTTPS GET**，打印 HTML 片段。
- **验证点**：证明 trait 桥接成功——`hyper` 跑在你的引擎上，而非 Tokio 默认运行时。

## 本仓库

完整 hyper+TLS 链路按书在独立 crate 实现；主工程仍以 Tokio 模块作对照阅读。
