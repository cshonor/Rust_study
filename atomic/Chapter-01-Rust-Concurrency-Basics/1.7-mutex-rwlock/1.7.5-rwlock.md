**多读 OR 一写**：

| Guard | 含义 |
|-------|------|
| **`ReadGuard`** | 多个线程可同时读 |
| **`WriteGuard`** | 独占写；与任何读/写互斥 |

适用**读多写少**；注意 **写者饥饿（Writer Starvation）**：读者持续占锁时写者可能长期等待。

```rust
use std::sync::RwLock;

let lock = RwLock::new(5);
{
    let r1 = lock.read().unwrap();
    let r2 = lock.read().unwrap();
    println!("{} {}", *r1, *r2);
}
{
    let mut w = lock.write().unwrap();
    *w += 1;
}
```
