## 1.6.3 `Send` 精读：所有权跨线程

> 书 §1.6 · 索引：[1.6-send-sync.md](./1.6-send-sync.md) · 前：[1.6.2 类型速查](./1.6.2-common-types-table.md) · 后：[1.6.4 Sync](./1.6.4-sync-trait.md)

---

```rust
use std::thread;

let text = String::from("hello");
let h = thread::spawn(move || {
    println!("{text}"); // 子线程拥有 text
});
h.join().unwrap();
// println!("{text}"); // 编译错误：已 move
```

**未实现 `Send` 的反例**：`Rc<i32>` 不能 `move` 进 `spawn`（计数器非原子，跨线程 move 会破坏 `Rc` 不变量）。
