## 1.3.4 克隆 + 变量遮蔽

> 书 §1.3 · 索引：[1.3-shared-ownership](./1.3-shared-ownership.md) · 前：[1.3.3 static/leak](./1.3.3-static-and-leak.md) · 后：[1.3.5 对照与易错](./1.3.5-scope-arc-static-compare.md)

---

多个 `spawn` 都要 `move` 同一份 `Arc`，若每次 `let a1 = a.clone()` 会很快变成命名地狱。**变量遮蔽**写法：

```rust
use std::sync::Arc;
use std::thread;

let a = Arc::new(vec![1, 2, 3]);
let mut handles = vec![];

for _ in 0..3 {
    let a = Arc::clone(&a); // 遮蔽外层 a，闭包里 move 的是 clone
    handles.push(thread::spawn(move || {
        println!("{:?}", a);
    }));
}

for h in handles {
    h.join().unwrap();
}
// 外层 a 仍可用
```

**要点**：`Arc::clone` 只增引用计数，**不复制 `Vec` 本体**；每个线程 `move` 走的是栈上的 `Arc` 句柄。
