## 整体脉络

**本章核心**：全书**项目与灵感** — 进阶同步模式、**无锁**入门，以及 **设计 / 优化 / 排错 / 面试** 收尾。

| 第 9 章（刚学完） | 第 10 章（本章） |
|-------------------|------------------|
| **手写有锁** Mutex / Condvar / RwLock | **无锁** RCU、无锁栈、SeqLock |
| 三态 + `atomic_wait` | **CAS 循环** + **内存回收**难题 |
| 工程上用 `std` | 生产用 **`crossbeam` / `parking_lot`** 等库 |

**两条主线**：

1. **10.1 → 10.6**：五种**进阶模式**（信号量 → RCU → 无锁链表 → Parking lot → 阻塞锁 → SeqLock）；
2. **10.7**：全书**自顶向下设计**、性能实践、Miri 排错、面试浓缩。

**一句话**：**第 9 章手写锁；第 10 章模式 + 无锁 + 全书落地。**

## 模式速查（一句话）

| 模式 | 一句话 |
|------|--------|
| **Semaphore** | 计数同步；二元≈Mutex 积木 |
| **RCU** | 读指针→拷→CAS→延迟回收 |
| **无锁栈** | 头插 CAS 易；删+回收难；防 ABA |
| **Parking lot** | 等待队列全局化，锁对象极小 |
| **SeqLock** | seq 奇=写中；读者前后 seq 相等且偶则快照有效 |
