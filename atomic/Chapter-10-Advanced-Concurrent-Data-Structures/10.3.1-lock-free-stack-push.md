## 10.3.1 无锁栈 — 头插 CAS

## 结构

```text
Node { data, next: AtomicPtr<Node> }
head: AtomicPtr<Node>
```

---

## Push 算法（CAS 循环）

1. 分配 `new_node`，`next = current_head`。  
2. `head.compare_exchange_weak(current_head, new_ptr, Release, Relaxed)`。  
3. 失败则更新 `current_head`，**重试**。

```rust
// 伪代码
loop {
    let cur = head.load(Acquire);
    new_node.next.store(cur, Relaxed);
    if head.compare_exchange_weak(cur, new_ptr, Release, Relaxed).is_ok() {
        break;
    }
}
```

→ [CAS 专题](../Chapter-02-Atomics/CAS与Fetch-Modify专题.md)：**loop 内用 `compare_exchange_weak`**

Demo：[10.3-lock-free-stack-push-demo.rs](./10.3-lock-free-linked-list/10.3-lock-free-stack-push-demo.rs)

---

## 为何「头插」相对简单？

只争用 **`head`** 一个原子变量；**不**涉及中间节点删除与回收。
