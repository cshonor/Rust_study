## 作用域（Lexical Scoping）

C / Java / Rust 等：**内层可遮蔽外层同名**。

| 结构 | 做法 |
|------|------|
| **多层链表** | 作用域链查找 |
| **散列栈** | 进入块 push、退出 pop；带 scope 标记 |
| **单表 + 深度** | 查最近匹配 depth |

**jlox Resolver**（ch11）在 AST 上标注**解析深度 / 槽位** — 符号表思想的变体。

→ [CI ch11 Resolver](../../../01_Crafting-Interpreters/part02_jlox/chapter11_resolving-and-binding/README.md)

## 符号表 vs IR 名字

| | **符号表** | **IR 临时名 / SSA** |
|---|------------|---------------------|
| 管什么 | **源程序实体** | **编译器生成的中间值** |
| 生命周期 | 全编译过程 | 某 IR 层 / 某函数体 |
| 查什么 | 声明、类型、scope | def-use、优化 |
