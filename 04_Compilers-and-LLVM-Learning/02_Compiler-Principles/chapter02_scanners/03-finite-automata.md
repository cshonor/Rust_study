# 第 2 章 · 扫描 · §3 有穷自动机（Finite Automaton, FA）

---

RE 描述模式；**有穷自动机（FA）** 是**识别**这些模式的执行模型。

---

## NFA · 非确定性有穷自动机

| 特点 | 说明 |
|------|------|
| **多路径** | 同一输入字符，可能有多条状态转移 |
| **ε-转换** | 可不读输入就转移状态 |
| **构造** | 便于从 RE **直接构造**（Thompson） |
| **执行** | 难以用代码**高效**直接模拟 |

---

## DFA · 确定性有穷自动机

| 特点 | 说明 |
|------|------|
| **唯一转移** | 任意状态 + 任意输入字符 → **唯一**下一状态 |
| **无 ε** | 标准 DFA 不含 ε-转换 |
| **扫描器** | **实际执行的理想模型** — 每读一字符，查表/跳转一次 |

---

## 对照表

| | **NFA** | **DFA** |
|---|---------|---------|
| 确定性 | 否 | 是 |
| 从 RE 构造 | ✅ 容易（Thompson） | 需经子集构造 |
| 扫描器直接用 | ❌ 低效 | ✅ 理想 |
| 状态数 | 通常较少 | 可能指数膨胀（再最小化） |

---

```text
RE  ──Thompson──►  NFA  ──子集构造──►  DFA  ──Hopcroft──►  最小化 DFA
```
