## 超局部值编号（Superlocal VN）

| 范围 | **沿单一控制流路径**上的多个基本块 |
|------|-----------------------------------|
| 思路 | 把路径当作「加长基本块」做 VN |
| 限制 | 不处理路径分叉后再汇合的复杂情况 |

→ [§1 Superlocal](./01-scope-of-optimization.md)

## 基于支配者的值编号（Dominator-based VN）

| 概念 | 说明 |
|------|------|
| **支配（Dominate）** | 节点 A 支配 B ⇔ 到 B 的每条路径都经 A |
| **VN + 支配树** | 在支配边界内传播值编号；汇合点用 φ（SSA）或保守合并 |

使编译器在**复杂 CFG** 中仍追踪「哪些值相同」。

→ [ch5 SSA φ](../chapter05_ir/05-ssa-form.md) · ch10 深化
