## 为什么买这本

- **真编译器**：实现 C♭（C 子集，含指针）→ **ELF 可执行文件**，不是玩具语法。
- **全链路**：编译器、汇编器、链接器、硬件与运行时环境一并讲清——比只啃龙书前端更「落地」。
- **承接 CI**：读完 [01 Crafting Interpreters](../01_Crafting-Interpreters/README.md) 在线版后，用纸质书系统做一遍「能跑的二进制」。
