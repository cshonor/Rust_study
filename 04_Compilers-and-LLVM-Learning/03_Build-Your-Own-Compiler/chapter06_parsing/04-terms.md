## 项（term）的角色

**项** = 二元运算 **一侧** 的操作数 — 比 `expr` 层更 **基础** 的语法单位。

表达式塔 **最底** 通过 **项** 规则接到 **原子** 符号。

```text
expr1  …  binary op …
              ↓
         unary / postfix / primary
```

---

## 前置：`unary`

**一元（前置）运算符** 前缀于操作数：

| 运算符（概念） | 含义 |
|----------------|------|
| `++` `--` | 前置自增/减 |
| `*` | 解引用 |
| `&` | 取地址 |
| `+` `-` `!` `~` | 一元算术/逻辑/按位 |

```text
unary():  ( "++" | "--" | "*" | "&" | … ) unary()
       |  postfix()
```

**递归** `unary()` — 多级前缀 `**p` 等。

---

## 后置：`postfix`

**后缀** 运算 — 紧跟在 **primary** 之后链式出现：

| 形式 | 示例 |
|------|------|
| 后置 `++` `--` | `i++` |
| **下标** | `a[i]` |
| **成员** | `s.memb` |
| **函数调用** | `f(a,b)` |

```text
postfix():  primary() ( "++" | "--" | "[" expr() "]" | "." ID | "(" args ")" )*
```

**左结合** 后缀链：`f()[0].x` — 用 `*` 重复 postfix 片段。

---

## 末端：`primary`

语法树 **最叶** — 字面量、名、括号表达式：

| 种类 | 示例 |
|------|------|
| **整数字面量** | `42` |
| **字符/字符串** | `'A'` `"hi"` |
| **标识符** | 变量名 |
| **括号** | `( expr )` — **强制改变优先级** |

```text
primary():  <INTEGER> | <CHAR> | <STRING> | <IDENTIFIER> | "(" expr() ")"
```

→ ch4 **TOKEN** 种类在此 **消费**。

---

## 四层关系（复习）

```text
compilation_unit … stmt … expr … expr10…expr1
                                      ↓
                            unary → postfix → primary
```

**第1部分收束**：`.jj` 从文件顶到 **primary** 可 **parse 合法 C♭**；ch7 起 **action 填 AST**。
