## 主要目的

排除 **语义非法的类型定义** — 语法可写但类型系统不允许。

| 错误类 | 示例/说明 |
|--------|-----------|
| **`void` 成员** | `void` 不能作为 **数组元素 / 结构体·联合体成员** |
| **成员重复** | 同一 struct/union **同名成员** |
| **循环定义** | struct 间接 **自引用** 未成指针 — 无限大小 |

→ [ch8 struct/typedef](../chapter08_build-ast/03-decl-ast.md) 定义节点在此 **验合法性**。

---

## 循环定义 · 有向图 + DFS

**最棘手** — A 含 B，B 含 A（或非指针自嵌套）→ **类型大小无法确定**。

**建模**：

```text
节点 = Type（或类型定义）
边   = 「该类型 **引用** 另一类型」（成员类型）
```

**算法**：**有向图 DFS** — 节点状态：

| 状态 | 含义 |
|------|------|
| 未访问 | 尚未进入 |
| **查找中** | 当前 DFS 栈上 |
| 已完成 | 子图已检查 |

```text
DFS 中若遇到「查找中」节点 → 存在 **环** → 循环定义错误
```

**合法自引用**：`struct Node { struct Node* next; }` — 边是 **指针**，不形成 **按值嵌套** 环。
