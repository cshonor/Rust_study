Lox 内置类型**精简**：

| 类型 | 说明 |
|------|------|
| **Boolean** | `true` / `false` |
| **Number** | 浮点（书中实现为 double） |
| **String** | 字符串字面量 |
| **Nil** | 「无值」，类似 `null` |

无数组、无字典（标准库也极简）——聚焦语言核心。
