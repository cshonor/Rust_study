### 为何需要 CFG

- **正则表达式 / 扫描器**只能处理**线性**、**不嵌套**的结构。
- 表达式如 `1 + (2 * 3)` 需要**任意嵌套** → 超出正则能力 → 交给 **Parser** + **CFG**。

### 文法规则与 BNF

- **上下文无关文法（CFG）**：用产生式描述「非终结符 → 符号串」。
- **巴科斯范式（BNF）**：形式化书写语法规则的常用记法。

### 增强符号（语法糖）

| 符号 | 含义 |
|------|------|
| `\|` | 分支（或） |
| `*` | 重复 0 次或多次 |
| `+` | 重复 1 次或多次 |
| `?` | 可选（0 或 1 次） |

用这些符号精确定义 **Lox 表达式**的语法蓝图（与 ch3 语义对应，ch6 按此实现 parser）。
