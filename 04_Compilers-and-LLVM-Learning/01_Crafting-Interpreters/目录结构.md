# Crafting Interpreters — 目录结构

> **约定**：与 [`01-ER/目录结构.md`](../../01-ER/目录结构.md) 相同——**一章 = 一个目录**；笔记与 `README.md` **同级**，`README` 只做导航 hub。

## 布局

```text
01_Crafting-Interpreters/
├── CI-本书目录.md          # 可选别名：本书目录.md
├── 本书目录.md             # 全书 30 章 + 附录索引
├── 目录结构.md             # 本文件
├── README.md               # 丛书入口
├── cover.png
├── part01_welcome/         # Part I · 第 1～3 章
│   └── chapter01_introduction/
│       ├── README.md       # hub：状态、一句话、专项笔记链接
│       └── 01-welcome-introduction.md
├── part02_jlox/            # Part II · jlox · 第 4～13 章
├── part03_clox/            # Part III · clox · 第 14～30 章
├── backmatter/             # 附录 I / II
└── scripts/
    ├── flatten-chapter-notes.py
    └── update-chapter-readmes.py
```

## 命名

| 路径 | 含义 |
|------|------|
| `chapterNN_<slug>/README.md` | **导航 hub**：状态、在线链接、一句话、专项笔记索引 |
| `chapterNN_<slug>/NN-<slug>.md` | **章笔记正文**（与 README 同级，原 `notes/` 已废弃） |
| `chapterNN_<slug>/cheat-sheet.md` | 可选：速记（可从正文「本章速记」拆出） |
| `chapterNN_<slug>/demo/` | 可选：jlox/clox 代码片段或对照实验 |

## 与 01-ER 对照

| ER | Crafting Interpreters |
|----|------------------------|
| `Chapter-01-Types/` | `part01_welcome/`、`part02_jlox/`、`part03_clox/` |
| `Item-NN-slug/` | `chapterNN_<slug>/` |
| `Item-NN-slug/*.md` | `chapterNN_<slug>/NN-*.md` |
| `ER-本书目录.md` | `本书目录.md` |

## 新增章节笔记时

1. 在 `chapterNN_<slug>/` 下写 **`NN-*.md`**（正文），勿再建 `notes/` 子目录。
2. 更新同级 **`README.md`** hub（或运行 `scripts/update-chapter-readmes.py`）。
3. 在 [`本书目录.md`](./本书目录.md) 表格中链到 **`./part0X/.../NN-*.md`**。

历史笔记曾放在 `notes/` 子目录；已用 `scripts/flatten-chapter-notes.py` 扁平化，交叉链接中的 `/notes/` 已批量修正。
