## 第十章模式（一句话）

| 模式 | 一句话 |
|------|--------|
| **Semaphore** | 计数同步；二元≈Mutex 积木 |
| **RCU** | 读多写少：读指针→拷→CAS→延迟回收 |
| **无锁链表** | 头插 CAS 易；删+回收难；防 ABA |
| **Parking lot** | 等待队列进全局表，锁对象极小 |
| **SeqLock** | seq 奇数=写中；读者前后 seq 相等且偶数则快照有效 |

## 无锁要点（摘要）

- **AtomicPtr** 换版本；**loop** + **`compare_exchange_weak`**。  
- 摘下节点**不能立刻 drop** → epoch / hazard / 成熟库。  
- 高竞争 CAS 可能输给 Mutex。

---

## 全书落地三条（摘要）

1. **Send/Sync + 借用** — 编译期防竞态。  
2. **精简状态机 + futex** — 无竞争零 syscall。  
3. **RwLock / RCU / SeqLock** — 按读写比例选型。
