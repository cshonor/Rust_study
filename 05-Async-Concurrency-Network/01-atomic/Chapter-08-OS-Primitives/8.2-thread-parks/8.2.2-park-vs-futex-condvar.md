## 对照表

| | **`thread::park`** | **futex WAIT/WAKE** | **`Condvar::wait`** |
|---|-------------------|---------------------|---------------------|
| **抽象层级** | Rust 线程 API | Linux 内核原语 | Rust 同步原语（封装 OS cond） |
| **绑定条件** | 无；自己管状态 | 绑在用户态**原子地址** + 期望值 | 绑 **Mutex + 条件谓词** |
| **释放锁** | 不涉及 | 不涉及（Mutex 逻辑另管） | wait 时**自动释放 Mutex** |
| **丢 notify** | unpark 可先于 park | WAIT 前须检查原子值 | wait/notify 成对设计 |
| **典型场景** | 简单挂起/恢复、线程池 worker | Mutex/Condvar **底层** | 生产者-消费者、任务队列 |
