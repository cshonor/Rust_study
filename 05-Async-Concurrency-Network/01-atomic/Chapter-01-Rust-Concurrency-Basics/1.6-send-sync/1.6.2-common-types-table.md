## 1.6.2 常见类型速查

> 书 §1.6 · 索引：[1.6-send-sync.md](./1.6-send-sync.md) · 前：[1.6.1 定义](./1.6.1-send-sync-motivation.md) · 后：[1.6.3 Send](./1.6.3-send-trait.md)

---

| 类型 | Send | Sync | 记忆 |
|------|------|------|------|
| `i32`、`bool`、`String` | ✓ | ✓ | 基础类型 / 拥有堆数据 |
| `Rc<T>` | ✗ | ✗ | 引用计数非 atomic |
| `Cell<T>`（`T: Send`） | ✓ | ✗ | 可 move；共享则竞态 |
| `RefCell<T>` | ✓* | ✗ | 运行时借用非线程安全 |
| `Arc<T>` | 看 `T` | 看 `T` | `T: Send + Sync` 时通常两者皆可 |
| `Mutex<T>` | 看 `T` | 看 `T` | 锁保护下的 `T` |
| `*const T` / `*mut T` | ✗ | ✗ | 裸指针无自动保证 |

\* `RefCell` 本身可 move 到子线程，但**不能**在多线程间共享 `&RefCell`。
