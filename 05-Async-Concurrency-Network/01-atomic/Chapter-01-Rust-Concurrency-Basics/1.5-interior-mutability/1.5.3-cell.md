## 1.5.3 `Cell<T>` 精读

> 书 §1.5 · 索引：[1.5-interior-mutability.md](./1.5-interior-mutability.md) · 前：[1.5.2 类型一览](./1.5.2-interior-mutability-types.md) · 后：[1.5.4 RefCell](./1.5.4-refcell.md)

---

**适用**：`T: Copy`（`i32`、`bool` 等）。通过 **`&Cell<T>`** 调用 `set` / `get`，**不**对外暴露 `&mut T`。

```rust
use std::cell::Cell;

let cell = Cell::new(5);
let r: &Cell<i32> = &cell;
r.set(10);
println!("{}", r.get()); // 10
```

### 典型模式：`&self` 里改字段

```rust
use std::cell::Cell;

struct Counter {
    count: Cell<u32>,
}

impl Counter {
    fn increment(&self) {
        self.count.set(self.count.get() + 1);
    }
}

let c = Counter { count: Cell::new(0) };
let r1 = &c;
let r2 = &c; // 多个 &Counter 可共存
r1.increment();
r2.increment();
```

### `take`

`take()` 拿走内部值，原位填 **`T::default()`**（`i32` 为 0）。
