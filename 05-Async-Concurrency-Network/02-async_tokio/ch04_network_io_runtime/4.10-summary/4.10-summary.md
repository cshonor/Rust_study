# 4.10 Summary（总结）

## 核心梳理

- **Trait 是桥梁**：自定义 **Executor / Connector / AsyncRead / AsyncWrite** 让 `hyper` 等库跑在你的运行时上。
- **`mio`** 赋予直接 **Poll Socket** 的能力，可自建乃至理解 Web 框架底层协议栈。
- **生产建议**：日常仍推荐 **Tokio + hyper/axum**；本章偏底层教育与极致控制场景。

## 易错点回顾

| 点 | 说明 |
|----|------|
| 全书难度 | 可先读完后续章再回看 |
| hyper 版本 | Executor/IO trait 随版本变化 |
| 阻塞 read/write | 在 `poll` 路径禁止 |
