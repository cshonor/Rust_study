# 9.1 Building an Isolated Module（构建隔离模块）

## 问题

如何把异步安全集成进**庞大同步代码库**，又避免高风险全面重写？

## 同步入口 + 内部 Tokio

- 对外：**同步 Facade**（阻塞式 API）；
- 对内：维护 **任务 ID → `JoinHandle`** 的 `HashMap`，后台 `Runtime` `spawn`；
- 同步方只传 **ID**，阻塞 `join`/轮询映射表取结果。

## 实现要点

| 点 | 说明 |
|----|------|
| 侵入面 | 外部不见 `async`，渐进接入 |
| 代价 | 阻塞线程、取消/错误语义需单独设计 |
| 通道 | 可与 `mpsc` 配合上报进度 |

## 配套 Demo

`9.1-building-isolated-module/`：标准库占位；完整 Facade + Runtime 按书在独立 crate。
