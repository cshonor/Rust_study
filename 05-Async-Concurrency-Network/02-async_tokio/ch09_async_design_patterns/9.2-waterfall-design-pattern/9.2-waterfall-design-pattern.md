# 9.2 Waterfall Design Pattern（瀑布 / 责任链）

## 核心

又称 **Chain of Responsibility**：多个异步步骤用 **`?`** 串联，**上一步输出 = 下一步输入**。

- 流程清晰、可预测；
- 单一组件可在不同流水线中复用。

## Rust 写法

```rust
async fn pipeline() -> Result<Response, Error> {
    let a = step_one().await?;
    let b = step_two(a).await?;
    step_three(b).await
}
```

失败时 `?` 提前返回，避免嵌套 `match` 地狱。
