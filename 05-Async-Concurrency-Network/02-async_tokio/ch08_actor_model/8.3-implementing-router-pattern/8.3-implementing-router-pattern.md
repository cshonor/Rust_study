# 8.3 Implementing the Router Pattern（实现路由模式）

## 问题

深层调用链中层层传递 **`mpsc::Sender`** 导致**强耦合**、类型爆炸。

## 路由 Actor

- 全局 **`ROUTER_SENDER`**（常配合 `OnceLock` / `LazyLock`）作为**统一入口**；
- 消息为 **`enum`**：`Set` / `Get` / `Delete` 等；
- 路由 Actor 解析后转发给后端 **KV 存储 Actor**。

## 好处

调用方只依赖路由 API；后端 Actor 可替换、扩容或分片而不改客户端。
