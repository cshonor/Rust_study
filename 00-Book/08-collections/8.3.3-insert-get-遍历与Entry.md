## 一、`insert` 与所有权

```rust
let field_name = String::from("Favorite color");
let field_value = String::from("Blue");
let mut map = HashMap::new();
map.insert(field_name, field_value);
// field_name、field_value 移入 map → 后续不能用，编译报错
```

| 类型 | `insert` 行为 | 原变量 |
|------|---------------|--------|
| **无 `Copy`**（`String`、自定义 struct） | **move** 进 Map | 失效 |
| **实现 `Copy`**（`i32`、`bool`、`u8`） | **拷贝** 进 Map | 仍可用 |

```rust
let num = 10;
map.insert("A", num);
println!("{}", num); // ✅ i32 拷贝
```

> 想保留原数据：`insert(&field_name, &field_value)`，Map 存**借用**（生命周期须合规）。

## 二、`get` 取值：`Option<&V>`

### 三种写法（对标 Vec `[]` / `get`）

| 写法 | 返回 | 无 key 时 | 场景 |
|------|------|-----------|------|
| **`map["key"]`** | `&V` | **panic** | key 一定存在 |
| **`map.get(&key)`** | `Option<&V>` | `None` | key 不确定 |
| **`map.get_mut(&key)`** | `Option<&mut V>` | `None` | 修改 value |

```rust
let team_name = String::from("Blue");
let score = scores.get(&team_name); // Option<&i32>
```

1. **`get(&key)`** 传键的引用，**不消耗** key 所有权  
2. 存在 → `Some(&V)` 借用 Map 内数据；不存在 → `None`，**不 panic**  
3. **`contains_key(&key)`** 只判存在，不取值  

### 常用链

```rust
let n = scores.get(&team_name).copied().unwrap_or(0);
// .copied()：Option<&i32> → Option<i32>（V: Copy）
// .unwrap_or(0)：取不到给默认值
```

```rust
if let Some(v) = scores.get_mut("Blue") {
    *v += 5;
}
```

## 三、遍历

```rust
for (key, value) in &scores {
    println!("{key}: {value}");
}
```

| 写法 | 元素类型 | Map 之后 |
|------|----------|----------|
| **`&scores` / `scores.iter()`** | `(&K, &V)` | 仍可用 |
| **`for (k,v) in scores`**（无 `&`） | `(K, V)` | **move 整个 Map**，遍历后失效 |
| **`scores.iter_mut()`** | `(&K, &mut V)` | 可改 value |

```rust
for k in map.keys() { /* … */ }
for v in map.values() { /* … */ }
for (_, v) in scores.iter_mut() { *v += 1; }
```

- HashMap **无序**，打印顺序随机

## 四、Entry：四种更新策略

`map.entry(key)` **只哈希查找一次** → `Occupied`（有键）/ `Vacant`（无键）。

### 1. `insert` — 无脑覆盖

```rust
scores.insert(String::from("Blue"), 10);
scores.insert(String::from("Blue"), 25); // 覆盖 → 25
// 返回 Option<V>：Some(旧值) 或 None
```

### 2. `entry` + `or_insert` — 缺则插，有则不动

```rust
scores.entry(String::from("Yellow")).or_insert(50);
scores.entry(String::from("Blue")).or_insert(50); // 已有 → 不改
```

`or_insert(x)` → 始终返回 **`&mut V`**。

### 3. 词频统计 — 基于原值累加

```rust
let text = "apple pear apple banana apple";
let mut map = HashMap::new();
for word in text.split_whitespace() {
    let count = map.entry(word).or_insert(0);
    *count += 1;
}
```

### 4. 链式 `and_modify`（拓展）

```rust
map.entry("k").and_modify(|v| *v *= 2).or_insert(10);
// 存在 → 翻倍；不存在 → 插 10
```

| 方法 | 作用 |
|------|------|
| **`or_insert_with(\|\| …)`** | 不存在时闭包延迟生成值 |
| **`and_modify(\|v\| …)`** | 仅当键已存在时修改 |

---

## 总结表

| 操作 | 写法 | 特点 |
|------|------|------|
| 全覆盖 | `insert(k, v)` | 覆盖旧值 |
| 缺则插 | `entry(k).or_insert(v)` | 存在不动 |
| 基于原值改 | `*entry(k).or_insert(0) += 1` | 词频 |
| 安全查 | `get(&k)` | `Option<&V>` |
