## 核心

`String` / `&str` 内容都是 **UTF-8**；`.len()`、切片按**字节**，不是「字符个数」。

---

## 新建与更新

```rust
let mut s = String::new();
let s = String::from("initial contents");
s.push_str("bar");
s.push('l');

let s1 = String::from("Hello, ");
let s2 = String::from("world!");
let s3 = s1 + &s2; // s1 被 move
```

| 操作 | 说明 |
|------|------|
| `push_str(&str)` | 追加 UTF-8 片段 |
| `push(char)` | 追加单个 Unicode 标量 |
| `s1 + &s2` | `s1` 被 move；`s2` 仍可用 |
| `format!(...)` | 得到 `String`，不 move 参数 |

---

## 为什么不能用 `s[i]`？

```rust
// let h = s[0]; // ❌
```

类似 `Vec<u8>`；「第 i 个」是字节还是 `char` 语义不清，且不能保证 O(1) 随机访问。

---

## 字符串 slice（字节 range）

```rust
let hello = "Здравствуйте";
let s = &hello[0..4]; // "Зд" — 须在 UTF-8 字符边界上
// &hello[0..1]; // ❌ 运行 panic
```

---

## 遍历

```rust
for c in "नमस्ते".chars() { /* Unicode 标量 */ }
for b in "नमस्ते".bytes() { /* 字节 u8 */ }
```

---

## 记忆卡片

| 要点 | 一句话 |
|------|--------|
| `.len()` | **字节数**，不是字符数 |
| 索引 | 无 `s[i]`；用 `.chars()` / 字节 slice |
| slice | range 必须在 UTF-8 **边界**上 |
| 拼接 | `+` move 左操作数；`format!` 更灵活 |
