## 一、四段式对照（`add` 为例）

```rust
fn add(x: i32) -> i32 { x + 1 }           // 普通 fn 对照

let add = |x: i32| -> i32 { x + 1 };      // 1. 完整：参数 + 返回 + {}
let add = |x: i32| { x + 1 };             // 2. 省略 ->，保留参数类型
let add = |x: i32| x + 1;                  // 3. 单表达式，省略 {}
let add = |x| x + 1;                      // 4. 最简：全靠推断

add(5); // 6
```

---

## 二、省略规则（核心考点）

### 1. 参数类型可省

`|x|` 不写 `x: i32` — **第一次调用实参确定类型后永久固定**：

```rust
let add = |x| x + 1;
add(5i32);   // x 锁定 i32
// add(5u64); // ❌ 编译报错
```

→ 完整讲解：[13.1.3 类型推导易错点](./13.1.3-闭包类型推导易错点.md)

### 2. 返回值 `-> T` 可省

| 形式 | 规则 |
|------|------|
| **单表达式** `\|x\| x + 1` | 不用写 `->`，返回自动推导 |
| **多语句 `{}`** | 须显式 `-> T`，否则编译报错 |

```rust
let add = |x: i32| -> i32 {
    let tmp = x + 2;
    tmp - 1
};
```

### 3. 单表达式可省 `{}`

- **一条表达式**：`|x| x + 1`
- **多条语句 / 带分号语句**：必须 `{}`

---

## 三、闭包 vs `fn`

| | **`fn`** | **闭包** |
|--|----------|----------|
| 类型标注 | 签名须全标注 | 可推断 |
| 捕获环境 | ❌ | ✅ |

```rust
let base = 10;
let add = |x| base + x; // 捕获外层 base
println!("{}", add(5)); // 15
```

→ 形参 vs 捕获详解：[13.1.2](./13.1.2-参数与捕获.md)
