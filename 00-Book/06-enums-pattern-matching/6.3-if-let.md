# 6.3 if let 简单控制流

对应 demo 项目：`00-Book/06-enums-pattern-matching/6.3-if-let-demo/`

`if let` 是 `match` 的语法糖：当你只关心**匹配某一个模式**时，用更少的样板代码完成同样的事。

---

## 用 match 只关心一个分支（会显得啰嗦）

```rust
let some_u8_value = Some(0u8);
match some_u8_value {
    Some(3) => println!("three"),
    _ => (),
}
```

为了满足 match 的穷尽性，必须写 `_ => ()`。

---

## 用 if let（更简洁）

```rust
let some_u8_value = Some(0u8);
if let Some(3) = some_u8_value {
    println!("three");
}
```

语法：`if let PATTERN = EXPR { ... }`

权衡：
- **优点**：更短、更少缩进、更少样板代码
- **缺点**：不像 `match` 那样强制穷尽性检查

---

## if let + else

`else` 相当于 `match` 的 `_` 分支。

以硬币为例：对 Quarter 打印州名，否则计数：

```rust
let mut count = 0;
if let Coin::Quarter(state) = coin {
    println!("State quarter from {:?}!", state);
} else {
    count += 1;
}
```

---

## 小结

- **match**：强制穷尽，适合要处理多种情况
- **if let**：只处理一种情况时更简洁
