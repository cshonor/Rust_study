# 3.3 函数

Rust 用 `fn` 声明函数。函数名和变量名使用**蛇形命名法**（snake_case）：全小写，单词间用下划线。

对应 demo 项目：`00-Book/03-common-concepts/3.3-functions-demo/`

---

## 核心结论

1. 函数若要返回**非 `()` 的值**，**必须显式写** `-> 类型`（Rust **不支持**函数返回类型自动推导）。
2. 函数体**最后一行是表达式且不带 `;`** → 该表达式的值就是返回值；**加了 `;`** 变成语句 → 该处等价于 `()`。
3. 只返回 **`()`** 的函数可省略 `-> ()`（如 `fn foo() { ... }`）。

---

## 函数定义与调用

```rust
fn main() {
    println!("Hello, world!");
    another_function();
}

fn another_function() {
    println!("Another function.");
}
```

- 定义顺序无所谓，只要在作用域内定义即可
- 调用：`函数名()` 

---

## 参数 (Parameters)

函数签名中**必须**为每个参数标注类型。

```rust
fn another_function(x: i32) {
    println!("The value of x is: {}", x);
}

// 多参数用逗号分隔
fn print_labeled_measurement(value: i32, unit_label: char) {
    println!("The measurement is: {}{}", value, unit_label);
}
```

- 形参 (parameter)：函数定义中的变量
- 实参 (argument)：调用时传入的具体值
- Rust 要求在定义处显式标注类型，方便推导

## 语句与表达式

| 类型 | 说明 | 返回值 |
|------|------|--------|
| **语句** (statement) | 执行操作，不返回值 | 无 |
| **表达式** (expression) | 计算并产生值 | 有 |

- `let y = 6;` 是语句，不能写 `let x = (let y = 6);` 赋值给别的变量
- `5 + 6`、函数调用、宏调用都是表达式
- 代码块 `{}` 也是表达式，其值为**最后一个表达式**的结果

```rust
let y = {
    let x = 3;
    x + 1   // 注意：末尾无分号！有分号就变成语句，不返回值
};
// y = 4
```

**要点**：表达式末尾**不加分号**；加分号会变成语句，不再返回值。

## 带返回值的函数

### 必须写返回类型（硬性规则）

与**变量**不同，Rust **不会**为函数推断返回类型。要返回 `i32` 等具体类型，签名里必须写 `-> i32`：

```rust
fn add(a: i32, b: i32) -> i32 {
    a + b   // 末尾无分号
}
```

省略返回类型且函数体实际产生非 `()` 的值会编译失败：

```rust
// fn add(a: i32, b: i32) {
//     a + b   // ❌ expected `()`, found integer
// }
```

### 两种返回形式

**1. 末尾无分号 → 返回表达式结果**

```rust
fn one() -> i32 {
    100
}
```

**2. 末尾有分号 → 语句，该位置贡献 `()`**

只打印、无有意义返回值时，返回类型为 **`()`**，通常**省略**不写：

```rust
fn print_msg() {
    println!("hello");
}

// 等价于
fn print_msg() -> () {
    println!("hello");
}
```

若声明了 `-> i32` 却在末尾写 `x + 1;`，会得到 **`expected i32, found ()`**。

### `return` 提前返回

同样须写 `-> 类型`；`return` 后也可跟表达式：

```rust
fn max(a: i32, b: i32) -> i32 {
    if a > b {
        return a;
    }
    b
}
```

### 基础示例

```rust
fn five() -> i32 {
    5
}

fn plus_one(x: i32) -> i32 {
    x + 1
}
```

### 常见错误：末尾加分号

```rust
fn plus_one(x: i32) -> i32 {
    x + 1;   // ❌ 语句，返回 ()
}
// expected `i32`, found `()`
```

## 小结

| 概念 | 要点 |
|------|------|
| 命名 | snake_case |
| 参数 | 必须标注类型 |
| 语句 | 不返回值，如 `let x = 6;` |
| 表达式 | 返回值，末尾无分号 |
| 返回值 | 非 `()` 须写 `-> 类型`；最后表达式无 `;`，或 `return` |
| 无返回值 | 省略 `-> ()`，语句正常加 `;` |
