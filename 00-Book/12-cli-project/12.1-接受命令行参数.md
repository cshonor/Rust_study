# 12.1 接受命令行参数

我们要做一个简化版 `minigrep`，先让程序能接受两个命令行参数：

- **query**：要搜索的字符串
- **filename**：要搜索的文件路径

对应 demo 项目：`00-Book/12-cli-project/12.1-minigrep-args-demo/`

---

## 读取参数：`std::env::args`

`std::env::args()` 返回一个迭代器，常见做法是 `collect()` 成 `Vec<String>`：

```rust
use std::env;

fn main() {
    let args: Vec<String> = env::args().collect();
    println!("{args:?}");
}
```

注意：

- `args[0]` 通常是程序名（例如 `target/debug/minigrep`）
- 我们真正需要的两个参数在 `args[1]` 与 `args[2]`
- 如果参数里包含无效 Unicode，`args()` 会 panic；需要更底层兼容时可用 `args_os()`

---

## 保存参数到变量

```rust
let query = &args[1];
let filename = &args[2];
```

之后就能在程序其它部分使用 `query` 与 `filename`。

---

## 运行 demo（Windows 重点）

Cargo 传递参数给程序时，需要在参数前加 `--`：

```bash
cd 00-Book/12-cli-project/12.1-minigrep-args-demo

# 不带参数
cargo run

# 带参数（注意这个 --）
cargo run -- needle haystack.txt
```
