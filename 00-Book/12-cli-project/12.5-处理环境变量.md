# 12.5 处理环境变量

我们为 `minigrep` 增加一个可选功能：通过环境变量控制是否大小写敏感搜索。

- 默认：**大小写敏感**
- 当设置 `CASE_INSENSITIVE` 环境变量时：**大小写不敏感**

对应 demo 项目：`00-Book/12-cli-project/12.5-minigrep-env-demo/`

---

## `Config` 增加开关：`case_sensitive`

```rust
let case_sensitive = std::env::var("CASE_INSENSITIVE").is_err();
```

- **没设置**环境变量 → `is_err()` 为 true → 大小写敏感
- **设置了**环境变量（任何值）→ `is_err()` 为 false → 大小写不敏感

---

## 运行 demo

```bash
cd 00-Book/12-cli-project/12.5-minigrep-env-demo

# 跑测试
cargo test

# 默认：大小写敏感（只匹配小写 to）
cargo run -- to poem.txt
```

PowerShell 开启大小写不敏感（两步）：

```powershell
$env:CASE_INSENSITIVE=1
cargo run -- to poem.txt
```
