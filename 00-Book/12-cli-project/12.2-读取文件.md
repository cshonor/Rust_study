# 12.2 读取文件

在拿到命令行参数 `query` 和 `filename` 后，下一步是读取 `filename` 指定的文件内容（先把内容打印出来，验证流程正确）。

对应 demo 项目：`00-Book/12-cli-project/12.2-minigrep-readfile-demo/`

---

## 准备测试文件 `poem.txt`

在 demo 项目根目录创建 `poem.txt`，内容是多行文本（后续用于搜索功能）。

---

## 读取文件：`fs::read_to_string`

标准库的 `fs::read_to_string(filename)` 会返回 `Result<String, io::Error>`：

- `Ok(contents)`：文件内容
- `Err(e)`：读取失败（文件不存在、权限不足等）

示例里先用 `expect(...)` 简化错误处理（后续会重构）。

---

## 运行 demo（Windows 记得 `--`）

```bash
cd 00-Book/12-cli-project/12.2-minigrep-readfile-demo
cargo run -- the poem.txt
```
