## 私有性（privacy）：默认私有，`pub` 才公开

| 规则 | 说明 |
|------|------|
| **默认私有** | 模块内项同模块可访问；模块外不能直接访问 |
| **`pub`** | 模块或函数标记公有后，才可通过路径从外部访问 |

```rust
mod front_of_house {
    pub mod hosting {
        pub fn add_to_waitlist() { /* … */ }
        fn seat_at_table() {}  // 私有：仅 hosting 内部
    }
}
```

- `add_to_waitlist`：`pub` → crate 外可 `use` / 路径调用
- `seat_at_table`：无 `pub` → 只有 `hosting` 内部能调

**模式**：对外 `pub fn eat_at_restaurant()`，内部调私有实现，不暴露细节。

## demo：lib 模块树 + main 调公开 API

| 文件 | Crate | 内容 |
|------|-------|------|
| `src/lib.rs` | lib root | 餐厅 `mod front_of_house { … }` + `pub fn eat_at_restaurant()` |
| `src/main.rs` | bin root | `use 包名::eat_at_restaurant()` |

两棵**独立**模块树；main **不**重复声明 `front_of_house`，只调 lib 已 `pub` 的 API。

```bash
cd 00-Book/07-packages-modules/7.2-modules-privacy-demo
cargo run
```
