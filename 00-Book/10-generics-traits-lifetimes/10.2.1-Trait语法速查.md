## 汇总表

| 写法 | 作用 | 核心特点 |
|------|------|----------|
| `fn(x: impl Trait)` | 参数位 impl 语法糖、**静态分发** | 每个 `impl` 独立匿名类型；多参数可传**不同**具体类型，**无法绑定同类型**；代码简洁 |
| `fn<T: Trait>(x: T)` | 标准泛型 trait 约束 | 同 `T` 贯穿全函数，**多参数强制同一实体类型**；可绑定多参数类型关联 |
| `fn(x: impl A + B)` / `<T: A + B>` | 多 trait 叠加约束 | `+` = **同时满足全部** trait，缺一不可 |
| `<T> where T: A + B` | `where` 拆分约束 | 泛型多、约束长时拆分，可读性更好；多泛型分别写约束 |
| `fn() -> impl Trait` | 返回位 impl、类型隐藏 | 函数内部**固定只返回一种具体类型**；分支不能返回不同类型；闭包/迭代器标配 |
| `impl<T: Bound> Struct<T> { … }` | 结构体**条件**方法 | 仅当 `T` 满足 `Bound`，结构体才有这批方法；不满足则**无此方法** |
| `impl<T: Bound> Trait for T` | **Blanket impl**（通配实现） | **批量**给所有满足 `Bound` 的类型自动 impl 目标 trait（如 `Display` → `ToString`） |

---

## 一句话速记

1. **单参数**简写 → `impl Trait`；**多参数要同类型** → 必用泛型 `<T>`。
2. **返回隐藏类型**、返回闭包/迭代器 → `-> impl Trait`；**运行期多种类型** → `Box<dyn Trait>` → [10.2.3](./10.2.3-impl-Trait全解.md) · [17.2](../17-oop/17.2-为使用不同类型的值而设计的trait对象.md)。
3. **Blanket impl** = 批量给一类类型自动实现 trait。

---

## 选型速查

| 需求 | 选 |
|------|-----|
| 单参数、简单约束 | `impl Trait` |
| 两参数必须**同类型** | `<T: Trait>(a: T, b: T)` |
| 两参数**可以不同类型** | `(a: impl Trait, b: impl Trait)` |
| 约束很长 / 多泛型 | `where` |
| **同一容器混多种实现类型** | `Vec<&dyn Trait>` / `Vec<Box<dyn Trait>>` → [10.2.5 §二](./10.2.5-impl与dyn-Trait对比.md#二为何泛型--impl-trait-做不了运行时二选一) |
| **运行时分支返回不同类型** | `Box<dyn Trait>` / `&dyn Trait` → [10.2.5 §二](./10.2.5-impl与dyn-Trait对比.md#二为何泛型--impl-trait-做不了运行时二选一) |

---

## 易错点

1. **`-> impl Trait` 分支返回不同类型** → `Box<dyn Trait>`（运行时二选一同理）→ [10.2.5 §二](./10.2.5-impl与dyn-Trait对比.md#二为何泛型--impl-trait-做不了运行时二选一)
2. **裸 `dyn Trait` 当参数/变量** → `!Sized`，须 **`&dyn Trait`** 或 **`Box<dyn Trait>`** → [10.2.5 §一](./10.2.5-impl与dyn-Trait对比.md#一sized-铁律为何常见-dyn-trait)
3. **trait 定义里 `fn() -> impl Trait`** → 见 [10.2.3 §六](./10.2.3-impl-Trait全解.md)
4. **孤儿规则** — 不能给外部类型 impl 外部 trait
5. **`largest` 大类型** — 优先 `&T`，别强 `Copy`

## 自测小题（口答）

1. `fn f(a: impl Summary, b: impl Summary)` — a、b 能否是 `Tweet` 和 `NewsArticle`？**能**。
2. `fn f<T: Summary>(a: T, b: T)` — 同上？**不能**（必须同类型 T）。
3. `fn g() -> impl Summary { if true { Post {} } else { News {} } }` — 编译？**不过**。
4. `10i32.to_string()` 为何合法？**Blanket impl**：`Display` → `ToString`。
