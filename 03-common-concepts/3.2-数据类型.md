# 3.2 数据类型

Rust 是**静态类型**语言，编译期必须知道所有变量类型。编译器可推导类型，歧义时需显式标注。

对应 demo 项目：`03-common-concepts/3.2-data-types-demo/`

```rust
let guess: u32 = "42".parse().expect("Not a number!");
// 无标注会报错: type annotations needed
```

---

## 标量类型 (Scalar)

标量类型代表单个不可分割的值，Rust 标量类型共 4 种：整型、浮点、布尔、字符。

### 整数类型

| 长度 | 有符号 | 无符号 | 范围 (有符号) | 范围 (无符号) |
|------|--------|--------|---------------|---------------|
| 8 位 | i8 | u8 | -128~127 | 0~255 |
| 16 位 | i16 | u16 | -(2^15)~2^15-1 | 0~2^16-1 |
| 32 位 | i32 | u32 | -(2^31)~2^31-1 | 0~2^32-1 |
| 64 位 | i64 | u64 | -(2^63)~2^63-1 | 0~2^64-1 |
| 128 位 | i128 | u128 | ... | ... |
| arch | isize | usize | 取决于架构 (32/64位) | 用作索引 |

- **默认**：`i32`
- **isize/usize**：集合索引

### 整型字面量

| 形式 | 示例 |
|------|------|
| 十进制 | `98_222` |
| 十六进制 | `0xff` / `0xFF_AA_00` |
| 八进制 | `0o77` |
| 二进制 | `0b1010_1100` |
| 字节(u8) | `b'A'` |

#### 数字里的下划线 `_`

1. **仅作视觉分隔**，编译器**直接忽略**，不改变数值。
2. **分组位数自由**：`98_222`（千分位习惯）与 `9_8222`（万位习惯）**完全等价**。
3. **非法写法**：开头/结尾 `_`（`_123`、`123_`）、连续 `__`（`12__34`）。

```rust
let n1 = 98222;
let n2 = 98_222;
let n3 = 9_8222;
assert!(n1 == n2 && n2 == n3);

let bin = 0b1010_1100;
let hex = 0xFF_AA_00;
```

类型后缀如 `57u8`；也可用 `_` 作**模式**中的占位符（与字面量分隔符不同，见第 18 章）。

### 整型溢出

- **debug 模式**：溢出会 panic
- **release 模式**：二进制补码包裹 (wrapping)，如 u8 的 256→0
- 显式处理：`wrapping_*`、`checked_*`、`overflowing_*`、`saturating_*`

### 浮点类型

| 类型 | 大小 | 精度 |
|------|------|------|
| f32 | 32 位 | 单精度 |
| f64 | 64 位 | 双精度（**默认**） |

- 遵循 IEEE-754
- 默认 `f64`，在现代 CPU 上与 f32 速度相近但精度更高

### 数字运算

加 `+`、减 `-`、乘 `*`、除 `/`、取模 `%`

- 整数除法**向下取整**：`2 / 3` → `0`

### 布尔类型

```rust
let t = true;
let f: bool = false;
```

- 值：`true` / `false`
- 大小：1 字节

### 字符类型 char

```rust
let c = 'z';
let z = 'ℤ';
let heart_eyed_cat = '😻';
let zhong = '中';
```

- **单引号**（区别于字符串双引号 `"..."`）
- **4 字节**，表示 **Unicode 标量值**（不是完整 grapheme；组合 emoji 等可能需多个 `char` 或 `str`）
- 支持 ASCII、中文、emoji 等

**常见错误（编译期）**：

```rust
// let c = "z";   // 错误：期望 char，得到 &str
// let c = 'ab';  // 错误：char 只能是一个 Unicode 标量
```

**与 `str` 区分**：`'a'` 是单个标量；`"a"` 是 UTF-8 字符串切片（长度可能 > 1 字节）。

**延伸阅读**：`3.2-Unicode与字符串.md` + `3.2-unicode-demo/`（字节 vs 字符、截取/翻转模板）。

## 复合类型 (Compound)

### 元组 (tuple)

- **固定长度**，元素类型可不同（异构）
- 小括号创建

```rust
let tup: (i32, f64, u8) = (500, 6.4, 1);

// 解构：一次性绑定所有元素
let (x, y, z) = tup;

// 索引：tup.0、tup.1…（从 0 开始，字面量索引）
let five_hundred = tup.0;
```

- **单元类型** `()`：无元素元组；无返回值的表达式隐式得到 `()`

**常见错误**：

```rust
// let (a, b) = tup;           // 错误：元素个数不匹配
// let fourth = tup.3;         // 错误：只有 .0 .1 .2
// let bad: (i32, i32) = (1,); // 错误：单元素元组要写成 (1,)
```

**运行 demo**：`cd 03-common-concepts/3.2-data-types-demo && cargo run`

**Playground**：把 `3.2-data-types-demo/src/main.rs` 里「§6 char」「§7 tuple」两段复制到 [Rust Playground](https://play.rust-lang.org/) 即可单文件运行（无需 `Cargo.toml`）。

### 数组 (array)

- 固定长度，元素类型**相同**
- 分配在栈上
- 方括号创建

```rust
let a = [1, 2, 3, 4, 5];
let a: [i32; 5] = [1, 2, 3, 4, 5];  // 类型标注
let a = [3; 5];  // [3, 3, 3, 3, 3]

// 访问
let first = a[0];
let second = a[1];
```

- 越界访问：**运行时 panic**，Rust 会检查索引
- 需动态大小用 **Vec**（第 8 章）

## 小结

| 分类 | 类型 | 要点 |
|------|------|------|
| 标量 | i8~i128, u8~u128, isize, usize | 默认 i32 |
| 标量 | f32, f64 | 默认 f64 |
| 标量 | bool | true/false |
| 标量 | char | 单引号，Unicode |
| 复合 | tuple | 解构或 .0/.1 访问 |
| 复合 | array | 固定长度，栈分配 |
