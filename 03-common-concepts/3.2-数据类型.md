# 3.2 数据类型

Rust 是**静态类型**语言，编译期必须知道所有变量类型。编译器可推导类型，歧义时需显式标注。

```rust
let guess: u32 = "42".parse().expect("Not a number!");
// 无标注会报错: type annotations needed
```

---

## 标量类型 (Scalar)

表示单个值，共 4 种：整型、浮点、布尔、字符。

### 整数类型

| 长度 | 有符号 | 无符号 | 范围 (有符号) | 范围 (无符号) |
|------|--------|--------|---------------|---------------|
| 8 位 | i8 | u8 | -128~127 | 0~255 |
| 16 位 | i16 | u16 | -(2^15)~2^15-1 | 0~2^16-1 |
| 32 位 | i32 | u32 | -(2^31)~2^31-1 | 0~2^32-1 |
| 64 位 | i64 | u64 | -(2^63)~2^63-1 | 0~2^64-1 |
| 128 位 | i128 | u128 | ... | ... |
| arch | isize | usize | 取决于架构 (32/64位) | 用作索引 |

- **默认**：`i32`
- **isize/usize**：集合索引

### 整型字面量

| 形式 | 示例 |
|------|------|
| 十进制 | `98_222` |
| 十六进制 | `0xff` |
| 八进制 | `0o77` |
| 二进制 | `0b1111_0000` |
| 字节(u8) | `b'A'` |

可用 `_` 分隔，如 `1_000`；类型后缀如 `57u8`。

### 整型溢出

- **debug 模式**：溢出会 panic
- **release 模式**：二进制补码包裹 (wrapping)，如 u8 的 256→0
- 显式处理：`wrapping_*`、`checked_*`、`overflowing_*`、`saturating_*`

### 浮点类型

| 类型 | 大小 | 精度 |
|------|------|------|
| f32 | 32 位 | 单精度 |
| f64 | 64 位 | 双精度（**默认**） |

- 遵循 IEEE-754
- 默认 `f64`，在现代 CPU 上与 f32 速度相近但精度更高

### 数字运算

加 `+`、减 `-`、乘 `*`、除 `/`、取模 `%`

- 整数除法**向下取整**：`2 / 3` → `0`

### 布尔类型

```rust
let t = true;
let f: bool = false;
```

- 值：`true` / `false`
- 大小：1 字节

### 字符类型 char

```rust
let c = 'z';
let z = 'ℤ';
let heart_eyed_cat = '😻';
```

- **单引号**（区别于字符串双引号）
- 4 字节，Unicode 标量值 (U+0000~U+D7FF, U+E000~U+10FFFF)
- 支持 ASCII、中文、emoji 等

---

## 复合类型 (Compound)

### 元组 (tuple)

- 固定长度，元素类型可不同
- 小括号创建

```rust
let tup: (i32, f64, u8) = (500, 6.4, 1);

// 解构
let (x, y, z) = tup;

// 索引访问
let five_hundred = tup.0;
let six_point_four = tup.1;
let one = tup.2;
```

- **单元类型** `()`：无值元组，表达式无返回值时隐式返回

### 数组 (array)

- 固定长度，元素类型**相同**
- 分配在栈上
- 方括号创建

```rust
let a = [1, 2, 3, 4, 5];
let a: [i32; 5] = [1, 2, 3, 4, 5];  // 类型标注
let a = [3; 5];  // [3, 3, 3, 3, 3]

// 访问
let first = a[0];
let second = a[1];
```

- 越界访问：**运行时 panic**，Rust 会检查索引
- 需动态大小用 **Vec**（第 8 章）

---

## 小结

| 分类 | 类型 | 要点 |
|------|------|------|
| 标量 | i8~i128, u8~u128, isize, usize | 默认 i32 |
| 标量 | f32, f64 | 默认 f64 |
| 标量 | bool | true/false |
| 标量 | char | 单引号，Unicode |
| 复合 | tuple | 解构或 .0/.1 访问 |
| 复合 | array | 固定长度，栈分配 |
