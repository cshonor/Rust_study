# 3.5 控制流

控制执行流的两种主要结构：**if 表达式** 和 **循环**。

---

## if 表达式

### 基本语法

```rust
if condition {
    // 条件为 true 时执行
} else {
    // 条件为 false 时执行
}
```

- 条件**必须**是 `bool`，Rust 不会自动转换
- 分支代码块叫 arm（与 match 类似）

### 条件必须是 bool

```rust
let number = 3;
if number { }           // ❌ 错误: expected `bool`, found integer
if number != 0 { }      // ✓ 正确
```

### else if 多重条件

```rust
if number % 4 == 0 {
    println!("divisible by 4");
} else if number % 3 == 0 {
    println!("divisible by 3");
} else {
    println!("not divisible by 4 or 3");
}
```

- 只执行**第一个**为真的分支
- 过多 else if 建议用 **match**（第 6 章）

### 在 let 中使用 if

if 是**表达式**，可赋值给变量：

```rust
let number = if condition { 5 } else { 6 };
```

**要点**：if 与 else 分支的**返回值类型必须相同**！

```rust
let n = if condition { 5 } else { "six" };  // ❌ 类型不兼容
```

## 循环

### 1. loop — 无限循环

```rust
loop {
    println!("again!");
    // break; 退出循环
    // continue; 跳过本次，进入下一次
}
```

- 需用 `break` 或 `ctrl+c` 退出

### 循环标签 (loop label)

嵌套循环时，用标签指定 break/continue 作用于哪层：

```rust
'counting_up: loop {
    loop {
        break;           // 只退出内层
        break 'counting_up;  // 退出外层
    }
}
```

### 从 loop 返回值

```rust
let result = loop {
    counter += 1;
    if counter == 10 {
        break counter * 2;  // 返回值
    }
};
// result = 20
```

### 2. while — 条件循环

```rust
while number != 0 {
    println!("{}!", number);
    number -= 1;
}
```

### 3. for — 遍历集合

**推荐**：安全、简洁，Rust 最常用循环。

```rust
let a = [10, 20, 30, 40, 50];
for element in a {
    println!("the value is: {}", element);
}
```

**Range 倒计时**：

```rust
for number in (1..4).rev() {
    println!("{}!", number);
}
// 输出: 3!, 2!, 1!
// LIFTOFF!!!
```

- `1..4`：1, 2, 3（不含 4）
- `1..=4`：1, 2, 3, 4（含 4）
- `.rev()`：反转区间

---

## 小结

| 结构 | 用途 |
|------|------|
| if / else if / else | 条件分支，条件必须 bool |
| loop | 无限循环，break 返回值 |
| while | 条件循环 |
| for | 遍历集合，最常用 |
